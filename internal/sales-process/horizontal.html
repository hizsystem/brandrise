전체 영업 동선 — 가로 한 줄

어느 채널로 들어와도 종착은 브랜드라이즈 진단(상담 신청)이다. 페이즈마다 표준 산출물과 자동 알림톡이 붙는다. 헤더가 검은 칸 = 최종 수주.

영업
접수 → 분별 → 담당 배치
1상담 신청 접수
광고·랜딩·채널에서 진단 신청이 들어온다. 접수되면 알림톡 자동 발송.
🔔 신청 접수
2콜 · 1차 분별
전화로 매출 규모·의지(열)·가능성·유입경로를 확인하고 대표(★) 건과 일반 건으로 가른다."돈 되는 건"을 빨리 알아보는 감각이 첫 단추. 규모·온도가 담당 배치와 견적 결을 가른다.
유입경로★ 대표
3스케줄링 — 담당 배치
운영 시트에 담당자·일정을 잡는다. 대표(★) 건은 따로 분기. 일정 잡히면 알림톡.
🔔 일정 확정★ 대표
영업 핵심받아서 거르고, 누구에게 붙일지 정한다. 매출 규모·의지로 ★대표·일반을 가르는 게 영업의 첫 일.
PHASE 1
1차 · 상담
사전질문 → PREP → 상담 → 진단서
1메일 — 사전 질문지 발송
상담 전 질문지를 보내 자료를 미리 받는다. 받은 자료가 PREP 재료가 된다.
사전 자료
2PREP 작성 (내부)
상담 전 내부 준비. 받은 자료·리서치로 "막힌 지점" 가설을 미리 세운다.
📄 PREP
3상담 진행
대면/콜. 첫 15분 회사소개 후 마이크를 넘긴다. 대표(★) 건은 대표가 직접 전략 제시.진행 골격·화법은 상담 플레이북을 따른다.
일반★ 대표
4상담 완료 → 진단서
끝나면 3일 안에 진단서 발송. 완료 처리하면 알림톡.
🔔 완료
1차 산출물PREP(내부 준비) + 진단서(고객 제출·견적 근거). 진단서 병목 번호가 견적 블록과 1:1 대응.
PHASE 2
2차 · 견적
초안 → 컨펌 → 발송 → 견적콜
1견적 초안 작성
진단서 병목을 그대로 견적 블록으로 옮긴다. 규모에 따라 HTML/시트로 나눠 만든다.
📄 견적서
2컨펌 — 내부 → 대표
내부에서 한 번 보고 대표가 최종 컨펌. 총액이 규모에 맞는 티어인지 확인.
★ 컨펌
3견적 발송 — A/B안
두 안을 같이 보낸다. 차이가 눈에 보이게, 추천안을 분명히 짚는다.단가를 즉석에서 깎거나 번복하지 않는다.
A / B안
4견적콜
전화로 설명이 부족했던 부분을 채운다. 안 풀리면 대표 SOS.콜 화법은 견적콜 플레이북 참고.
결정 지원
2차 산출물견적서(A/B안) — A=작은 yes 받는 진입 앵커, B=권장(풀 범위)
PHASE 3
3차 · 계약
계약 → 셋업(마스터시트)
1계약
범위·금액 확정. 부담스러워하면 데이터 진단 하나만 먼저 착수하는 가벼운 진입도 연다.
수주
2셋업 — 마스터시트
월간 플랜·관리 데이터를 마스터시트에 깐다. 첫 워크숍에 대표가 들어가 전략을 잡는다.
마스터시트
첫 끝발첫 워크숍·디자인 미팅은 대표가 직접 — 전략→녹음→매뉴얼화. 다음부터 팀이 받는다. 첫 단추만 잘 끼우면 쉬워진다.
RESULT
수주
운영 인하우스 전수
✓운영 착수
마스터시트로 매월 운영하고, 운영 노하우를 인하우스(클라이언트 내부)로 넘긴다.
전수
목표대표 개인기를 누구나 굴리는 시스템으로. 쌓인 데이터가 다음 영업의 근거가 된다.
자동화 포인트 · 알림톡 3회① 상담 신청 · ② 일정 확정 · ③ 상담 완료 — 스케줄링과 완료 처리만 사람이 누르면 이 세 알림톡은 자동으로 나간다.
