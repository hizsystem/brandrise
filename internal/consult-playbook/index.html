WHY

이 매뉴얼이 하는 일

영업이 "대표 개인기"에 묶이면 회사가 못 큰다. 대표·실장의 상담·견적콜을 녹취 → 반복 가능한 플레이북으로 만들어, 신입도 같은 무브를 쓸 수 있게 한다.

읽는 법아래 원칙은 "왜 그렇게 말하는가", 단계별 플레이는 "언제 무엇을 말하는가", 멘트는 대표의 실제 표현이다. 외우지 말고 왜 먹히는지를 소화할 것.
v0.1 한계현재 NGT 상담 1건(대표)에서 추출. 애라 실장 상담·견적콜 녹취가 쌓이면 v0.2로 확장(특히 실장의 1차 분별·견적콜 클로징). 임애라 실장 검토·보강 예정.
PRINCIPLES

상담 5대 원칙

NGT 녹취에서 반복된, 수주를 만드는 핵심 무브.

1
진단을 "근거 패키지"로 판다
"리브랜딩 비용"이 아니라 "의사결정할 근거가 없어서 못 정하는 문제"를 푸는 패키지로 포지셔닝. 고객의 진짜 갈증을 입구로.
2
외부 주도 명분을 준다
"직원은 리브랜딩 못 한다, 내부에서 하면 망한다." 외부(HIZ)가 해야 하는 이유를 논리로 깔아 수주 명분을 만든다.
3
커머디티를 디밸류한다
"예쁜 패키지·예쁜 글은 ChatGPT도 한다." 흔한 것을 깎아내리고 진단·전략(우리만 하는 것)의 가치를 띄운다.
4
범위를 선긋는다
"광고 직접 운영은 너무 헤비, 안 받습니다." 안 할 것을 명확히 거절해서 오히려 신뢰와 전문성을 산다.
5
데이터를 열게 + 날짜를 박는다
"메타·매출·회원수 주세요" = 진단 입구 클로징. "12월 세일 → 1월 런칭" 캘린더로 긴급도를 고정해 미룸을 막는다.
PLAY · 1차 상담

단계별 플레이 — 무엇을, 언제

영업 프로세스 1차(상담) 흐름에 매핑. 각 단계의 목표와 대표의 실제 멘트.

콜 · 규모 파악
매출·체급·열을 먼저 읽는다
대면 전 콜에서 매출 규모·관심도·가능성을 가늠해 담당(대표 별표 vs 일반)을 가른다. "돈 냄새"를 맡는 감각이 배치를 결정.
오프닝
고객이 "진짜 못 푸는 것"을 말하게 한다
제품·매출 히스토리를 듣되, 표면 요청(리브랜딩) 아래의 진짜 페인을 끌어낸다.
대표님이 아직 잘 모르겠어서 결정을 못 하시는 거예요. 정보를 제대로 넣어주는 사람이 없는 거죠— 권윤정 (27:56)
축 재정의
숨은 축을 찾아 무게중심을 옮긴다
고객/브리프가 생각한 축이 아니라, 상담 중 드러난 진짜 해자로 축을 재정의(NGT: 여성웰니스 → "100% 제조사 정통성").
100% 제조사 기반이 강점이에요. 소비자는 원산지에서 야채 사 먹고 싶어 한다. 그거예요— 권윤정 (19:37)
진단 포지셔닝
진단 = 의사결정 근거로 띄운다 (원칙 1·3)
진단을 비용이 아니라 "건강성 검사 + 결정 근거"로.
리브랜딩은 동의. 근데 패키지 예쁘게 바꾸고 예쁜 글 쓰는 건 ChatGPT가 잘한다. 더 중요한 건 지금 우리의 건강성을 진단하는 것— 권윤정 (11:50)
골격 제시
6개월 3단계 로드맵을 그린다
서베이→사업계획 / 리브랜딩(외부 주도, 원칙 2) / 마케팅(예산 안에서 선택). "3개월 성과는 거짓말, 6개월 프로젝트."
직원은 리브랜딩 못 합니다. 내부에서 하면 대부분 망해요. 저 같은 외부 사람은 쉽게 하죠— 권윤정 (14:33)
범위 선긋기
안 할 것을 거절한다 (원칙 4)
광고 직접 운영은 헤비하다며 제외. 현 대행 ROAS 0.3은 이관 권고만.
ROAS 0.3은 바로 이관해야죠. 다 모았다가 한 번을 제대로 터뜨리는 걸 추천— 권윤정 (36:18)
클로징
데이터를 열게 하고 날짜를 박는다 (원칙 5)
진단 입구를 합의로 만들고, 데이터 요청 + 캘린더로 긴급도 고정.
진단도 받고 견적도 받고 싶어서 → 메타 데이터 + 월별·채널별·제품별 매출 + 회원수 주세요— 대표/권윤정 (54:00)
상담 후3일 내 진단서 + 견적서 송부. 진단의 병목 번호가 견적 블록에 1:1 대응 (영업 프로세스 1차→2차 연결).
QUOTE

견적 포지셔닝 원칙

앵커링진입 앵커 = 작은 yes (데이터 진단 단독 착수 가능). A/B 2안으로 "차이를 눈으로" 보게 해 권장안을 선택하게 한다. 공통 항목 단가는 브랜드 불문 동일, 차이는 블록 깊이로.
티어 기준대표가 들어가는 컨설팅 = 최소 5천(또는 3~4천 진입가, 억대 확장 확인 시). 견적은 싯가 — 기업 규모·브랜딩 의지로 총액을 잡고 기획파트로 미세조정.

※ 견적콜 단계의 멘트·반론 처리는 견적콜 녹취 확보 후 v0.2에서 보강.

온도 읽기 — 관심·긴급·예산

상담 중 고객의 3축 온도를 읽어 클로징 압력을 조절한다 (NGT 기준 예시).

축신호읽는 법 · 대응
관심도높음 ★★★★본인이 직접 신청, 라이브로 데이터까지 연다 → 진단 입구는 거의 확정. 입구를 빨리 합의로 굳힌다.
긴급도중간 ★★★위기의식은 있으나 "몇 년째 못 정함" → 날짜(세일·런칭)로 데드라인을 박아 미룸을 차단.
예산의지중간 ★★★컨설팅엔 열려 있으나 "다 바꿀 순 없다" 신중 → 단계적 흡수(진입 앵커 → 확장)로 부담을 쪼갠다.
CASE

케이스 — NGT (이 매뉴얼의 원천)

2026-06-10 NGT 김수안 대표 상담(1:44). 진단+견적 송부 합의까지 도달. 위 원칙·플레이가 실제로 쓰인 한 건.

무브이 상담에서
축 재정의"여성 웰니스" → "100% 제조사 정통성" (자사 원료공장·K-낙산균 = 경쟁사 못 따라오는 해자)
진단 포지셔닝"건강성 진단이 먼저" — 리브랜딩 견적이 아니라 의사결정 근거로
외부 주도 명분"직원은 리브랜딩 못 한다" → 외부 HIZ가 해야 하는 논리
범위 선긋기광고 직접 운영 제외, ROAS 0.3 이관 권고만
클로징데이터 요청 + "12월 세일 → 1월 런칭" 캘린더로 긴급도 고정
결과진단 입구 수주확률 80% · 6개월 리테이너 50%(진단 퀄리티가 클로징 레버). 전체 녹취·상담랩업은 로컬 비공개(`13.내추럴굿띵스`).
