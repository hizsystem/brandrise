MANUALS

매뉴얼 — 어떻게 말하는가

단계마다 복제 가능한 플레이북. 녹취에서 반복되는 무브를 뽑아 누구나 같은 수준으로 플레이한다.

진입회사소개 매뉴얼
상담 첫 15분 회사 소개를 누구나 같은 수준으로. ①~⑥ 통대본 + 사례 카드 선택표 + 금지 5가지 + 부록 A(STEP·체크리스트).
매뉴얼 열기 ›v4 · 대표 녹취 3건 + 갭 보강
1차상담 플레이북
대표·실장 상담을 복제·암기·소화. 상담 5단계 · 2패턴 · 넥스트 질문 8종 · 결핍 직언 5종 · 견적 화법 11종 · 클로징.
플레이북 열기 ›v3 · 대표 녹취 14건 + 실장 녹취 12건 실증
2차견적콜 플레이북
견적 발송 후 결정을 돕는 콜. 5대 원칙 + 단계별 플레이 + 반론 처리 + 바로 쓰는 멘트.
플레이북 열기 ›v0.1 · 쿠엔즈버킷 견적콜(권윤정 대표)
DELIVERABLES

산출물 표준 템플릿 — 무엇을 만드는가

단계 산출물은 인디고 클린 단일 표준으로 통일. 복제해 콘텐츠만 교체한다. 공통 항목 단가는 브랜드 불문 동일.

1차PREP
상담 전 내부 준비 — 직접 리서치 기반 가설 정리. 고객에게 안 나가는 무장 문서.
템플릿 열기 ›research/_template · 좌측 사이드바형
1차진단서
