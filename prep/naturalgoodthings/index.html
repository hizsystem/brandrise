ONE-LINER

한 줄 요약

사업 본질 한 줄

"지속가능한 여성 웰니스" D2C 자생 성장형 중견 건강기능식품 브랜드 — 김수안 대표(CEO Kim Soo An)가 이끄는 9년차 자체 R&D·OEM 운영사. 주력 SKU "속편한 생생효소(곡물발효효소)" = 누적 1억 포(100M 사쉐) 돌파 + 한국소비자만족지수(KCSI) 효소 카테고리 6년 연속 1위(2019~2024). 2024년 아시아 진출(일본·태국·싱가포르) 개시, 2026년 대만 인플루언서 Halo Mavis 라이브 커머스 4회 누적 25만 개 판매 — 글로벌 D2C 전환 핵심 시점에 컨설팅 의뢰.

ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 위타민(우르실) / 리솔츠 (가장 가까움) — 자체 R&D 보유 + 자사몰 + 글로벌 진출 시점 구조 = 거의 동일. 다만 NGT는 이미 매출 100억대 + 카테고리 1위 + 글로벌 LIVE COMMERCE 성과 검증 = 위타민/리솔츠보다 1~2단계 위 스테이지(스케일 전환 후반기).
  • 케어비네스트 / 절대콜라겐 — 건기식 D2C + 자사몰 + 멀티 채널 = 비즈모델 일치. 다만 NGT가 매출·글로벌 진척 모두 우위.
  • 레베로(LEVERO) / 마키노차야 — 지속가능·웰니스 브랜드 정체성·콘텐츠 시스템 측면 인접.
  • 요약: HIZ 풀에서 "자체 R&D 보유 + 카테고리 1위 + 글로벌 다중 채널 동시 진출" 동시 갖춘 케이스 없음 = NGT가 컨설팅 풀에서 가장 스케일 큰 D2C 잠재 계약. T2~T3 핏.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 글로벌 콘텐츠 시스템 + 데이터 대시보드 + 채널 운영 통합 1순위. 브랜드 정체성·BI는 이미 정착, 단 글로벌 진출 시점 재정의 필요. - 30일 (진단): 자사몰 vs 오픈마켓 vs 글로벌 채널별 매출 분해 / 본 계정·KOL 콘텐츠 자산 인벤토리 / 글로벌 진출 4개국(일·태·싱·대만) 채널·인플루언서·매출 매핑 / "여성 웰니스" 마스터 브랜드 아키텍처 진단 - 90일 (시스템 구축): 글로벌 콘텐츠 SOP(본사 콘텐츠 풀 → 4개국 큐레이션·번역 시스템) / 자사몰 LTV·재구매·CRM 대시보드 v1 / 본 계정 인스타·유튜브 콘텐츠 정상화(주 3~5회 발행 시스템) / Halo Mavis 모델 확장 = 4개국 KOL 시드 각 10명 - 180일 (스케일): 자사몰 → 글로벌 → B2B(약국·드럭스토어·면세) 매출 분해 대시보드 / 신규 카테고리(여성 호르몬·체중관리·뷰티이너) 런칭 캠페인 / 미국·유럽 영문 자산 v1

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T2 Build 상단~T3 Scale 진입 (5,000만~9,000만 / 4~6개월)
  • 근거: 매출 169억 + 9년차 + 카테고리 1위 + 글로벌 4개국 동시 진출 = T1 진단 단계 이미 자체 통과한 케이스. 글로벌 콘텐츠 시스템·데이터 대시보드·다중 채널 통합 = 풀스코프 빌드 필수. 매출 규모 대비 마케팅 예산 흡수력 충분.
  • 단계적 접근 옵션 (권장): T1 Foundation(1,500~2,000만/4주) 진단 → T2/T3 Build 승급 단계 추천. 김수안 대표가 자생 성장 9년 = 외부 컨설팅 도입 첫 케이스 추정 = 흡수력 검증 후 본격 빌드 단계 권장.
  • 대안:
  • 자사몰 D2C 단일 영역만 풀고 글로벌·B2B 별도 단계 → T2 Build 단독(3,000~5,000만/3개월)
  • 4개국 동시 글로벌 시스템 + 신규 카테고리 런칭 캠페인 + 데이터 대시보드 풀스택 → T3 Scale(7,000만+/6~12개월) 적극 권장
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
