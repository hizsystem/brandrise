PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 브랜드 정체성·포지셔닝·타겟 정의가 아직 안 됨 → 콘텐츠/SNS를 시작해도 무엇을 만들지 막막
  • E-커머스라 채널(자사몰/오픈마켓/SNS 커머스) 선택지가 많은데, 어디부터 시작해야 할지 우선순위 부재
  • 1인 또는 소규모 운영 추정 — 촬영·제작·운영을 외주냐 인하우스냐의 의사결정 기준 부재
  • "처음" 단계 = 학습 의지는 있으나 정보 격차 큼 → 짧은 진단·워크숍이 실효 큼
  • 명시 예산 100만 = 외주 풀 파트너십 진입가 미달, 단발 진단·로드맵 단위로만 ROI 확보 가능
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

약점 영역: 브랜드 정체성 + 콘텐츠 + 채널 (전 영역 시작점). 본인 기대치가 "맞춤형 로드맵"이라 T0 진단형 또는 T1 Foundation 하단(500~700만)이 적합. 6영역 중 정체성·콘텐츠·채널 3영역 동시 약점. - 30일: 무료/단발 진단 6영역 → 사업 본질·차별점·타겟 1차 정렬, 콘텐츠/채널 우선순위 결정, 자사몰·인스타 1차 셋업 가이드 - 90일: SNS 1개 채널 집중 운영 표준 + 촬영/제작 외주 vs 인하우스 분기 기준 + 첫 콘텐츠 시드 5~10개 - 180일: 자사몰·오픈마켓 데이터 기반 콘텐츠 KPI 사이클 (T1 Foundation 본격 진입 시점)

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T0 진단형 (100~300만 / 2주) 또는 T1 Foundation 하단(500~700만 / 4주)
  • 근거: 명시 예산 100만 + "브랜딩 처음" + 식별 정보 부재 = T1 풀스코프 진입 불가. T0 진단 → 검증 후 T1/T2 업셀 동선이 정석.
  • 대안:
  • 미팅 결과 카테고리·매출·예산 추가 투입 의지 확인 시 → T1 Foundation 500~1,500만/4주 진입
  • 가용 예산 100만 단발 + 추가 투입 의지 없음 → 무료 6영역 진단 + 단발 워크숍 모드 + 6개월 후 재상담
  • 단계적 접근(권장): 본 건은 T0 진단 → T1 Foundation → T2 Build 3단계 업셀 동선이 가장 적합. 첫 진입 비용을 최소화하고 본인 학습 → 데이터 확보 → 본격 빌드 단계 권장.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
