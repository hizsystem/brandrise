한 줄 요약

사업 본질 한 줄

한·일 듀얼 거점 멀티 브랜드 그룹 — 한국법인 ㈜웨어컴퍼니(대표 황현우, 2024-09-02 설립)가 wearwhere(K-아이웨어 D2C 브랜드) 운영, 일본법인 Wear Company Japan, Inc(대표 NAKANISHI RINA, 오사카)가 MIMIC LENS(컬러 콘텍트렌즈, 8K 기술) 운영. 두 법인 모두 도메인 wearwhereyou.com 공유 = 동일 그룹 두 브랜드 결론 (한 파일 통합).

추정 단계

현재 추정 단계

시드~초기 (1년차 미만 영리법인 + 일본법인 동시 가동). 한국법인 설립이 2024-09-02로 8개월차에 불과하지만, 일본법인은 이미 자사몰·Qoo10·LIPS 후기 등 운영 중 = 일본 시장 선진입 후 한국법인을 통한 K-아이웨어 본진 확장 시작 패턴 강한 추정. 한국 직원 3명 = 슬림 운영, 일본 채널은 상대적 두텁게 노출.

ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 참고 케이스 풀에 정확한 매칭 없음 — 핏 검증 필수. HIZ 표준 D2C SOP는 단일 시장(한국) 전제. 한·일 듀얼 브랜드 + 의료기기(콘택트렌즈) 카테고리는 신규 영역.
  • 노다지파이랩스 / 와이림: 1년차 미만 + 직원 3명 슬림 운영 + 자사몰 부재가 아닌 자사몰 보유 = 운영 구조 인접도 일부.
  • 리솔츠 / 위타민: 뷰티·아이웨어·렌즈 = "외모 직결 D2C" 카테고리라는 점에서 콘텐츠 전략 인접도 있음. 단 의료기기(콘택트렌즈) = 광고법(의료기기법) 추가 제약 = HIZ 미경험 영역.
  • WAYD / 위타민 초기: 사명 동명 충돌 다수 케이스 = 브랜드 정체성·검색 점유 1순위 페인 인접.
PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 한·일 듀얼 브랜드 = 브랜드 아이덴티티 분산 위험. wearwhere(아이웨어) + mimic lens(렌즈)가 한 그룹임에도 외부 인지는 별개. 그룹 마스터 브랜드 전략(엄브렐러 vs 멀티 브랜드 독립) 미정 추정.
  • 한국법인 8개월차 + 직원 3명 = 한국 D2C 콘텐츠·채널 운영 미정착. wearwhere.co.kr 자사몰 + 인스타 운영 빈도·인게이지먼트 미확인. 일본 매출 의존도가 크고 한국은 시작 단계일 가능성.
  • 사명 동명 충돌 다수 (wherewear.kr, wearb, whatwewear 등) → 검색·핸들·SEO에서 식별 비용 큼. "WearWhere"의 의미 전달도 모호.
  • 의료기기(콘택트렌즈) 광고 규제 + 일본 의료기기 판매허가 = 한국에서 동일 라인 판매 시 KFDA 의료기기 인허가 별도 필요. 한·일 마케팅 메시지 동기화 어려움.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(그룹·브랜드 아키텍처) + 채널 인프라(한국 D2C 셋업) + 콘텐츠 시스템 1순위. 데이터 대시보드는 매출 규모 확인 후 검토. - 30일: 그룹 브랜드 아키텍처 정의 (Wear Company 마스터 vs wearwhere/mimic lens 독립 / 엄브렐러 / 하우스 오브 브랜드 중 선택) + 동명 충돌 매핑 + 한국 자사몰·SNS 진단 - 90일: wearwhere 한국 인스타·유튜브 콘텐츠 시스템 구축(아이웨어 룩북·수면관리 Deep Life 시리즈 콘텐츠), 핵심 카피·BI 정리 - 180일: 한국 퍼포먼스 ROAS 시범 운영 + 일본 mimic lens 운영 효율화 자문(범위 제한), 한국 KFDA 렌즈 라인업 런칭 검토 시 추가 스코프

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (1,000~1,500만 / 4주) → T2 Build 승급(2,500~4,000만 / 3개월) 단계적 접근
  • 근거: 한국법인 8개월차 + 직원 3명 + 자사몰 KRW 59,000~99,000 가격대 = 매출 규모 추정 작음. T2 풀스코프 흡수력 미확정 → 진단·아키텍처 정의(T1) 우선, 결과 보고 빌드 단계 결정.
  • 대안:
  • 일본 mimic lens 매출이 한국법인 대비 크고 그룹 차원 마케팅 예산 흡수력 있다면 → T2 Build(2,500~4,000만/3개월) 바로 진입 가능
  • 한·일 동시 글로벌 시스템 구축 + 의료기기 인허가 자문까지 → T3 Scale(7,000만+/6개월+) — 단, HIZ 단독 풀스코프 어려움(법무·인허가 파트너 필요)
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
RISK SIGNALS

리스크 시그널

GO 분기점 — 의사결정·핏·예산·동명·운영 5종

  • 의사결정 라인 불명 — 신청자 "이사/임원" = 한국 ㈜웨어컴퍼니인지 일본법인 측인지 미확인. 황현우 대표 직접 결재 vs 임원 위임 범위 확인 필수.
  • 법인 매출·예산 흡수력 미확인 — 8개월차 + 3인 운영 = T2 풀패키지 흡수 가능성 미검증. 미팅에서 한·일 합산 그룹 매출 규모 청취 필요.
  • 카테고리 핏 한계 — 의료기기(콘택트렌즈) = HIZ 표준 D2C SOP 외 영역. 광고법·의료기기법·일본 약기법 등 규제 자문은 HIZ 단독 어렵고 외부 법무 파트너 필요.
  • 그룹 사명 분산 위험 — Wear Company / wearwhere / mimic lens / wearwhereyou(도메인 메인) = 외부 인지 분산. 브랜드 아키텍처 합의 없으면 마케팅 ROI 떨어짐.
  • 동명 충돌 다수 — wherewear.kr, wearb, whatwewear 등과 검색·SEO 경쟁. 신청 브랜드의 차별화 한 줄 미확정 시 GTM 비용 큼.
