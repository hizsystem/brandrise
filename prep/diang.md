## 유사 케이스 인접도
- **참고 케이스 풀에 정확 매칭 없음 — HIZ B2B 친환경 제조 브랜드 컨설팅 신규 카테고리.** HIZ 표준 SOP는 D2C 전제. 디앙은 B2B 카페·프랜차이즈 + B2C/D2C 병행 + 글로벌 수출 = 비즈모델 다층.
- **LEVERO (가장 가까움)** — 비건 라이프케어 + ESG·지속가능 정체성 + B2B/B2C 병행 + 글로벌 진출 모색 = 카테고리 정체성 가장 인접. LEVERO 컨설팅 SOP 일부 적용 가능.
- **마키노차야** — F&B + 친환경 정체성 + 17년차 안정기 단계 = 운영 단계 인접.
- **부족한녀석들(어프리데이)** — 17년차는 아니지만 **글로벌 진출 + 수상 인증 자산 + 시리즈 진입 직전** 단계가 유사. ESG·임팩트 자산화 전략 인접.
- **요약**: B2B 제조 강소기업이 **글로벌·D2C 브랜드 확장**을 시도하는 케이스 = HIZ 첫 시도. **카테고리 핏 검증 필수**, 단 ESG·임팩트 자산화는 HIZ가 잘하는 영역.

## 핵심 페인 (가설)
- **B2B 제조 강자 vs 브랜드(B2C/D2C) 자산 격차 (강한 추정).** 매출 19.2억 + 17년 누적 인증 3개(EL724·FSSC22000·TUV) = 제조 신뢰 자산 막대. 다만 **공개 인스타·유튜브 채널 부재** + 검색 시 한경TV·월간인터뷰 PR 위주 = 브랜드 자산이 매출 규모 대비 약함. **"동일프라텍 = B2B 제조사" vs "디앙 = B2C 글로벌 브랜드"** 분리 정체성 미정착.
- **친환경/생분해 카테고리 = ESG·규제 흐름 의존 = 가격 경쟁 함정 위험.** 빨대 단가 경쟁 격화(중국·동남아 PLA 빨대 저가 공세) → **인증·임팩트·스토리 자산 활용 차별화 미흡** 시 가격 함정. ESG 임팩트 가시화(데이터 대시보드·임팩트 리포트) 미정착 추정.
- **글로벌 수출 의지 명시 (월간인터뷰 언급) vs 영문 자산·해외 채널 미흡.** "해외에서도 큰 관심" 언급되지만 디앙 영문 자사몰·아마존·해외 인플루언서·국가별 GTM 부재 추정.
- **매출 19.2억 대비 직원 19명 = 1인당 매출 1억** = 제조업 평균 대비 슬림 운영. **마케팅·콘텐츠·글로벌 영업 인하우스 인력 거의 없음** 추정 = 외주 의존 또는 대표 직접 핸들.
- **국내 B2B 카페 vs 수출 vs B2C/D2C 매출 비중 미공개** = 다음 성장 축 의사결정 정보 부재.
