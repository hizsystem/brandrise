HIZ 기존 케이스 풀과의 매핑

  • 참고 케이스 풀에 정확한 매칭 없음 — 핏 검증 필수. HIZ 표준 D2C SOP는 자체 브랜드 전제. 해외 브랜드 한국 수입·총판/유통 카테고리는 신규 영역.
  • WTM (위타민·우르실) 초기: 해외 라이선스·수입 모델 일부 인접 (별도 검증 필요).

  • 카테고리 가설(시계·주얼리·라이프스타일)이라면: HIZ 풀에는 직접 케이스 없음. 단 D2C 자사몰·인플루언서·PR 시딩 작업은 HIZ 표준 가능.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 수입 유통 모델 특성 — 본사가 정한 브랜드 가이드·콘텐츠 자산을 한국 시장에 현지화하는 GAP. 본사 메시지 그대로 사용 시 한국 소비자 인지 부족 위험.
  • 카테고리 운영 복잡도 — 시계/주얼리/라이프스타일이라면 D2C·자사몰 + 백화점·편집숍·오픈마켓의 멀티채널 운영 부담 큼.
  • 한국 시장 ATW 인지도 미확보 — 외부 검색 노출 거의 없음 = 인플루언서·PR로 브랜드 빌드업 1순위.
  • 본사 vs 한국지사 의사결정 라인 — 마케팅 결재가 본사 가이드 의존 시 한국 현지 캠페인 자유도 제한될 가능성.
  • 사업 본질 미식별 = 컨설팅 가설 자체가 미팅 후 확정 — 카테고리 확정 전까지 모든 가설이 가정 단계.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(한국형 포지셔닝) + 콘텐츠(본사 자산 현지화) + 채널(SNS·인플루언서) + PR 1순위 (카테고리 확정 후). 카테고리 핏 검증이 가설 검증보다 우선. - 30일: 본사 브랜드 가이드 vs 한국 시장 핏 GAP 진단, 카테고리·경쟁사 맵, 라이선스 구조·결재 라인 확정 - 90일: 인스타 + 한국형 콘텐츠 시스템, 인플루언서 시딩 캠페인, 자사몰 한국화 - 180일: PR·매거진·편집숍 협업 + 자사몰 ASO/SEO + 데이터 대시보드

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (500~1,500만 / 4주) 진단형
  • 근거: 사업 본질 미식별 + 매출·인력 공개 부재 = T2 풀스코프 진입 위험. T1 진단형으로 카테고리 확정 + 본사 가이드 검토 + 한국 현지화 GTM 가설 수립이 1순위 순서.
  • 대안:
  • 카테고리 확정 + 매출·예산 흡수력 검증되면 → T2 Build(2,000~5,000만/3개월) 빌드 풀스코프 승급 (한국 채널 셋업 + 인플루언서 시딩 + 콘텐츠 시스템)
  • 본사 글로벌 멀티 마켓 동시 진출(미국·동남아·일본 등) 동반 → T3 Scale(7,000만+/6개월+) 검토
  • 카테고리가 HIZ SOP와 거리 있다고 판단되면 → T0 진단형(100~300만/2주) 최소 점검 후 종료/소개
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
