PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 사업자 신생 4주차 = 출발선 진단 필요 — 무엇을 파는지, 누구를 타깃하는지, 어떤 채널부터 칠지 모든 결정이 미정 가능성. 온라인 자산 0 = 자사몰·SNS·검색 노출 일체 없음.
  • 사명 "가족컴퍼니" 일반명사 조합 = 검색·SEO 매우 약함 — 네이버 검색 시 "가족회사" 정책 용어와 충돌, 동명 사업자 다수 추가 존재 → 브랜드 변별력 확보가 1순위 과제.
  • 본점 정읍 / 운영 파주 이중구조 = 가족 거주지·창고·실제 활동지 분리 추정. 로컬 채널보다 온라인 D2C 100% 의존 구조가 될 가능성 — 자사몰·마켓플레이스·SNS 채널 0→1 셋업 부담 큼.
  • 전자상거래 소매업 = 카테고리 미정 — 패밀리·키즈·돌봄·라이프스타일·반려 등 다수 가능, 미팅 첫 5분 청취 필수.
  • 공동대표 의사결정 구조 — 부부·가족 창업이면 결재 속도 빠르나 역할 분담·자본금·투자 미확인 = 컨설팅 흡수력 검증 필요.

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation 강력 추천 (300~500만 / 4주 진단형)
  • 근거: 설립 4주차 신생 + 공동대표 2인 + 카테고리 미정 + 온라인 자산 0 = T2 풀 빌드 흡수 거의 불가. 진단·전략으로 첫 6개월 로드맵 확정 후 T2 추후 결정이 안전. 정부지원 트랙(소진공·창업진흥원) 동반 시 가격 장벽 추가 해체.
  • 대안:
  • 자본금·자금 매우 슬림 시 → T0 진단형(100~300만/2주) 카테고리·SKU·네이밍 검증 1장 보고서
  • 진단(T1) 결과 후 카테고리·예산 확정되면 → T1 상단(500~1,500만/4주) 자사몰 v1 + 인스타 1개 셋업 포함
  • T2 풀 빌드(2,000만+)는 6개월 이후 매출 검증 후 결정, 현시점 비추천
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
