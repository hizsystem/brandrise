ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • DAYNESTA(데이네스타): 1년차 미만 신생 D2C + 사명·도메인 분산 + 4 페르소나 라인 분산 = 인접도 매우 높음. 단 DAYNESTA는 한미 듀얼 진출 + 4 페르소나 라인, 본 건은 단일 시장 + 단일 사이트(aromesoi) = 본 건이 더 단순한 구조.
  • wearwhere: 1년차 미만 + 직원 3명 + 사명 분산 = 운영 구조 인접도 높음.
  • 케어비네스트 / 위타민: D2C 신생 + 자사몰 시작 단계 = 카테고리 인접도(향수·뷰티는 인접).
  • 루키노바: 페르소나 기반 차별화 + D2C = 향수·향기 카테고리 차별화 메시지 인접도 일부.
  • 요약: HIZ 신생 D2C 셋업 SOP 핏 양호. 단 사명·도메인 분산 정리가 컨설팅 1순위 의제.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 신생 D2C — 브랜드 정체성·콘텐츠·채널 모두 0에서 빌드업 단계 가능성. 1년차 미만 + 직원 3명 = 운영 자원 매우 슬림
  • 사명 "올웰80" 의 "80" 의미 미확인 (시니어? 80년대생? 80SKU? 80가지 향?) → 미팅 시 1순위 질문
  • 사명(올웰) ↔ 도메인(aromesoi) ↔ 컨설팅 신청 도메인(allwell80) 3중 네이밍 분산 → 브랜드 일관성 진단의 명백한 첫 단추. 신규 고객 검색·기억·재방문 모두 비용 큼
  • 향(香) 카테고리 추정 — Arôme Soi(프랑스어 "자기 향") = 향수·디퓨저·룸스프레이 등 향기 라이프스타일 카테고리 추정. 단 실 SKU·가격대·USP 미확인
  • 신생 D2C 자원 운영 — 직원 3명 = 인하우스 마케팅·디자인·콘텐츠 풀세트 불가. 외주 의존도 높을 추정

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (500~1,500만 / 4주) 진단부터 (T2 승급은 진단 결과 후 결정)
  • 근거: 1년차 미만 + 직원 3명 + 매출 미검증 = T2 풀스코프 흡수력 미확정. 사명·도메인 분산 정리만으로도 T1 진단 가치 명확. 진단 후 카테고리·예산 흡수력 검증 후 T2 빌드 승급.
  • 대안:
  • 미팅 결과 매출 검증 + 자금 흡수력 명확 = T2 Build (2,500~4,000만 / 3개월) — 브랜드 정체성 정리 + 콘텐츠 시스템 + 자사몰 ASO 풀패키지
  • 미팅 결과 자금 매우 슬림 = T0 진단형 (200~300만 / 2주) — 디스커버리 + 사명·도메인 의사결정 한 장 보고서
  • 단계적 접근(권장): 본 건 강력 권장. 신생 1년차 미만 + 직원 3명 + 흡수력 미검증 = T1 진단(4주) 후 결과 보고 T2 승급. 모범 케이스: wearwhere · DAYNESTA 동형.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
