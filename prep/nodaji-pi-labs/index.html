ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 참고 케이스 풀에 정확한 매칭 부재 — HIZ 표준 D2C SOP는 한국 단일 시장 + 물리 제품·콘텐츠 카테고리 전제. Web3 + 글로벌 한류 + EduFi 3중 카테고리는 신규 영역. 그러나 정식 리서치 v2에서 핏 검증 완료(GO 4.0).
  • 와이림 / 위타민(우르실): 1년차 미만·인력 슬림·외부 협업 경험 부재라는 운영 구조에서 인접. 단 와이림은 자사몰 부재 가전 셀러, 위타민은 D2C 건기식 — 카테고리 차이 큼.
  • 마키노차야: 가맹 모집 단계 = 외부 채널 확장 압축 일정이라는 점에서 작업 톤 인접.
  • HIZ 풀 안에서 가장 가까운 매칭은 본 브랜드 자체 — 정식 리서치 v2가 첫 케이스, 향후 Web3·글로벌 토큰 프로젝트 들어올 시 본 케이스 SOP 적용 가능.

CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(3중 카테고리 한 줄 메시지) + 콘텐츠(시드 15개) + 채널(X · 유튜브 쇼츠 · 텔레그램 시드) + PR/바이럴(KOL 30명) 4축 풀패키지 핏. 데이터 대시보드는 런칭 이후 D+30 이후 검토.

  • 30일(D-50 ~ D-20): 5단 정체성 아키텍처 확정 + 3중 카테고리 한 줄 메시지 5안 → 1안 결정 + 6채널 우선순위 → 시드 채널 3개 결정 + 영문 1-pager·BI v1
  • 60일(D-20 ~ Launch): 시드 콘텐츠 15개(한류 숏 5 + 학습 변형 숏 5 + 토크노믹스 설명 5) + KOL 30명 1차 시드(영어권 10 · 동남아 10 · 중화권 10) + X·유튜브·텔레그램 3채널 셋업·첫 점화
  • 90일(Launch+30): 런칭 24시간 콘트롤룸 운영 + 6개월 50만 가입자 KPI 분기 점검 + ROAS 자체 광고 플랫폼 v1 정렬 + 4채널(틱톡·인스타·디스코드·기타) 확장 결정
TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T2 Professional (3,000만 / 8주 압축런) — 정식 리서치 v2 결론 그대로
  • 근거: D-50 압축 일정 = 단발 진단(T1 500만/2주) 불가 (진단만 하고 7월 론칭 실기). 풀스코프 T3(7,000만+/16주) = 시드 라운드 단계 예산 흡수력 미검증 + 시간 부족. T2 압축런이 시간·예산·범위 3변수 최적.
  • 대안:
  • 시드 라운드 자금 풀 입금되어 있고 글로벌 6채널·다국어 동시 진출 = T3 Scale(7,000만+/12~16주) 검토 가능. 단 HIZ 단독 풀스코프 어려움(글로벌 KOL 풀·다국어 카피라이팅 외부 파트너 필요)
  • 자금 라운드 미입금 + 미팅 결과 가용 예산 1,500만 이하 = T1 Essential(1,500만/4주) 시드 채널 3개 + KOL 15명 + 콘텐츠 7개 미니 버전. 단계적 접근 옵션 없음 — D-50 압축 일정 = 한 번에 빌드 진입 필수.
  • 단계적 접근(권장 케이스 X): 본 건은 D-50 일정상 T1→T2 단계 분리 불가. 정식 리서치 v2도 압축 단발 권고. 미팅에서 시드 자금 입금 시점·시드 라운드 클로징 일정 확인 → T2 즉시 진입 가능성 판단.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+

정식 리서치 승급 시 우선 파야 할 영역

정식 승급 시 심화 영역

> 정식 리서치 v2는 이미 보유. 본 PREP은 미팅 후 v3 갱신을 위한 4영역 명시:

  • 자금 라운드 구조: 시드 규모·VC·전략 투자자·자금 사용 계획(마케팅 vs 개발 vs 법인 설립) + 클로징 일정 — v3에서 가장 중요한 정량 갱신 영역
  • 6채널 첫 점화 1개 결정: 미팅 청취 + 채널별 한류 팬덤 도달 데이터(X 점유율 · 유튜브 쇼츠 글로벌 K-pop 트래픽 · 텔레그램 Web3 커뮤니티) 추가 분석
  • 정부지원 트랙 매핑: 강원(콘텐츠진흥원·강원특별자치도 디지털 자산) · 중소벤처(글로벌 강소기업·딥테크) · 콘진원(K-콘텐츠 글로벌) · 소진공(소상공인) — 본사 강원 정선 = 강원 전용 트랙 적격성 매핑 + 제안서 동반 작성 옵션 검토
  • 법인·재단 구조 확정: 두바이 NodajiFi Foundation 설립 일정 + DAO 거버넌스 도입 시점 + ROC 지역 운영회사 우선 국가 — 토큰 발행·증권성 자문은 외부 법무 파트너(블록체인 전문 로펌) 필요
  • (선택) Web3·토크노믹스 법무 자문: Gem/NUSD 발행 시 한국 가상자산법·미국 SEC·EU MiCA 적용 시점 — HIZ 단독 어렵고 외부 법무 파트너 매칭 필수
