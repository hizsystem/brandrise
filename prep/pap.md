## 유사 케이스 인접도
- **참고 케이스 풀에 정확한 매칭 없음 — 핏 검증 필수.** HIZ 표준 D2C SOP는 단일 법인·단일 브랜드 전제. 대기업 사내 신사업·콘텐츠 IP는 신규 영역.
- **위타민(우르실) / 리솔츠**: 본사 자산 위에 새 브랜드 출범 단계 = 모회사↔하위 브랜드 분리/통합 의사결정 인접도 일부.
- **node.X / WAYD**: 콘텐츠 IP→커머스 확장 시도 인접도 있음. 단 본 건은 모회사가 이미 커머스 강자(KREAM) = 역방향(콘텐츠가 모회사 커머스에 기여) 구조.
- **참고 케이스 부재 영역**: 대기업 사내 브랜드 PRJ = HIZ 미경험. 본사 KPI·승인 라인·재무회계 정합성 = 외부 컨설팅 자율도 제한적, 사전 합의 영역 큼.

## 핵심 페인 (가설)
- **모회사 KREAM의 압도적 브랜드 자산 안에서 PAP만의 독립 정체성 확립 난이도 큼.** "KREAM 매거진" 인식 vs "독립 매거진 IP" 인식 사이 포지셔닝 미정 추정.
- **매거진 IP→수익화 로드맵 불명확.** 콘텐츠 자산은 다수 발간(Editorial·VROOM·Envio·Dystopia 등) 보유하나 커머스·이벤트·B2B(브랜드 협업·광고) 수익 모델 명시 부재 추정.
- **본사 KPI vs 브랜드 장기 자산화 사이 긴장.** 사내 신사업 특성상 단기 트래픽·전환 KPI 요구 vs 매거진 IP 장기 빌드업 사이 자원 배분 갈등 가능.
- **팀장 결재 권한 범위 불명.** 마케팅 의사결정·예산 집행·외부 컨설팅 계약 = 본사 임원·재무·법무 승인 필요할 가능성. 결재 라인 청취 1순위.
- **외부 자산화 권한 제한.** 본사 소유 IP를 외부 컨설팅 산출물로 표준화·공개 가능한 범위 불명. NDA·산출물 귀속 사전 합의 필요.

## 추정 Tier 견적 범위
- **추천 Tier**: **T1 Foundation (1,000~1,500만 / 4주) 진단 우선**
- **근거**: 대기업 사내 PRJ = 본사 결재 라인·승인 영역·산출물 귀속 등 사전 합의 영역 큼. 풀스코프 진입 전 진단·합의 단계 필수. 매거진 IP 자산은 풍부하나 **외부 컨설팅 자율도·예산 흡수력 미확정**.
- **대안**:
  - 본사 결재 라인 명확 + 외부 컨설팅 자율도 보장 + 연간 마케팅 예산 큰 경우 = **T2 Build (3,000~5,000만 / 3개월)** 콘텐츠 시스템 구축 + 채널 확장 + 수익화 v1
  - 본사 KPI 단기 트래픽 강하게 요구 + PAP 독립 정체성 불필요 판정 = **T0 진단형 (200~300만 / 2주)** 1장 자문 보고서로 종결
  - 매거진→커머스·이벤트·B2B 풀스택 확장 = **T3 Scale (7,000만+ / 6개월+)** — 단 HIZ 단독 어렵고 모회사 KREAM 인프라 파트너십 필수
- **단계적 접근(권장)**: **본 건 권장.** 결재 라인 미확정 + 외부 자산화 권한 제한 가능성 = T1 진단부터, 본사 합의 후 T2 승급.
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
