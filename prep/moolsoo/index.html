한국 천연 섬유(모시·실크) 기반 샤워타월 D2C 브랜드. "사라져가는 한국 전통 섬유를 일상 케어 제품으로 부활시킨다"는 브랜드 미션, "밀크 타올·솔트 타올" 2종 라인. 와디즈 단독 펀딩(2025.7.31~8.31, 31,200원 가격대)으로 D2C 런칭 진행. USPTO에 "MOOLSOO" 상표 출원(출원인: Suji KANG, 신청자 강수지 대표 본인과 일치 추정) → 글로벌 진출 의지.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 전통 섬유 = 설명 난이도 높음 + 마이크로플라스틱·환경 메시지 = 가치 소비 타겟 정의 필요 — "왜 모시·실크 샤워타월인가"의 카피·콘텐츠 시스템 미정착 가설
  • 와디즈 펀딩 종료 후 D2C 자사몰·정기구매 전환 미흡 가능성 (gmail 이메일 = 자사몰 인프라/CRM 부재 시그널)
  • 글로벌(미국 USPTO 출원) 의지는 있으나 국내 D2C 발판이 약한 상태에서 글로벌 동시 진행 시 자원 분산 리스크
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

  • 6영역 중 브랜드 정체성(전통 섬유 스토리텔링) + 콘텐츠(가치 소비 카피·릴스) + 채널(자사몰·인스타 정착) 핏 매우 높음. 퍼포먼스는 자사몰 인프라 정착 후 단계
  • 30일: 브랜드 메시지·타겟 정의 + 와디즈 펀딩 데이터 분석 / 90일: 인스타·자사몰 콘텐츠 시스템 + 정기구매·앵콜펀딩 v1 / 180일: 퍼포먼스 ROAS 안정화 + 미국 D2C v1
  • GO 가능성 중~높음 — 케어비네스트/위타민 케이스 유사(D2C + 가치소비 + 글로벌 진출)
