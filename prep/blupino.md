- **180일**: 해외 채널(영·일·중) 우선순위 1개 픽 + 자사몰 ROAS 안정화 + B2B(카페·F&B) 채널 매뉴얼화

## 추정 Tier 견적 범위
- **추천 Tier**: **T1 Foundation (500~1,500만 / 4주) 진단 → T2 Build (2,000~3,500만 / 3개월) 단계적 승급**
- **근거**: Early Seed + 매출·인력 미공시 + 사명 통합 의사결정 선행 필요 = **T1에서 정체성·채널 진단 우선, 결과 보고 T2 빌드**. T2 즉시 진입 시 사명 분산 상태로 빌드 = ROI 낮음.
- **대안**:
  - 매출 규모·예산 흡수력 검증되고 글로벌 동시 진출 의지 강하면 → **T2 Build (3,000~5,000만 / 3~4개월)** 즉시 진입
  - 자금 매우 슬림 = **T0 진단형 (200~300만 / 2주)** 정체성 1장 보고서
  - 정부지원(농식품부·중기부 청년창업·소진공) 트랙 동반 옵션 검토
- **단계적 접근(권장)**: **본 건 권장.** Early Seed + 사명 통합 선행 필요 = 2단계 분할이 합리적.
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
