PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 신규 매장 인지도 0 → 동네 유입·재방문 만들기가 최우선 과제. 인스타·네이버 플레이스·당근 등 로컬 채널 정착 미진 가설
  • 젤라또 시장 = 경쟁 극심(말라카이트·Maman Gelato·Scooper·glt젤라또·설악젤라또 등 인스타 인지 강자 다수) → 차별점(원료? 위치? 콘셉트?) 정의·노출 미진 가설
  • 만 23세 1인 창업자 = 매장 운영·제품 개발·마케팅 동시 운영 = 마케팅에 쓸 시간/자원 절대 부족 가설
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

  • 6영역 중 브랜드 정체성(매장 콘셉트·페르소나 정의) + 채널(인스타·네이버 플레이스 셋업) + 콘텐츠(릴스·매장 영상 시스템) 핏. 단 F&B 단일 매장 = 컨설팅 예산 절대 규모 작음 예상
  • 30일: 매장 콘셉트·타겟·차별점 진단 + 인스타·네이버 플레이스 v1 / 90일: 릴스·로컬 인플루언서 시드·재방문 쿠폰 시스템 / 180일: 2호점·납품·온라인 굿즈 등 확장 옵션
  • PASS 또는 매우 소형 견적(Tier 0 진단형 100~300만)만 매칭 — F&B 단일 매장 + 청년 창업자 예산 규모는 HIZ 표준 견적과 미스핏 가능성 높음. 단, 장기 IP·다점포 비전 있으면 GO 재검토
