PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 사명 충돌 1순위 페인 — "눈사랑안과" = 인간 안과 다수, "안과동물병원" = 카테고리 선두 다수 → 검색·브랜드 자산 분산
  • 의료광고법 제약 = 일반 D2C·퍼포먼스 마케팅 그대로 적용 불가 (보건복지부·수의사회 가이드)
  • 카테고리 차별화 부재 가설 — "안과 전문" 동물병원이 이미 다수, 후발이라면 진료 차별점 정의 필요
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

  • ⚠️ 핏 검증 필요 — HIZ는 D2C 브랜드(뷰티·웰니스·건기식·생활가전) 중심. 수의병원은 SOP 비적합 가능성
  • 적용 가능한 영역: 브랜드 정체성(네이밍·포지셔닝 차별점) + 콘텐츠(보호자 교육·증례 콘텐츠 시스템) + 로컬 SEO·블로그·플레이스 운영
  • 비적용/조심 영역: 퍼포먼스 광고(의료광고법 심의), 인플루언서 캠페인 형식 다수, 일반 ASO
  • 30일: 사명 충돌·카테고리 차별점 진단 / 90일: 보호자 콘텐츠 라인 1개 + 플레이스·블로그 SEO / 180일: 증례·후기 시스템화 — 단, 미팅 후 핏 결정
