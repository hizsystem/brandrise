현재 추정 단계

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 브랜드명 검색 시 웰리스 의료 플랫폼·웰리스 공기청정기·웰버 약국·웰니스약국 등 헬스/약국 카테고리 동음 브랜드 다수에 묻히는 SEO/브랜드 식별 문제 즉각 직면
  • 약사 창업 헬스케어 D2C는 카테고리 자체가 의약외품·식약처 규제 + 광고 심의(KFDA·의약품 광고 규정) 이슈로 콘텐츠·퍼포먼스 마케팅 룰셋이 일반 D2C보다 까다로움 — 전문성 필요
  • 약사 1인 운영 가설 시 콘텐츠/채널 운영 리소스 부족, 약사 본업과 마케팅 병행 어려움
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

  • 6영역 중 브랜드 정체성(약사 전문성 시그널·USP 명확화) + 채널(약사 콘텐츠 기반 인스타·유튜브 0→1) + 콘텐츠(약사 전문성을 소비자 언어로 번역) + 운영(의약품 광고 심의 대응) 핏 매우 높을 가능성.
  • 우리의 케어비네스트(건기식)·바디노트(웰니스 구독) 케이스와 매우 인접한 구조 — 약사 신뢰자산 + D2C 헬스케어는 HIZ 강점 영역과 정확히 매칭.
  • 30일: 약사 전문성 진단 + 카테고리·USP·심의 룰셋 정의 / 90일: 약사 본인 IP 콘텐츠 시스템(인스타·유튜브 숏폼) + 자사몰 ASO / 180일: 퍼포먼스 ROAS 검증 + 약국 채널 + D2C 통합 전략
  • GO 가능성 중상 (조건부) — 약사 면허·법인 등록·실제 제품/서비스 카테고리 확인 시 GO 가능. 미팅 시 카테고리·매출·예산 즉시 확인 필수.
MEETING QUESTIONS

미팅 시 확인 질문

사전상담 도입 5분 어젠다 + 정량 진단

1
주식회사 웰리스온의 정확한 사업 카테고리는? (건기식·일반의약품·의약외품·헬스케어 서비스·약국 플랫폼?) 신민경 대표님은 약사 면허 보유자가 맞나요?
2
현재 제품/서비스 SKU 수와 약국·온라인몰·자사몰 등 판매 채널 구성은? 월 매출 규모는?
3
의약품/건기식 광고 심의 경험과 현재 대응 체계는? (식약처·KFDA 광고 심의 사전 통과 여부)
4
약사 본인 IP(인스타·유튜브)를 활용한 콘텐츠 마케팅 의향이 있으신가요? 본업과 병행 가능 시간은?
5
컨설팅 후 6개월 기대 1순위 성과 — D2C 매출? 약국 채널 확장? 약사 IP 인지도?
