ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 리솔츠 (가장 가까움) ★ — 뷰티 디바이스 D2C + 자체 R&D + 자사몰 + 시드/Pre-A 직후 + 글로벌 진출 모색 = 거의 1:1 매칭. HIZ 리솔츠 컨설팅 SOP 그대로 적용 가능.
  • 위타민(우르실) — 자사 디바이스+이너뷰티 융합 + 시드 단계 + 자사몰 + 글로벌 = 인접도 높음.
  • 케어비네스트 — 와디즈 펀딩 + 자사몰 정착 + D2C = 펀딩→정착 단계 인접.
  • 노다지파이랩스 / 와이림 — 슬림 본사 + 빠른 스케일 시도 = 운영 구조 인접 일부.
  • 요약: HIZ 컨설팅 풀에서 리솔츠/위타민 모델 = 동일 카테고리 + 동일 단계 = 가장 자신 있는 핏. 컨설팅 진입 마찰 최저.
PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • B2C 디바이스 런칭 직전 → 자사몰·인플루언서·콘텐츠 시스템 전면 미정착. dualtetra.co.kr 자사몰 + 와디즈 펀딩 + 네이버 쇼핑라이브 단발 = 자사몰 ASO·랜딩·CRM·재구매 시스템 부재 강한 추정. 인스타·유튜브 본 계정 운영 빈도·인게이지먼트 미확인 = HIZ 1순위 진단 영역.
  • 5중 초음파·복합 기술 = 소비자 베네핏 언어 번역 부재. "DualPoration™" "IntraSonix™" "WaveSync™" 기술 카피 = 일반 소비자 이해 어려움 → 기술→베네핏 카피 시스템·콘텐츠 시각화 시급. 와디즈 슈퍼얼리버드 50% 할인 의존 = 정가 정착 전 정체성 고정 필요.
  • 글로벌(일·중화권·북미·유럽) 동시 진출 = 다국가 브랜드 일관성·현지 채널 부담. 토요타 통상 매칭만으로는 현지 D2C 실제 매출 보장 안 됨. 한국 자사몰 → 일본 자사몰/Qoo10·중화권 Tmall·북미 Amazon 매핑 SOP 미정.
  • 두뇌케어(SonoBrain) + 스킨케어(DualTetra) 듀얼 라인 = 브랜드 아키텍처 정리 시급. "큐라펄스 마스터" vs "듀얼테트라/SonoBrain 독립" 의사결정 미정 추정.
  • 부산 본사 + 안산 운영지 분산 = 인하우스 16명 운영 효율 추정. 빠른 채용 확장(1년 +20명) = HR·운영 시스템 동시 압박.
