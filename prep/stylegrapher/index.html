PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 대표 1인 IP 의존 = 매출 시간 천장 명확 (1순위 페인) — 콘텐츠 제작·스타일링·운영 모두 본인 시간 → 스튜디오·1:1 컨설팅 모델은 시간 제약 절대값. 시스템화·온라인 확장 없이는 매출 X배 불가능
  • 온라인 비즈니스로의 확장 미진 — AI 분석 라인이 있으나 온라인 클래스/패키지/디지털 상품화로 확장 시 추가 매출 레버리지. 본인 IP × 디지털 상품 = 시간 천장 해체 핵심
  • 채널 다변화 필요 — 인스타 단일 유입 채널 의존 가설. 웨딩·취업·기업 임원 케어 등 B2B 채널 신규 개척 영역
  • 개인 IP ↔ 브랜드 정체성 결합 — 이사금(개인) ↔ 스타일그래퍼(브랜드) 결합 IP 전략 미정착 가능성. 본인 콘텐츠가 본인 브랜드 자산으로 축적되는 시스템 설계 필요

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T2 Build (3,000~5,000만 / 4~6개월) 직입
  • 근거: 1인 IP 자산이 이미 강함(노스페이스 MD + 숙명여대 의류학 + 4대 서비스 라인 + 멀티 채널) = 진단(T1)만으로는 레버리지 낮음. 시스템화·패키지·온라인 확장 빌드가 곧 매출 임계점 돌파의 핵심 = T2 풀스코프 직입 권장. Tier A 케이스(사이즈 작지만 핏 명확).
  • 대안:
  • 매출·예산 흡수력 미검증 우려 시 → T1 Foundation (1,000~1,500만 / 4주) 진단 후 T2 승급 (보수 옵션)
  • 본인 시간 흡수력 확보되고 동시 글로벌 IP 확장 의지 강하면 → T3 Scale (7,000만+ / 6~12개월) 검토
  • 단계적 접근: 본 건 권장하지 않음. 1인 IP 자산 이미 강함 + 시스템화 빌드가 곧 핵심 의제 = 진단 분할 비효율.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
