한 줄 요약

사업 본질 한 줄

대표 박영(약 10년 국내외 서브컬처 씬 아트워크 경력)이 운영하는 라이프스타일 브랜드. 비대칭 핸드메이드 세라믹·테이블웨어를 중심으로 시그니처/베이직/에디션/협업 라인 운영. 자사몰 + 인스타그램(@youngeyedproject) + 29CM + CJ온스타일 라이브커머스 채널 가동, "2026년 가을 아트 플랫폼 확장" 계획 명시.

STAGE

추정 단계

현재 추정 단계

초기~성장 단계 — 사업자등록 2023년(통신판매업 2023-서울마포-0086) = 2~3년차. 자사몰 + 29CM + CJ온스타일 멀티채널 진입 = 단순 1인 공방을 넘어선 단계. 한국일보 보도(2024년 9월) 노출 = 미디어 인지도 확보 중. 인스타 팔로워는 검색 결과 기준 6,720명(보강 검증 필요). 매출/인력 공개 정보 미확보.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 신청자가 마케터(이세빈) = 대표(박영)와 별도 → 의사결정·예산 권한 라인 미팅 시 필수 확인. "대표 동의 필요"로 컨설팅 진행 지연 리스크
  • 라이프스타일 + 아트 컬래버 브랜드 = "감성/취향" 중심 → 퍼포먼스 광고 ROAS 계산이 어려운 카테고리, 채널 ROI 측정 시스템 부재 가능성
  • 29CM·CJ온스타일·자사몰 등 멀티채널 운영 = 채널별 톤·SKU·가격 정책 통합 부담. 브랜드 일관성 vs 채널별 최적화 균형 문제
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(아트 플랫폼 확장 = 차세대 포지셔닝 재정의 기회) + 콘텐츠(협업 아티스트 스토리 자산화) + 채널(인스타 그로스 + 자사몰 vs 외부몰 균형) + 데이터(채널별 매출/CAC 대시보드) 4영역 적합. 박영 대표 본인이 콘텐츠 크리에이터 강점 = 콘텐츠 시스템화하면 폭발력 큼.

  • 30일: 브랜드 진단(아트 플랫폼 확장 콘셉트 구체화) + 채널 매출/CAC 진단 + 인스타·자사몰·외부몰 갭 분석
  • 90일: 협업 아티스트 콘텐츠 시스템 + 인스타 그로스(릴스·콜라보 시드) + 자사몰 KPI 정착
  • 180일: 2026 가을 아트 플랫폼 런칭 사전 빌드(브랜드 메시지·랜딩·PR) + 채널 ROI 대시보드

GO 가능성 중상 — 카테고리(라이프스타일·세라믹·아트)는 HIZ가 직접 보유한 케이스가 적으나, "콘텐츠 자산화 + 채널 최적화" 방법론은 적용 가능. 의사결정 라인(마케터 신청 vs 대표 박영 동의)이 가장 큰 변수.

MEETING QUESTIONS

미팅 시 확인 질문

사전상담 도입 5분 어젠다 + 정량 진단

1
박영 대표님 동의·결재 단계 — 컨설팅 도입 시 의사결정자는 누구이며, 이세빈 마케터님의 권한 범위는 어떻게 되나요?
2
현재 자사몰 / 29CM / CJ온스타일 매출 비중과 채널별 마진율은? 자사몰 비중 확대 계획이 있나요?
3
"2026 가을 아트 플랫폼 확장" 구체적 내용 — 신규 라인? 오프라인 공간? B2B/도매? 마케팅 우선순위는?
4
컨설팅 종료 6개월 시점 1순위 KPI (자사몰 매출? 인스타 팔로워? 아트 플랫폼 런칭 사전 PR?)?
5
(선택) 협업 아티스트 풀은 어떻게 확보하고 계신가요? 매월 신규 1팀 유지 가능한 구조인가요?
