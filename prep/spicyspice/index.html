PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 글로벌 SEO·도메인·SNS 핸들 점유 경쟁 = "spicyspice" 키워드 = 일본 식당·미국 LLC·TikTok 개인 계정 다수 점유. 영문 사명 유지 시 도메인·핸들 확보 비용 큼 + 한국어 검색도 인덱싱 0건 = 0→1 단계.
  • "global" 의지 vs 실제 채널 갭 — 이메일 핸들 글로벌 표기 명확하나, 영문 자사 채널·아마존·해외 SNS 일체 미확인 = 의지와 실행 사이 큰 갭.
  • 카테고리 미확정 = 청양 지역 + spice 네이밍으로 청양고추 가공 가설은 합리적이나 외부 식별 출처 0건 = 사업 본질 자체가 외부에 미공개. SKU·매출·해외 진출 진척 모두 미확인.
  • 자사 채널 0 상태 = 자사몰·인스타·페이스북·블로그·언론 모두 0건. 마케팅 인프라 자체가 처음부터 셋업 필요.
