- **케어비네스트**: 건기식 D2C + 단일 카테고리(면역·관절) = 요프리 Rh2 B2C 부분만 인접도 높음. R&D·B2B 소재 부분은 무관.
- **위타민(우르실)**: 건기식+뷰티 D2C = B2C 부분 인접. R&D 자체 자산은 위타민이 더 약함.
- **루키노바**: 페르소나 기반 라인 + 의학적 백그라운드 = 요프리 Rh2 방광 건강 니치 콘텐츠 시스템 인접도 일부.
- **참고 케이스 부재 영역**: B2B 소재·기능성 원료 플랫폼 = HIZ 미경험. 제약·바이오 R&D 자산 활용 마케팅은 별도 외부 파트너(바이오 PR·CRO 매칭) 필요 가능성.
- **요약**: B2C 요프리 Rh2 단독 풀스코프 = HIZ 핏 양호 / B2C+B2B 동시 풀스코프 = 핏 거리 있음, 미팅 시 스코프 압축 필요.

## 핵심 페인 (가설)
- **기술·소재 설명 난이도 극상.** AI 활성물질 분석 · 희귀 진세노사이드 · 프리미엄 제형 = R&D 자산 풍부하나 **B2C 소비자 언어로 번역** 부재. "왜 비싼지·왜 효과 있는지·경쟁 진세노사이드 제품 대비 차별점" 1줄 명제 부재 가능성.
- **요프리 Rh2 = 방광 건강 니치 카테고리.** 타깃: 40대+ 여성·요실금·과민성 방광 의심군. **검색 수요 작고 경쟁자는 일반 크랜베리·호박즙 OTC + 약국 의약품(베타미가·디트로판 등).** 카테고리 진입 비용 큼.
- **B2C 단일 SKU 의존도** — 7대 기술·다양한 진세노사이드 보유에도 B2C 제품은 요프리 Rh2 1종. 차세대 SKU 파이프라인 미공개 = 매출 다각화 부담.
- **김종우 CSO ↔ 한종서 대표 의사결정 라인 = 1순위 리스크.** CSO·Founder 신청 = 마케팅 의사결정 권한 있을 가능성 / 단 컨설팅 계약·예산 결재는 대표 결재 가능. **미팅에서 한종서 대표 동행 여부 1순위 청취**.

| 5 | **운영 흡수력** | ⚠️ 미검증 + 강원 춘천 입지 제약 — 인하우스 마케팅·콘텐츠 인력 수 미공개. 춘천 본사 = 서울 콘텐츠 에이전시·인플루언서 풀 접근성 제약 → HIZ 원격 협업 가능하나 촬영·미팅·KOL 모집 시 출장 부담. B2B vs B2C 운영 인력 분리 여부 청취 필요 |

## 미팅 시 확인 질문 5개
1. **(1순위)** **한종서 대표와 김종우 CSO·Founder 역할 분담** — **브랜드/마케팅 의사결정과 예산 집행 권한 누구**? 컨설팅 계약 주체 한종서 대표 직접 결재? **한종서 대표 동행 미팅 별도 필요 여부**?
2. **요프리 Rh2 매출·MAU·재구매율·인스타 팔로워**? 단일 SKU 의존도 + **차세대 SKU 파이프라인 (다른 진세노사이드 응용 제품)**과 런칭 시점?
3. **B2B 소재·기능성 원료 매출 비중 vs B2C 요프리 매출 비중**? 컨설팅 스코프 B2C 한정? B2B 포함? B2B 포함 시 별도 비용 구조 필요
4. **최근 6개월 인스타·검색광고·인플루언서 시도 결과** — CAC·전환·ROAS? **인하우스 마케팅·콘텐츠 인력 수**? 강원 춘천 본사 → 서울 출장/원격 협업 가능 구조?
5. 컨설팅 후 6개월 1순위 KPI — **요프리 단일 매출 ○배** / **브랜드 인지도** / **B2B 채널 진입(병의원·약국)** / **차세대 SKU 런칭** 중? **가용 월 예산 범위**? **시리즈 A 라운드 시점**·자금 사용 계획?
