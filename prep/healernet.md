# 힐러넷 · 사전상담 PREP

## 추정 단계
**시드~초기** (추정·미확인) — 강원 춘천 소양강로 소재(춘천 바이오 특화단지 인접), 사업자번호 608-86-13545. 자사 도메인 + B2C 제품 1종(요프리 Rh2) 런칭 단계로 보이며, 매출/투자 공개 정보 미확보. 진세노사이드 Rh2(항암·면역 강화 효능 보고 성분)·CK 등 고부가 소재 보유 = R&D 비중 큰 바이오 스타트업 패턴.

## 핵심 페인 (가설)
- 기술/소재(AI 활성물질 분석·희귀 진세노사이드·프리미엄 제형) = 설명 난이도 극상. **B2C 소비자 언어 번역**(왜 비싼지·왜 효과 있는지·경쟁 진세노사이드 제품 대비 차별점) 부재 가능성
- "요프리 Rh2"는 **방광 건강**이라는 매우 좁은 니치(타깃: 40대+ 여성·요실금·과민성 방광 의심군) — 카테고리 진입 자체가 어려움(검색 수요 작고 경쟁자가 일반 크랜베리·호박즙 OTC)
- 강원 춘천 본사 = 서울 콘텐츠/광고 에이전시 접근성·인플루언서 풀 활용 어려움 가능성

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
6영역 중 **브랜드 정체성(기술→소비자 베네핏 번역) + 콘텐츠(헬스 카피·증언 시스템) + 채널(방광·요실금 카테고리 SEO/SNS) + 퍼포먼스(니치 키워드 ROAS)** 4영역 모두 1순위 적합. 다만 의료적 클레임 규제(건기식·일반식품 가이드라인) 필터링 필수.

- **30일**: 브랜드 정체성(Rh2·CK 가치 번역) + 요프리 Rh2 D2C 채널 진단 + 경쟁 분석(크랜베리 류 카테고리)
- **90일**: 인스타·블로그·검색광고 콘텐츠 시스템 + 마이크로 인플루언서(40대+ 여성 헬스) 시드 30~50명
- **180일**: 퍼포먼스 ROAS 안정화 + 사용자 증언 라이브러리 + B2B/제휴(병의원·약국) 채널 가설 검증

**GO 가능성 중상** — 케어비네스트/위타민과 유사 헬스 D2C 구조이나, ① 카테고리 니치(방광 건강) ② 신청자 직책(CSO·창업자) vs 대표(한종서) 분리 = 의사결정·예산 결정권 미팅 시 명확화 필수.
