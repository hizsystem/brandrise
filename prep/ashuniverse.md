## 유사 케이스 인접도
- **마키노차야 (가장 가까움)**: 대표 개인 스토리·교육 IP·소-중 규모 자생 성장 = 단계·구조 인접. 마키노차야가 F&B 매장 중심이라면 아슈니버스는 D2C 헬스푸드 중심.
- **레베로(LEVERO)**: 지속가능·웰니스 D2C + 대표 개인 가치관 출발 = 톤 인접.
- **케어비네스트 / 절대콜라겐**: 건기식·헬스푸드 D2C + 자사몰 자산 = 비즈모델 인접. 단 아슈니버스는 커뮤니티·교육 IP가 핵심 차별점.
- **네츄럴굿띵스**: "여성 웰니스" 카테고리 인접 (임신성 당뇨·저당 = 여성 건강 영역). 단 NGT는 매출 169억 카테고리 1위 = 아슈니버스보다 한참 위 단계.
- **요약**: HIZ 풀에서 **대표 개인 IP → 브랜드 시스템 전환**이 핵심 의제인 케이스는 마키노차야 + 아슈니버스 = HIZ 강점 영역.

## 핵심 페인 (가설)
- "교육형 커뮤니티 = 깊지만 좁다" — 챌린지·교육 콘텐츠 강점이 광범위 인지도·매출 스케일로 연결되는 구조 부재 가능성
- 대표 개인 IP(스토리/경험) 의존도가 높아 시스템·표준화·후임 운영 체계 약할 가능성
- 저당 카테고리 경쟁(닥터키친·풀무원녹즙·랭킹닭컴 등) 대비 브랜드 차별화는 명확하나 채널·퍼포먼스 자산화 미흡 추정
- 100일 챌린지 = 강력한 자산이지만 챌린지 → 정기구독·재구매 전환 퍼널 시스템화 미정 추정
- LTV·재구매율·커뮤니티 회원수 등 데이터 트래킹 구조 미정 추정 (gmail 운영 = 데이터 인프라 약함 시그널)

## 추정 Tier 견적 범위
- **추천 Tier**: **T1 Foundation (1,000~1,500만 / 4주) → T2 Build (2,000~3,000만 / 3개월) 단계적 승급**
- **근거**: 커뮤니티·교육 IP는 자산화되어 있으나 시스템화·데이터 트래킹 미정 = 진단(T1) → 본격 빌드(T2) 단계 분리 필요. 매출 규모·예산 흡수력 미검증 = 풀스코프 즉시 진입 불가. **개인 IP → 브랜드 시스템 분리는 시간·합의 비용 큰 작업** = 4주 진단으로 방향성 확정 후 빌드.
- **대안**:
  - 미팅 결과 매출·자금 흡수력 충분 시 → T2 Build 2,000~3,000만/3개월 즉시 진입
  - 100일 챌린지 단발 캠페인·시즌 운영 단위 검토 시 → T0 진단형(100~300만/2주) + 단발 워크숍
- **단계적 접근(권장 케이스 ★)**: 본 건은 **T1 진단 → T2 본격 빌드** 단계 분리가 정석. 대표 개인 IP 분리는 4주 합의 작업 필요. 풀스코프 즉시 진입 = 과부담 위험.
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
