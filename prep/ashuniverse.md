# 아슈니버스 (Ashuniverse) · 사전상담 PREP

## 한 줄 요약
"설탕없는 세상(OUT SUGAR + UNIVERSE)" — 임신성 당뇨·전당뇨 개인 경험에서 출발한 저당/저탄수 헬스푸드 브랜드. 책 300권 + 실경험 기반의 "교육형 커뮤니티 브랜드" 포지셔닝, '아슈 다이어트 100일 챌린지' 운영.

## 추정 단계
**초기~성장기 진입** — 브랜드 스토리·커뮤니티(100일 챌린지) 자산은 명확히 구축됨. 다만 도메인(.co.kr) + gmail 사용 + 검색 노출 패턴으로 보아 **법인 규모는 아직 소-중 사이즈** 추정.

## 핵심 페인 (가설)
- "교육형 커뮤니티 = 깊지만 좁다" — 챌린지·교육 콘텐츠 강점이 광범위 인지도·매출 스케일로 연결되는 구조 부재 가능성
- 대표 개인 IP(스토리/경험) 의존도가 높아 시스템·표준화·후임 운영 체계 약할 가능성
- 저당 카테고리 경쟁(닥터키친·풀무원녹즙·랭킹닭컴 등) 대비 브랜드 차별화는 명확하나 채널·퍼포먼스 자산화 미흡 추정

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
- **6영역 중 약점 추정**: 채널(IG/YT 시스템화) / 콘텐츠(개인 IP → 브랜드 시스템) / 퍼포먼스(챌린지 → 유료 전환 퍼널) / 데이터(LTV 측정)
- **30일**: 커뮤니티·챌린지 데이터 진단 + 개인 IP vs 브랜드 IP 분리 전략
- **90일**: 콘텐츠 시스템 표준화 + 챌린지→정기구독 퍼널 + 퍼포먼스 채널 테스트
- **180일**: LTV·리텐션 데이터 대시보드 + 카테고리 확장 로드맵
