## 유사 케이스 인접도
- **마키노차야 (가까움)**: F&B + 핸드메이드 + 아트 톤 = 라이프스타일 브랜드 인접도 높음.
- **오들리컴바인**: 라이프스타일 + 협업 아티스트 = 콘텐츠 자산화 패턴 인접.
- **루키노바**: 페르소나·라인 분기(시그니처/베이직/에디션) = 라인 시스템 인접.
- **DAYNESTA**: 4개 페르소나 라인 분산 = 라인 우선순위 압축 작업 인접.
- **W'GREEN**: 매체 자산 확장 + 아트 협업 = 글로벌 전개 시 인사이트 일부.
- **참고**: 핸드메이드·아트 협업 카테고리는 HIZ 직접 보유 케이스 드뭄. 단 "콘텐츠 자산화 + 채널 최적화" 방법론은 적용 가능.

## 핵심 페인 (가설)
- **신청자가 마케터(이세빈) = 대표(박영)와 별도** → 의사결정·예산 권한 라인 미팅 시 필수 확인. "대표 동의 필요"로 컨설팅 진행 지연 리스크 (1순위)
- **라이프스타일 + 아트 컬래버 브랜드 = "감성/취향" 중심** → **퍼포먼스 광고 ROAS 계산이 어려운 카테고리**, 채널 ROI 측정 시스템 부재 가능성
- **29CM·CJ온스타일·자사몰 등 멀티채널 운영** = 채널별 톤·SKU·가격 정책 통합 부담. **브랜드 일관성 vs 채널별 최적화** 균형 문제
- **박영 대표 본인이 콘텐츠 크리에이터 강점** but **콘텐츠 시스템화 부재 가능성** — 대표 1인 의존 콘텐츠 = 확장 한계
- **2026 가을 아트 플랫폼 확장 = 차세대 포지셔닝 재정의 시점** — 핸드메이드 세라믹 D2C에서 아트 플랫폼으로 비즈모델 전환 = 브랜드 메시지·랜딩·PR 사전 빌드 필요

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
6영역 중 **브랜드 정체성(아트 플랫폼 확장 = 차세대 포지셔닝 재정의 기회) + 콘텐츠(협업 아티스트 스토리 자산화) + 채널(인스타 그로스 + 자사몰 vs 외부몰 균형) + 데이터(채널별 매출/CAC 대시보드)** 4영역 적합. 박영 대표 본인이 콘텐츠 크리에이터 강점 = 콘텐츠 시스템화하면 폭발력 큼.

- **30일**: 브랜드 진단(아트 플랫폼 확장 콘셉트 구체화) + 채널 매출/CAC 진단 + 인스타·자사몰·외부몰 갭 분석
- **90일**: 협업 아티스트 콘텐츠 시스템 + 인스타 그로스(릴스·콜라보 시드) + 자사몰 KPI 정착
- **180일**: 2026 가을 아트 플랫폼 런칭 사전 빌드(브랜드 메시지·랜딩·PR) + 채널 ROI 대시보드

**GO 가능성 중상** — 카테고리(라이프스타일·세라믹·아트)는 HIZ가 직접 보유한 케이스가 적으나, "콘텐츠 자산화 + 채널 최적화" 방법론은 적용 가능. 의사결정 라인(마케터 신청 vs 대표 박영 동의)이 가장 큰 변수.

## 추정 Tier 견적 범위
- **추천 Tier**: **T1 Foundation (1,000~1,500만 / 4주) 진단 → T2 Build 2,500~3,500만 / 3개월 단계적 승급**
- **근거**: 2~3년차 + 멀티채널 입점 + 한국일보 노출 = 단가 흡수 가능성 있음. 단 신청자 ≠ 대표 = 진단 단계 거쳐 결재 권한·예산 라인 검증 후 T2 빌드 진입 안전.
- **대안**:
  - 미팅 결과 박영 대표 직접 결재 + 자금 흡수력 명확 = **T2 Build 즉시 진입(3,000~3,500만/3개월)** — 아트 플랫폼 사전 빌드 + 콘텐츠 시스템 풀패키지
  - 2026 가을 아트 플랫폼 풀스택 + B2B 도매·전시 런칭 = **T3 Scale (5,000~7,000만/6개월+)** 검토
  - 자금 슬림 = **T0 진단형 (200~300만 / 2주)** — 아트 플랫폼 콘셉트 1장 + 채널 ROI 진단
- **단계적 접근(권장)**: **본 건 권장.** 신청자 ≠ 대표 = T1 진단 통해 대표 박영 결재 권한·예산 확정 후 T2 승급.
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
