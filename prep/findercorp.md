## 핵심 페인 (가설)
- **카테고리 차별화 메시지 부족 가능성.** 후기에 "다이소 규조토는 흡수 1도 안 됨" 같은 비교 멘트가 있으나 **공식 브랜드 톤은 "오래도록 폭신하고 부드럽게" 정도 — 차별화 명제가 모호.** 3세대 R&D가 자산임에도 외부에 "왜 W'GREEN인가" 1줄 명제 미정
- **검색 가시성 약함.** "wgreen1993" 도메인 인덱싱은 자사몰 후기 페이지 위주 — 브랜드명 단독 검색 시 외부 인지 부족
- **카테고리 확장(슬리퍼·라운지웨어·홈악세서리) 시도는 있으나 메인 SKU(발매트) 의존도 추정 높음**
- **인스타 @wgreen_life 운영하나 팔로워·인게이지먼트 미확인** → 콘텐츠 빈도·후기 큐레이션·UGC 시스템화 여부 진단 필요
- **대표 직접 신청 + ceo@ 메일 = 의사결정 빠름, 예산 검토 권한 본인.** 긍정 신호

- **30일**: 브랜드 진단 — "3세대 라텍스+규조토"의 소비자 언어 번역, "휴식을 디자인하는 리빙브랜드" 카피의 구체화, 후기 1.5만+ 자산 인벤토리
- **90일**: 인스타·유튜브 숏폼 콘텐츠 시스템(언박싱·물흡수 ASMR·청소 루틴·후기 카드뉴스), 퍼포먼스 ROAS 측정 대시보드
- **180일**: 신규 SKU 확장(슬리퍼·홈 매트 외 욕실 솔루션 라인업) + 채널 다각화(쿠팡/29CM/오늘의집), 월 매출 1.5~2배 목표
- **Tier 권장**: **T2 Build 3,000~5,000만/3~4개월 가장 적합.** 1.5년 차 + 자사몰 + 후기 자산이 있어 빌드 흡수 가능. T1 진단부터 시작해도 좋음(500만/4주).
