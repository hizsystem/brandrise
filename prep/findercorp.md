- **차별화 메시지 부족** = "3세대 라텍스+규조토" R&D 자산은 있으나 외부 소비자 언어로 번역된 한 줄 명제 부재. 공식 톤 "오래도록 폭신하고 부드럽게" = 카테고리 일반 카피, 1세대(다이소)·해외 직구 대비 **왜 W'GREEN인가** 1문장 미정. 후기에 "다이소 규조토는 흡수 1도 안 됨" 같은 비교 멘트가 산발적 = 자산 자료는 있으나 시스템화 안 됨.
- **메인 SKU 의존도 추정 높음** — 발매트 외 슬리퍼·라운지웨어·홈악세서리 시도 흔적 있으나 매출 비중 확인 불가. **단일 SKU 천장**이 임박했을 가능성 — 신규 카테고리 확장 시스템·론칭 캠페인 필요.
- **검색 가시성 약함** — "wgreen1993" 도메인 인덱싱은 자사몰 후기 페이지 위주 = 브랜드명 단독 검색 시 외부 인지 부족. 인스타 @wgreen_life 팔로워·인게이지먼트 미확인 = 콘텐츠 빈도·UGC 시스템화 약함 추정.
- **외부 채널 다각화 미정** — 쿠팡·29CM·오늘의집 입점 여부 미확인. 자사몰 단일 채널 의존 시 트래픽 확보 비용 큼.
