PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 사업 본질이 외부 검색으로 식별되지 않음 = 가장 결정적 페인. 마케팅 담당자(황민영) 존재에도 검색·SNS 노출 0 = 콘텐츠·채널 운영의 시스템·방향성 부재 가능성 큼. "마케터는 있는데 마케팅이 안 됨" 패턴.
  • 사명 "와썹+하우스" 일반 어휘 결합 = 정체성·타겟 정의 모호 위험. "와썹" 캐주얼 톤 vs "하우스" 공간성 = 타겟 세대·페르소나·톤앤매너 정렬 부재 가능.
  • 개인 이메일(naver.com) 사용 = 브랜드 도메인 이메일 부재 = 외부 신뢰도·전문성 인식 약함. 법인 도메인·자사몰·공식 채널 전면 부재 추정.
  • 대표+마케터 2인 = 의사결정 속도·실행력 강점이나 인하우스 디자인·콘텐츠 풀세트 운영 불가 = 외부 파트너 의존 자연스러움. 컨설팅 산출물 운영 주체 합의 필요.
  • 사업자 형태·매출 단계 미확인 = 견적 흡수력 검증 불가. 미팅 청취 후 Tier 확정.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

사업 본질 미식별 = 가설 작성 한계 있음. 미팅 첫 5분 사업 본질 청취 후 가설 수립이 본질.

가정 시나리오: - A. F&B·공간 매장 = 마키노차야 SOP 인접, 30일 정체성·메뉴·공간 톤 정의 → 90일 인스타·블로그 콘텐츠 시스템 → 180일 가맹·확장 검토 - B. 쉐어하우스·임대 공간 = 인테리어·라이프스타일 콘텐츠 + 입주자 모집 퍼널 시스템화 - C. 라이프스타일 매장·D2C = HIZ 표준 D2C SOP 적용, 자사몰·인스타·인플루언서 시드 - D. 신규 런칭 전 단계 = MK 케이스 인접, BI·네이밍·MVP 채널 1개 셋업

공통 6영역 중 브랜드 정체성(타겟·톤·BI) + 콘텐츠 시스템(마케터 활용) 1순위. 30일: 정체성 정의 + 마케팅 SOP 점검. 90일: 콘텐츠 시스템 정착 + 핵심 채널 KPI. 180일: 퍼포먼스·데이터 가설 + 시즌·이벤트 운영.
