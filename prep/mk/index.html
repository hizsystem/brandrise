  • 노다지파이랩스: Pre-launch + 글로벌 채널 0 = 단계 인접도 일부.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 브랜드명·아이덴티티·포지셔닝 자체가 아직 미확정 (Zero-to-One). 사명 "MK" 확정 여부 + BI/CI / 타겟 페르소나 / 카테고리·SKU 모두 미정 추정.
  • 공동창업 2인 구조 = 역할 분담·의사결정·브랜드 비전 정렬 필요. 조은미 대표(예정)·강민주 이사(예정) R&R 미확정. 자본 비율·의사결정 권한 분담 미확정 가능.
  • 시장 진입 전략 미정. 어떤 카테고리·타겟·채널부터 진입할지 우선순위 부재 추정.
  • 자본·자금 라운드 미확인 = 견적 흡수력 검증 불가. 자기자본 / 외부 투자 / 정부지원 / 가족 자금 등 미확인.
  • 사명 "MK" 글로벌·국내 동명 다수 = 신규 브랜드 검색 점유 어려움. 사명 변경·서브 브랜딩 검토 1순위 의제 가능.
  • 법인 설립 직전 단계 = 법무·세무·자본 구조도 동시 정리 필요 = HIZ 단독 어려운 영역. 외부 법무·세무 파트너 동시 진행 권장.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

Pre-seed Zero-to-One = 워크샵형 진단이 본질. 사업 본질·카테고리·BI·네이밍·시장 진입 우선순위 등 전부 미확정 = 30/90/180일 가설보다 1~2주 압축 워크샵 SOP가 적합.

6영역 중 브랜드 정체성(Zero-to-One BI/네이밍/포지셔닝) + 채널(런칭 채널 우선순위) 우선. 콘텐츠·퍼포먼스·데이터·운영은 아직 시기상조.

  • 2주 워크샵 (T0): 사업 본질 워크샵(슈퍼세이브 SOP 응용) + 카테고리·타겟·페르소나 정의 + BI/네이밍 방향 5안 + 시장 진입 우선순위 1~3
  • 30일 (T1): 사업 본질 워크샵 + 브랜드 포지셔닝 도출 + BI/네이밍 확정 + 도메인·상표 출원 자문
  • 90일: MVP 채널(IG·스마트스토어·자사몰 등) 1개 셋업 + 핵심 카피·콘텐츠 v1
  • 180일: 런칭 + 초기 콘텐츠 시스템 + 첫 KPI 측정 체계

  • 자본 풍부 + 풀 런칭 풀스코프 = T2 Build (2,500~4,000만 / 3개월) BI + 자사몰 + 초기 콘텐츠 + 인플루언서 시드 시드
