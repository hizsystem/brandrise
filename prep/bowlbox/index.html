한 줄 요약

사업 본질 한 줄

"Fresh and Healthy Meal" 샐러드·라이스볼 직영·가맹 외식 D2C 브랜드 — 신지은 대표 운영, 한남본점·역삼역점 직영 + 가맹(Franchising) 모집 단계. 대표 메뉴 "라이스 보울"(쌀·현미·보리·귀리 잡곡밥 + 12시간 저온숙성 스테이크/치킨/슈림프/불고기/돼지/두부 토핑) + 샐러드 라인. 매장 단위 인스타 분산 운영(bowlbox_hannam·bowlbox50·역삼점 별도) = 본사 통합 디지털 자산 약함. 트립어드바이저·식신·망고플레이트 등 맛집 미디어 노출 보유. 페이스북 운영(bowlbox2019) = 2019년경 SNS 시작 추정 = 운영 5~6년차 외식 D2C, 가맹 본격화 시점.

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 마키노차야 (가장 가까움) ★ — F&B 매장 + 직영·가맹 모집 단계 + 본사 통합 브랜드 자산 강화 필요 = 거의 1:1 매칭. HIZ 마키노차야 컨설팅 SOP 적용 가능.
  • WTM(위타민→우르실) — D2C+오프라인 동시 운영 + 본 브랜드 정체성 재정의 시점 = 일부 인접.
  • LEVERO — 헬시 라이프스타일 정체성·SKU 매트릭스 정리 측면 인접.
  • 부족한녀석들(어프리데이) — F&B + 매장+유통 동시 + 가맹/B2B 진입 = 단계 유사 일부.
  • 요약: HIZ 컨설팅 풀에서 마키노차야 모델 = 동일 카테고리(F&B 외식 매장 + 가맹 모집) = 가장 자신 있는 핏. 단 마키노차야보다 매장 수·운영 규모 작음(2점 vs 마키노차야 다점포) = T1 단계적 접근 권장.
PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 매장 단위 인스타 분산 (검증). @bowlbox_hannam (한남점) + @bowlbox50 (별도) + 역삼역점 서브사이트 + 페이스북 bowlbox2019 = 본사 통합 디지털 자산 부재. 한남점 vs 역삼점 vs 본사 톤·콘텐츠·운영 주체 분산 = 가맹 모집 리드 ROI 떨어짐.
  • 가맹 모집 페이지 있으나 → 가맹 리드 끌어올 콘텐츠·퍼포먼스 시스템 부재 가능성. 외식 가맹 = 영업사원·박람회 의존 vs 디지털 인바운드 리드 = 보울박스는 디지털 인바운드 시스템 미정착 추정. 가맹 문의 페이지 단순 노출 vs 가맹점주 후보 콘텐츠(매출 시뮬·점포 후기·창업 케이스) 부재.
  • "건강식" 카테고리 경쟁(샐러디·스윗밸런스·슬로우캘리·F45·다이어트 도시락) 차별화 메시지 약함 가설. 샐러디 309점, 슬로우캘리 120점, 스윗밸런스 등 거대 가맹망 vs 보울박스 직영 2점 = 가맹점주가 보울박스를 선택할 이유 = "라이스 보울"(잡곡밥+토핑) 차별화 외 운영 시스템·마케팅 자산 약함 추정.
  • 신지은 대표가 역삼역점 직접 운영 = 본부 마케팅·가맹 영업 동시 핸들 부담. 본부 인하우스 마케팅·콘텐츠·가맹 영업 인력 거의 없음 추정.
  • 자사몰 bowlbox.co.kr = 단순 브랜드 소개 + 가맹 문의 페이지 수준 추정. 온라인 주문·배달·구독 D2C 채널 부재 = F&B 외식 D2C 다음 단계 미진입.

브랜드라이즈가 풀 수 있는 것

6영역 중 채널(통합 운영) + 콘텐츠(본사 브랜드 자산) + 퍼포먼스(가맹 리드) 1순위. 브랜드 정체성·BI는 일부 정착, 단 가맹 모집 시점 재정의 필요. - 30일 (진단): 본사 통합 인스타·키 메시지 정의 / 매장별 인스타(@bowlbox_hannam·@bowlbox50·역삼점) → 본사 통합 vs 매장 보조 구조 의사결정 / 가맹 모집 페이지 인바운드 리드 진단 / 경쟁 가맹(샐러디·슬로우캘리·스윗밸런스) 차별화 메시지 도출 - 90일 (시스템 구축): 본사 통합 인스타 본 계정 정상화(주 3~5회 발행) + 매장 콘텐츠 SOP(매장별 → 본사 큐레이션 시스템) + 가맹 리드 퍼포먼스 셋업(메타·네이버 키워드 "샐러드 가맹"·"라이스볼 창업") + 가맹점주 후보 콘텐츠(매출 시뮬·창업 케이스 인터뷰·점포 후기 카드뉴스) - 180일 (스케일): 가맹 성과 KPI 대시보드(월 가맹 문의 수·전환율·점포 수 증가) + 월 리뷰 시스템 + (선택) 배달·구독 D2C 채널 v1(쿠팡이츠·배민·자사몰 정기구독)

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (1,000~1,500만 / 4주) → T2 Build 승급 단계적 접근 ★ 강력 권장
  • 근거: 직영 2점 + 가맹 모집 시작 단계 + 매출·직원수 미공개 = 일반 T2 풀스코프(3,000~5,000만) 흡수력 미검증. T1 진단 + 본사 통합 인스타·가맹 리드 시스템 v1으로 시작, 결과 보고 T2 빌드 승급.
  • 단계적 접근 옵션 (강력 권장 케이스): T1(1,000~1,500만/4주) → T2(2,500~4,000만/3개월) 단계적 승급 — 매출 규모 + 신지은 대표가 역삼점 직접 운영 = 외부 컨설팅 동시 핸들 가능 여부 검증 필요 = 단계적 접근 가장 적합.
  • 대안:
  • 미팅에서 매출·자금 흡수력 충분하고 가맹 본격 확장 의지 강하면 → T2 Build(3,000~4,000만/3~4개월) 바로 진입 가능
  • 가맹 본격 확장 + 다점포(10~30점) 운영 시스템 + 배달·구독 D2C 풀스택 → T3 Scale(7,000만+/6개월+) — 단, 직영 2점 + 가맹 시작 단계에서는 과다 위험, 점포 수 10점+ 도달 후 권장
  • 진단 비용 부담 시 → T0 진단형(300~500만/2~3주) 단축 옵션
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
