## 유사 케이스 인접도
- **마키노차야 (가장 가까움) ★** — F&B 매장 + 직영·가맹 모집 단계 + 본사 통합 브랜드 자산 강화 필요 = **거의 1:1 매칭**. HIZ 마키노차야 컨설팅 SOP 적용 가능.
- **WTM(위타민→우르실)** — D2C+오프라인 동시 운영 + 본 브랜드 정체성 재정의 시점 = 일부 인접.
- **LEVERO** — 헬시 라이프스타일 정체성·SKU 매트릭스 정리 측면 인접.
- **부족한녀석들(어프리데이)** — F&B + 매장+유통 동시 + 가맹/B2B 진입 = 단계 유사 일부.
- **요약**: HIZ 컨설팅 풀에서 **마키노차야 모델 = 동일 카테고리(F&B 외식 매장 + 가맹 모집) = 가장 자신 있는 핏**. 단 마키노차야보다 매장 수·운영 규모 작음(2점 vs 마키노차야 다점포) = T1 단계적 접근 권장.

## 핵심 페인 (가설)
- **매장 단위 인스타 분산 (검증).** @bowlbox_hannam (한남점) + @bowlbox50 (별도) + 역삼역점 서브사이트 + 페이스북 bowlbox2019 = **본사 통합 디지털 자산 부재**. 한남점 vs 역삼점 vs 본사 톤·콘텐츠·운영 주체 분산 = 가맹 모집 리드 ROI 떨어짐.
- **가맹 모집 페이지 있으나 → 가맹 리드 끌어올 콘텐츠·퍼포먼스 시스템 부재 가능성.** 외식 가맹 = 영업사원·박람회 의존 vs 디지털 인바운드 리드 = 보울박스는 디지털 인바운드 시스템 미정착 추정. 가맹 문의 페이지 단순 노출 vs 가맹점주 후보 콘텐츠(매출 시뮬·점포 후기·창업 케이스) 부재.

- **자사몰 bowlbox.co.kr = 단순 브랜드 소개 + 가맹 문의 페이지 수준 추정.** 온라인 주문·배달·구독 D2C 채널 부재 = F&B 외식 D2C 다음 단계 미진입.

- **30일 (진단)**: 본사 통합 인스타·키 메시지 정의 / 매장별 인스타(@bowlbox_hannam·@bowlbox50·역삼점) → 본사 통합 vs 매장 보조 구조 의사결정 / 가맹 모집 페이지 인바운드 리드 진단 / 경쟁 가맹(샐러디·슬로우캘리·스윗밸런스) 차별화 메시지 도출
- **90일 (시스템 구축)**: 본사 통합 인스타 본 계정 정상화(주 3~5회 발행) + 매장 콘텐츠 SOP(매장별 → 본사 큐레이션 시스템) + 가맹 리드 퍼포먼스 셋업(메타·네이버 키워드 "샐러드 가맹"·"라이스볼 창업") + 가맹점주 후보 콘텐츠(매출 시뮬·창업 케이스 인터뷰·점포 후기 카드뉴스)
- **180일 (스케일)**: 가맹 성과 KPI 대시보드(월 가맹 문의 수·전환율·점포 수 증가) + 월 리뷰 시스템 + (선택) 배달·구독 D2C 채널 v1(쿠팡이츠·배민·자사몰 정기구독)

## 추정 Tier 견적 범위
- **추천 Tier**: **T1 Foundation (1,000~1,500만 / 4주) → T2 Build 승급 단계적 접근 ★ 강력 권장**
- **근거**: 직영 2점 + 가맹 모집 시작 단계 + 매출·직원수 미공개 = 일반 T2 풀스코프(3,000~5,000만) 흡수력 미검증. **T1 진단 + 본사 통합 인스타·가맹 리드 시스템 v1**으로 시작, 결과 보고 T2 빌드 승급.
- **단계적 접근 옵션 (강력 권장 케이스)**: T1(1,000~1,500만/4주) → T2(2,500~4,000만/3개월) 단계적 승급 — 매출 규모 + 신지은 대표가 역삼점 직접 운영 = **외부 컨설팅 동시 핸들 가능 여부 검증 필요** = 단계적 접근 가장 적합.
- **대안**:
  - 미팅에서 매출·자금 흡수력 충분하고 가맹 본격 확장 의지 강하면 → **T2 Build(3,000~4,000만/3~4개월)** 바로 진입 가능
  - 가맹 본격 확장 + 다점포(10~30점) 운영 시스템 + 배달·구독 D2C 풀스택 → **T3 Scale(7,000만+/6개월+)** — 단, 직영 2점 + 가맹 시작 단계에서는 과다 위험, **점포 수 10점+ 도달 후 권장**
  - 진단 비용 부담 시 → **T0 진단형(300~500만/2~3주)** 단축 옵션
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+

| 4 | **동명·사명 분산** | 🔴 운영사 미확인 — 검색 시 analogkitchen 혼동 정보 발견(별개 브랜드). 매장별 인스타 분산(@bowlbox_hannam·@bowlbox50·역삼점 서브사이트) = 외부 인지 분산. 본사 통합 인스타·도메인·핸들 정리 1순위 |
| 5 | **운영 흡수력** | 🔴 슬림 — 신지은 대표 본인이 역삼점 직접 운영 + 본부 마케팅·가맹 영업 인력 거의 없음 추정. 컨설팅 산출물 운영 주체(본부 인하우스 채용 vs HIZ 위탁 vs 외부 인력) 명확화 필수. 외주 마케팅 대행사 이력·만족도 청취 |

## 미팅 시 확인 질문
1. **(1순위 / 운영사·매출)** 보울박스 운영사 정식 사명·사업자등록번호? 직영 2점 월 매출 규모? **본사 매출 비중(직영 매출 vs 가맹비·로열티 vs 식자재 공급마진)**? 가맹점 수 현재·6/12개월 목표?
2. **(가맹)** 최근 6개월간 **가맹 문의·전환 추이** + 어떤 마케팅 채널(인스타·네이버 키워드·박람회·창업설명회·기존 점주 추천)이 가장 효과? 가맹비·인테리어·로열티 구조? 가맹 정보공개서 등록 여부?
3. **(인하우스)** 본사 마케팅·콘텐츠·가맹 영업 **인하우스 인력 구성**? 신지은 대표 본부 + 매장 동시 핸들 부담 여부? 외주 마케팅 대행사 이력?
4. **(채널·콘텐츠)** 매장별 인스타(@bowlbox_hannam·@bowlbox50·역삼점) **분산 운영 의도적 vs 통합 필요 인지**? 본사 통합 인스타·자사몰 운영 의지? 배달(쿠팡이츠·배민)·구독 D2C 채널 진입 계획?
5. **(컨설팅 기대 / 예산)** 6개월 후 컨설팅 결과로 가장 보고 싶은 지표 — 가맹 문의 수? 직영 매출? 브랜드 인지도? 신규 출점(한남·역삼 외)·해외 진출 로드맵? 가용 월 예산 범위?

## 정식 리서치 승급 시 우선 파야 할 영역
- **매장 수·매출 분해**: 직영 2점(한남·역삼) 월 매출 + 가맹점 수(공시 미공개) + 직영 매출 vs 가맹비·로열티·식자재 공급마진 비중 + 점포별 손익
- **가맹 전환 추이·시스템**:
  - 월 가맹 문의 수·전환율 (인스타·네이버·박람회 채널별)
  - 가맹비·인테리어·로열티 구조
  - 가맹점주 후보 콘텐츠 자산(매출 시뮬·창업 케이스·점포 후기)
  - 가맹 정보공개서 등록 여부
- **경쟁 3사 분석 (가맹 시장)**:
  - **샐러디** (가맹 309점, 총 창업비용 1.6억) — 시장 1위
  - **슬로우캘리** (가맹 120점, 총 창업비용 7,560만 원) — 효율형
  - **스윗밸런스** (공장 전처리 + 1~2명 운영) — 운영 효율
  - **샐러드박스** (2022년 70점, 초기부담금 가장 적음) — 진입 장벽 낮음
  - 보울박스 차별화: 라이스 볼(잡곡밥+토핑) + 12시간 저온숙성 스테이크 = 메뉴 매트릭스 차별화 매핑
- **신규 출점 로드맵**: 한남·역삼 외 직영 추가 vs 가맹 우선 확장 / 강남·여의도·송파·이대 등 입지 우선순위 / 해외 진출(일본·동남아 K-푸드 트렌드 활용) 가능성
- **(선택) F&B D2C 채널 확장**: 쿠팡이츠·배민·요기요 배달 비중 / 자사몰 정기구독 모델 / 단체급식·기업 케이터링 B2B 진입
