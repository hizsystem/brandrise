ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • HIZ 표준 케이스 풀에 출판사 매칭 없음 — 마키노차야(F&B), 케어비네스트(건기식) 등 D2C 제품 중심. 출판은 카테고리 신규 + 마진 구조·재고 회전·작가 IP 관리 등 별도 운영 룰셋 필요.
  • 리솔츠 / 위타민: D2C 자사 IP 콘텐츠 시스템 = 콘텐츠 시스템화 영역 부분 인접. 단 매출 회수 사이클(출판=장기·D2C=단기)이 달라 KPI 설계 별도.
  • 노다지파이랩스: Pre-launch 1인 운영 + 글로벌 채널 0 = 운영 슬림 구조 일부 인접.
  • 요약: 출판 카테고리 첫 케이스 = HIZ 표준 D2C SOP 적용 가능 영역과 출판 고유 영역 구분 필요. 콘텐츠 시스템·SNS 채널 강화는 SOP 적용 가능, 작가 발굴·서점 영업·재고 회전은 적용 불가.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 채널 인지·검색 가시성 매우 낮음 — @chobibooks 인스타 운영 확인되나 구글·네이버 1~2페이지에 거의 안 잡힘, 도서명 검색해야 출판사 노출. 출판사 단독 브랜드 검색량 미형성.
  • 이효재 IP 자산화 미흡 — 보자기·한복 거장 이효재 작가는 강력한 인지도 IP인데, 이걸 "초비북스의 세계관"으로 응집시키는 비주얼 시스템·시리즈 큐레이션·작가 페르소나 분리 미확인.
  • 1인 운영 = 콘텐츠·유통·마케팅 동시 부하 — 신간 출간 후 SNS·서점 행사·작가 인터뷰 등 기존 출판사 대비 외주·자동화 절실. 인스타 운영 빈도·인게이지먼트 미확인.
  • 카테고리 정의는 강함 — "풍류·일상" 키워드는 좋으나 외부 인지·소비자 검색어로의 연결고리 부재.
  • 재무 구조 외부 노출 0 — 매출·판매부수·재고·작가 인세 사이클 모두 미확인 = 컨설팅 흡수력 검증 1순위.

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 진단 (300~500만 / 2주) 또는 T2 빌드 (2,000~3,000만 / 3개월)
  • 근거: 출간 5종 · 8개월 운영 = 진단(T1)으로 카테고리 응집·검증 후 빌드 결정이 안전한 단계. 단 카테고리 정의·작가 IP가 이미 강함 = 빌드로 직진해도 흡수 가능성 있음. 출판 마진 구조상 풀 파트너십(T3) 흡수력 낮음 거의 확실 — T3 제외.
  • 대안:
  • 사업자 등록 미확정 + 1인 운영 흡수력 검증 안 됐다면 → T1 진단(300~500만/2주) 단계적 접근
  • 매출·판매부수 안정·인하우스 운영 인력 있다면 → T2 빌드(2,000~3,000만/3개월) 직진, 신간 런칭 키트 표준화까지 포함
  • 단발 신간 캠페인 단위면 → T1 Foundation(500~1,000만/4주) 작가별 캠페인 1개
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
