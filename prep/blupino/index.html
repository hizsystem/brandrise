PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • BLUPINO ↔ @kkul.apple_official 사명 분산 = 1순위 페인 — 인스타 계정명이 "꿀애플"로 BLUPINO와 분리 = 브랜드 정체성 일관성 부재. SEO·인스타 점유에서 BLUPINO 단일 브랜드 자산 미축적
  • 국문 D2C 인지·검색량 약함 — 다국어 사이트 대비 BLUPINO 한글 검색 시 블루보틀·블루핀 등 동음 브랜드에 묻힘 SEO 문제 추정
  • 카페쇼 B2B 리드 D2C 전환 시스템 미정착 — 박람회 = B2B 거래선 확보 가능하나 D2C 자사몰 전환율·재구매 시스템 별도 빌드 필요
  • 시드 매출·인력 흡수력 미검증 — 농업회사법인 + 청년창업 = T2 풀스코프(3,000만+) 흡수력 미팅 시 청취 필수. T1 진단(500~1,500만)이 안전한 진입점
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(BLUPINO ↔ 꿀애플 통합) + 채널(자사몰·인스타·해외 다국어 SEO) + 콘텐츠(농업회사 스토리텔링) 가 1순위. 퍼포먼스·데이터는 매출 규모 확인 후 적합도 판단.

  • 30일: BLUPINO vs 꿀애플 정체성 진단 + 자사몰/인스타 ASO·SEO 진단 + 부산카페쇼 B2B 리드 데이터 분석
  • 90일: 인스타 계정 통합 + 카페쇼·B2B 리드 D2C 전환 퍼널 설계 + 청송 농업회사 스토리텔링 콘텐츠 시리즈(릴스·유튜브)
  • 180일: 해외 채널(영·일·중) 우선순위 1개 픽 + 자사몰 ROAS 안정화 + B2B(카페·F&B) 채널 매뉴얼화
TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (500~1,500만 / 4주) 진단 → T2 Build (2,000~3,500만 / 3개월) 단계적 승급
  • 근거: Early Seed + 매출·인력 미공시 + 사명 통합 의사결정 선행 필요 = T1에서 정체성·채널 진단 우선, 결과 보고 T2 빌드. T2 즉시 진입 시 사명 분산 상태로 빌드 = ROI 낮음.
  • 대안:
  • 매출 규모·예산 흡수력 검증되고 글로벌 동시 진출 의지 강하면 → T2 Build (3,000~5,000만 / 3~4개월) 즉시 진입
  • 자금 매우 슬림 = T0 진단형 (200~300만 / 2주) 정체성 1장 보고서
  • 정부지원(농식품부·중기부 청년창업·소진공) 트랙 동반 옵션 검토
  • 단계적 접근(권장): 본 건 권장. Early Seed + 사명 통합 선행 필요 = 2단계 분할이 합리적.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
