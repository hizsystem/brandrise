- **간접 인접**: 마키노차야(오프라인 매장 + 지역 기반 + 로컬 SEO) 일부 인접 — 단 의료광고법 규제 영역 아니라는 점에서 본질 다름.

## 핵심 페인 (가설)
- **사명 충돌 1순위 페인** — "눈사랑안과" = 인간 안과 다수, "안과동물병원" = 카테고리 선두 다수 → 검색·브랜드 자산 분산. 보호자가 본 병원을 "눈사랑안과" 검색으로 도달하지 못함 + "동물 안과" 검색에서도 선두 3사에 밀림.
- **의료광고법 제약** = 일반 D2C·퍼포먼스 마케팅 그대로 적용 불가. 수의사법 + 보건복지부·수의사회 가이드라인 = 비교광고·치료 효과 표현·증례 콘텐츠 모두 사전 검토 필요.
- **카테고리 차별화 부재 가설** — "안과 전문" 동물병원이 이미 다수, 후발이라면 진료 차별점(특수 장비·전문의 자격·이종 동물 진료 등) 정의 필요.
- **온라인 자산 0 가설** — 자사 홈페이지·인스타·블로그 공개 검색 0건 = 보호자 검색 도달 경로 자체가 형성 안 됨.

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
**핏 검증 필요.** HIZ는 D2C 브랜드(뷰티·웰니스·건기식·생활가전) 중심. 수의병원은 SOP 비적합 가능성. 핏 검증 후 적용 가능한 영역만 한정:

- **적용 가능 영역**: 브랜드 정체성(네이밍·포지셔닝 차별점) + 콘텐츠 시스템(보호자 교육·증례 콘텐츠 가이드라인) + 로컬 SEO·블로그·플레이스 운영
- **비적용/조심 영역**: 퍼포먼스 광고(의료광고법 심의 부담), 인플루언서 캠페인 형식 다수, 일반 ASO, D2C 자사몰 구축
- **30일**: 사명 충돌·카테고리 차별점 진단 + 의료광고법 적용 범위 사전 검토 + HIZ 6영역 핏 거리 평가
- **90일** (핏 확정 후): 보호자 콘텐츠 라인 1개(증례·교육) + 플레이스·블로그 SEO + 로컬 검색 최적화
- **180일**: 증례·후기 시스템화 + 진료 차별점 콘텐츠 자산화 — **단, 미팅 후 핏 결정**
