핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 사명/도메인 불일치 = 1순위 페인 — 신청자 이메일 도메인(rebeat.co.kr) = 모회사 리비트지만 신청 브랜드는 "리상회". 사업자/도메인 정리 미완 상태 → 브랜드 자산이 리비트에 묻혀 검색·SNS에서 리상회가 독립적으로 자라지 못함
  • 카테고리 산만 — 리비트(라이프스타일 선물) + 리상회(식품 안주) = 카테고리·고객층·채널 전략이 분리되어야 하나 한 도메인·한 인스타(@rebeat.kr)에 혼재
  • 자사 브랜드 검색량/콘텐츠 자산 부재 — 카카오 톡딜·자사몰 의존 = 인스타 릴스·유튜브·블로그 SEO 미진. 안주 카테고리 = 비주얼·먹방 콘텐츠 핏 매우 높음에도 시스템 미정착
  • 매출·인력 흡수력 미검증 — 부산 본사 + 듀얼 브랜드 운영의 인하우스 인력 규모, T2 풀스코프(3,000만+) 흡수력 미팅 시 확인 필수
CONSULTING HYPOTHESIS

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(리비트·리상회 분리/통합 결정) + 콘텐츠(안주 카테고리 = 강한 비주얼·먹방·홈술 콘텐츠 핏) + 채널(인스타 릴스·유튜브 + 자사몰 SEO) + 퍼포먼스(캠핑·홈술·선물 키워드 ROAS) 4영역 적합. 안주는 D2C 식품 중 콘텐츠 적합도 매우 높은 카테고리(시각·미각·라이프 무드).

  • 30일: 리비트 vs 리상회 브랜드 구조 진단 → 분리(독립 도메인·인스타) vs 통합(엄브렐러) 의사결정 + 카카오 톡딜/자사몰 매출 진단
  • 90일: 리상회 인스타·릴스 콘텐츠 시스템(캠핑·홈술·선물 페르소나) + 안주 SKU별 키워드 SEO + 마이크로 인플루언서 시드 + 리비트 라이프스타일 선물 인프라와의 시즌 기프트 번들 가설
  • 180일: 자사몰 직판 비중 확대 + 퍼포먼스 ROAS 안정화 + 선물세트(명절·기념일) 시즌 캠페인
TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T1 Foundation (1,000~1,500만 / 4주) 진단 → T2 Build (2,500~3,500만 / 3개월) 단계적 승급
  • 근거: 매출·인력 미검증 + 브랜드 아키텍처 의사결정(분리 vs 통합)이 1순위 의제 = 진단 단계 T1에서 핵심 의사결정 확정 후 T2 빌드. T2 풀스코프 즉시 진입 시 아키텍처 미정 상태로 시스템 빌드 시 리스크 큼.
  • 대안:
  • 매출 규모·예산 흡수력 검증되고 듀얼 브랜드 시스템 동시 빌드 의지 강하면 → T2 Build (3,000~5,000만 / 3~4개월) 즉시 진입
  • 자금 매우 슬림 = T0 진단형 (200~300만 / 2주) 브랜드 아키텍처 1장 보고서로 의사결정만 지원
  • 단계적 접근(권장): 본 건 권장. 듀얼 브랜드 구조 정리 = 일회성 의사결정이 아닌 단계적 검증 필요.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+

  • 매출 분해: 자사몰 vs 카카오 톡딜 vs 기타 외부 채널 매출 비중 / 리비트(선물) vs 리상회(안주) SKU별 매출·마진
  • 브랜드 아키텍처 설계: 엄브렐러(리비트 마스터 + 리상회 하위) vs 하우스 오브 브랜드(완전 분리) vs 단일 브랜드 통합 의사결정 + Naming·도메인·SNS 분리 시뮬레이션
  • 경쟁 3사 분석: 캠핑·홈술 안주 D2C 경쟁사 (예: 안주야·홈술기획·이팩 등) + 시즌 선물세트 경쟁사 (마켓컬리·SSF·신세계몰 선물 카테고리)
  • 운영 인력·예산: 인하우스 마케팅·디자인·CS 풀세트 / 외주 의존 영역 / 가용 월 예산
  • (선택) 시즌 기프트 번들 가설: 리상회 안주 + 리비트 무드등·꽃다발 결합 선물세트 = 명절·생일·축하 시즌 캠페인 영역
