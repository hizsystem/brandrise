## 추정 Tier 견적 범위
- **추천 Tier**: **T0 진단형 (100~300만 / 2주) 또는 T1 Foundation 하단(500~700만 / 4주)**
- **근거**: 명시 예산 100만 + "브랜딩 처음" + 식별 정보 부재 = T1 풀스코프 진입 불가. **T0 진단 → 검증 후 T1/T2 업셀 동선이 정석**.
- **대안**:
  - 미팅 결과 카테고리·매출·예산 추가 투입 의지 확인 시 → T1 Foundation 500~1,500만/4주 진입

- **단계적 접근(권장)**: 본 건은 **T0 진단 → T1 Foundation → T2 Build** 3단계 업셀 동선이 가장 적합. 첫 진입 비용을 최소화하고 본인 학습 → 데이터 확보 → 본격 빌드 단계 권장.
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
