ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 참고 케이스 풀에 정확한 매칭 없음 — 핏 검증 필수. HIZ 표준 D2C SOP는 스타트업·시드~Pre-A 단일 법인 전제. 코스닥 상장 B2B 바이오 = 신규 영역.
  • 힐러넷 (Healernet): 진세노사이드 R&D 자산 기반 바이오 + B2B 소재 플랫폼 = R&D→마케팅 번역 의제 인접도 일부. 단 힐러넷은 D2C 요프리 Rh2 보유, 본 건은 자회사 Blisspack을 통한 D2C만 가능.
  • 위타민 · 케어비네스트: 자회사 Blisspack(동결건조 화장품 ODM) 인접도 — D2C 화장품·건기식 SOP 적용 가능 영역. 단 Blisspack은 ODM 사업이므로 D2C 직접 운영 여부 미확인.
  • 참고 케이스 부재 영역: 코스닥 상장사 IR/PR·B2B 바이오 학회·논문·KOL 마케팅 = HIZ 미경험. 외부 바이오 PR·IR 전문 파트너 필요.
  • 요약: 본 건 B2B 바이오·임상 컨설팅 영역은 HIZ 핏 거리 큼 / 자회사 Blisspack D2C 화장품 런칭 자문 가능성은 핏 양호 / 미팅에서 부서·과제 정의가 모든 것을 결정.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • B2B 바이오·의료기기 마케팅 문법 = HIZ D2C SOP와 결정적으로 다름. 학회·KOL·논문·임상시험 결과·인허가 자산 vs SNS·퍼포먼스 콘텐츠 = 별도 채널 전략 필수. HIZ 단독 풀스코프 어려운 영역.
  • 코스닥 상장사 = IR 커뮤니케이션 + B2B 인지도 + (자회사) B2C 인지도 3축 분리. 어느 축이 본 컨설팅 의제인지 정의 부재.
  • 신청자 부장급 = 의사결정·예산 결재 권한 제한 가능. 부서장·임원·CEO 결재 라인 미확정. CEO 직접 결재 vs 부장 위임 범위 청취 필수.
  • 2024 Blisspack 인수 후 D2C 화장품 영역 확장 신호 = 가장 강력한 HIZ 핏 가능성. 동결건조 화장품 ODM은 사명대로 ODM(타사 OEM) 사업이나 자체 D2C 브랜드 런칭·B2C 채널 진입 검토 가능.
  • 상장사 적자 추세 + R&D 비중 큰 자금 구조 = 외부 마케팅 컨설팅 예산 흡수력 미검증. IR 측면(상장사 마케팅 예산은 통제 강함)·신사업(자회사 인큐베이션 예산 별도 가능) 차이 큼.
