ONE-LINER

한 줄 요약

사업 본질 한 줄

서울문화사가 발행하는 30년 전통 인테리어·라이프스타일 월간지 — 공간/인테리어 트렌드, 쇼핑, 아트 칼럼을 다루며 매거진 + 웹 + 인스타·유튜브·블로그 + 오프라인 이벤트(2022~ DIVE IN SENSE 등)로 확장 중. 신청자가 편집장 본인(심효진) = 편집/브랜드 책임자 레벨 의사결정 가능하나, 발행사 서울문화사 차원 결재 라인 미팅 청취 필수.

추정 단계

현재 추정 단계

안정기 레거시 매체의 디지털 전환 단계 — 인쇄 매체 30년 + smlounge.co.kr 그룹 디지털 자산. 잡지 구독·광고 수익 안정성 하락 시기, 오프라인 팝업·전시·커머스 연계로 IP 확장 시도 중. 30년 IP 자산 보유라는 점에서 신생 D2C와 단계 자체가 다른 카테고리(레거시 매체 디지털 전환) — HIZ 풀에서 흔하지 않은 케이스.

ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • HIZ 풀 안에 직접 매칭 케이스 부재 — 핏 검증 필수. HIZ 표준 D2C SOP는 단일 D2C 브랜드 전제. 30년 매거진 IP + 발행사 그룹사 구조는 신규 영역.
  • 부분 인접 케이스:
  • 마키노차야 / 오들리컴바인: F&B·라이프스타일 브랜드 = 라이프스타일 카테고리 콘텐츠 톤 인접도 일부
  • 위타민 / 리솔츠: 브랜드 자산 보유 → 디지털 시스템 정착 단계 = 작업 패턴 일부 적용 가능
  • W'GREEN (캐나다 진출): 매체 자산 확장 시 인사이트 활용 가능
  • 레거시 매체 디지털 전환 영역 = HIZ 단독 풀스코프 검증 필요. 미팅 시 발행사 차원 요청 여부 확인 후 핏 판정.
PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 매거진 본지의 광고/유가부수 감소 압박 — 디지털·이벤트·커머스로 수익원 다각화 필요. 30년 매거진 = 인쇄 매체 환경 위축 영향 직격
  • SNS·유튜브 운영은 하고 있으나 매거진 톤·콘텐츠를 그대로 옮긴 수준 — 채널별 네이티브 포맷 최적화 부족 가능성. 잡지 화보형 vs 인스타 릴스형 vs 유튜브 영상형 분기 시스템 미구축 추정
  • 30년 브랜드 자산이 강하지만 MZ 독자(2030 인테리어/홈오피스 수요층) 유입 정체 가능성 — 매거진 본독자층 연령대 상승 + 신규 디지털 독자 유입 채널 별도 구축 필요
  • 편집국 인력으로 디지털 KPI·데이터 분석 운영 부담 — 편집 콘텐츠 인력과 디지털 그로스 인력은 다른 R&R, 인하우스 데이터 분석 인력 부재 추정
  • DIVE IN SENSE 오프라인 이벤트 → 디지털 퍼널 연계 시스템 미구축 추정 — 이벤트 방문자 데이터·콘텐츠 자산화 부재 가능성
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 콘텐츠 시스템(매거진 IP → 디지털 네이티브 변환) + 채널(인스타·유튜브·블로그 포맷 최적화) + 데이터(KPI 대시보드·CRM) 3축 풀패키지 핏. 정체성은 30년 자산으로 강하므로 운영 효율과 측정 체계가 핵심. - 30일: 인스타·유튜브·웹 현황 진단, 매거진 IP의 디지털 재해석 프레임 정의, DIVE IN SENSE 이벤트 디지털 연계 가능성 진단 - 90일: 채널별 콘텐츠 포맷 표준화 + 이벤트(DIVE IN SENSE류) 디지털 연계 퍼널 + 매거진 본지 → 디지털 콘텐츠 재가공 SOP - 180일: 구독·커머스·브랜드 협업(B2B) 데이터 대시보드, KPI 리뷰 시스템, 30년 IP 활용 B2B 브랜드 콜라보 사업 v1

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T2 Build (3,000~5,000만 / 3~4개월) 기본 / 발행사 차원 확정 시 T3 Scale (7,000~9,000만 / 6개월+) 확장 가능
  • 근거: 30년 매거진 IP + 발행사 서울문화사 그룹사 = 단가 흡수력 잠재적으로 큼. B2B 브랜드 협업 확장 시 컨설팅 단가 대비 ROI 회수 명확. 단 의사결정 라인 확정 전 풀스코프 진입 위험 → T0~T1 진단 단계 우선 권장.
  • 대안:
  • 편집장 단독 차원(예산 제한적) = T1 Foundation (1,000~1,500만 / 4주) — 콘텐츠 시스템 부분 진단 + DIVE IN SENSE 이벤트 디지털 연계 v1
  • 발행사 서울문화사 차원(그룹 전사 디지털 전환) = T3 Scale (7,000~9,000만 / 6~12개월) — 매체 그룹사 디지털 전환 풀스택 + B2B 브랜드 협업 사업 신규 라인 빌드
  • 미팅 결과 의사결정 라인·예산 미검증 시 = T0 진단형 (200~300만 / 2주) 으로 디스커버리 우선
  • 단계적 접근(권장): 본 건 권장. 발행사 차원 의사결정 라인 미확정 = T0~T1 진단 우선 후 T2 빌드 승급.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
