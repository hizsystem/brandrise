ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • HIZ 풀에 모바일 앱·AI 플랫폼 케이스 부재 — D2C 물리 제품 중심 SOP와 거리 있음. 단 ASO·콘텐츠 시스템·글로벌 채널은 표준 SOP 적용 가능.
  • 노다지파이랩스 (가장 가까움): Pre-launch/초기 + 글로벌 타겟 + 빌더형 대표 운영 + 외부 마케팅 협업 경험 부재 = 단계 인접. 다만 노다지는 Web3·D-50 압축 일정, 믹스룸은 양대 OS 출시 직후 = 시점 차이.
  • 위타민(우르실) / 리솔츠 초기: 자사 제품 + 자사 디자이너 시스템 + 글로벌 진출 의지 = 디자인 시스템·BI 의제 톤 인접.
  • 어프리데이 / 큐라펄스: 시드 단계 글로벌 진출 직전 = 자금·예산 흡수력 검증 필요라는 점에서 인접.
  • 요약: 글로벌 AI 앱 = HIZ 신규 영역이나 ASO·콘텐츠·BI 영역 SOP 적용 가능 + 단 글로벌 CAC·앱마케팅 풀스코프는 외부 파트너 필요.

  • 양대 OS 동시 출시 + 글로벌 무료 다운로드 = 데이터 트래킹·KPI 정립 시점
