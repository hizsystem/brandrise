# 큐라펄스 · 사전상담 PREP

## 한 줄 요약
초음파·펄스·광(光) 융합 자극 기술 기반 뷰티테크 스타트업. 5중 초음파(1·3·10·19·25MHz) 기술 보유, 자사 디바이스 브랜드 "듀얼테트라(DualTetra)" 전개, 두뇌케어(BrainStim/Sync)·스킨케어(WaveSync/DualPoration/IntraSonix) 라인 운영.

## 추정 단계
**Pre-A (시드 종료, Pre-A 유치)** — 뉴패러다임인베스트먼트 시드 투자, 2025년 TIPS 글로벌 트랙 선정, G&P 인베스트먼트 Pre-A 유치 보도. 일본(토요타 통상)·중화권·북미 OEM/ODM 추진 단계.

## 핵심 페인 (가설)
- B2C 디바이스 런칭 직전 — **자사몰(curapulse.cafe24.com) + 인플루언서 마케팅 본격화 직전 채널·콘텐츠 시스템 미정착** 가능성
- 5중 초음파·복합 기술 = 설명 난이도 높음. **기술 언어를 소비자 베네핏 언어로 번역**하는 카피·콘텐츠 시스템 부재 가능
- 글로벌(일본·중화권·북미) 동시 진출 — 다국가 브랜드 일관성 + 현지 채널 운영 부담

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
- 6영역 중 **브랜드 정체성 + 콘텐츠 + 채널 + 퍼포먼스** 풀패키지 핏 매우 높음 (D2C 디바이스 런칭 = HIZ 강점)
- 30일: 브랜드/카피 진단 + 듀얼테트라 자사몰 ASO·랜딩 진단 / 90일: 인스타·유튜브 콘텐츠 시스템 + 인플루언서 시드 30~50명 / 180일: 퍼포먼스 ROAS 안정화 + 글로벌 채널 v1
- **GO 가능성 매우 높음** — 이미 우리의 리솔츠/위타민/케어비네스트 케이스와 유사 구조

## 미팅 시 확인 질문
1. 현재 듀얼테트라 D2C 매출/MAU, 인스타·유튜브 팔로워, 광고비 월 집행 규모는?
2. 최근 6개월 인플루언서·퍼포먼스 광고 시도와 ROAS는?
3. Pre-A 자금 사용처 중 마케팅·브랜드 비중과 의사결정 구조(대표 단독 vs 보드)?
4. 컨설팅 후 6개월 내 기대하는 1순위 성과 (매출? 인지도? 글로벌 진입?)?
5. **(중요)** 글로벌(일본·중화권·북미) 채널은 자체 운영인가 현지 파트너 위탁인가?
