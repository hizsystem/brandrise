ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • HIZ 표준 케이스 풀에 B2B 기프트 매칭 없음 — 케어비네스트(D2C 건기식)·리솔츠(D2C 웰니스)·마키노차야(F&B 매장) 등 D2C/B2C 중심. B2B 카테고리는 신규 영역.
  • 위타민·우르실: D2C이지만 콘텐츠 시스템·자사 사이트 리뉴얼 영역 일부 인접. 단 B2B 사이클(견적→납품 수개월)과 D2C(즉시 구매) 차이 큼.
  • 참고 케이스 없음 — 핏 검증 필수: B2B 인바운드 마케팅(SEO·LinkedIn·블로그·케이스 스터디) 영역은 HIZ 표준 SOP 외. 단 콘텐츠 시스템·브랜드 정체성 재정의는 HIZ 강점 영역 적용 가능.
  • 요약: B2B 카테고리 = HIZ D2C 강점과 거리 있음. 리드 인입 시스템 + 콘텐츠 자산화 영역만 SOP 적용 가능, 퍼포먼스 광고는 B2B 특성상 우선순위 낮음.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • B2B 기프트 시장 = 카테고리 자체가 디지털화/브랜딩 약함 — 경쟁사(기프트윙·기프트조아·시너지기프트) 모두 옛 쇼핑몰형 사이트. 보르미의 강점("대기업 레퍼런스 + 통합 제작")이 디지털에서 전혀 보이지 않음.
  • 영업이 기존 거래처·소개·견적 의뢰 의존도 높음 가설 — 신규 리드 인입 채널(인바운드 마케팅) 부재. 신규 기업 고객(MD·마케터·인사팀) 의사결정자에게 신뢰 전달 어려움 = 케이스 스터디·포트폴리오·웰컴키트 사례·견적 가이드 부재.
  • 자사 도메인 boreumi.com 운영 + 자사 메일 = 디지털 기반은 있음 — 단 사이트 콘텐츠가 단순 소개 수준 = 리드 인입·CRM·견적 자동화 등 시스템 부재 가설.
  • 사명 "보르미" 동명 다수 — 검색 시 보르미테크·보르미몰·보르미네일 등과 혼동 = "Boreumi" 영문 풀네임·SEO 점유 전략 필요.
  • 김포·강서 이중 거점 = 사옥/물류/생산 분리 가능성 = 인하우스 마케팅 인력 보유 여부 미팅 청취 필요.
