ANALOG CASES

유사 케이스 인접도

HIZ 기존 케이스 풀과의 매핑

  • 위타민(우르실) / 리솔츠: 자사 뷰티·웰니스 브랜드 + 자사몰 + 글로벌 확장 시점이라는 점에서 구조 유사. 다만 하예진은 이미 11년차 + 매출 27.5억 + 멀티 지역 채널 보유 = 위타민/리솔츠보다 1단계 위 스테이지(스케일 전환기).
  • 레베로(LEVERO): 비건 라이프케어 + 멀티 SKU + 글로벌 채널 진입 단계 측면에서 인접. HIZ가 비건 카테고리 컨설팅 케이스 보유한 점에서 핏 양호.
  • 마케소울/제이엘코스메틱 등 K-뷰티 수출도매 + 자사 브랜드 병행 케이스 — 비즈모델 일치도 가장 높음 (HIZ 기존 케이스 풀에 정확히 매칭되는 사례는 없음, 인접 신규 카테고리).

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 글로벌 다채널 동시 운영 부담 — 8개+ 지역별 인스타 계정 분산 운영 중 → 브랜드 메시지·비주얼 일관성 + 현지화 톤 균형 시스템화 미흡 가능성. 일본·미국·인도네시아·EU 멀티 톤 관리 = 콘텐츠·운영 리소스 분산.
  • 본 계정 12K 팔로워 = 11년차·매출 27.5억 대비 인지도 약함 — B2B 수출도매 매출 비중이 크고 D2C 자사몰·SNS 마케팅이 후순위였을 가능성. 자사몰 트래픽·전환·LTV 데이터 시스템 미구축 추정.
  • 자체 브랜드(하예진) vs 수출도매(타 K-뷰티) 두 비즈 라인 분리 — 브랜드 자체 가치 극대화 전략 부재 가능성, "도매상사가 운영하는 브랜드" 정체성에 갇힘.
CONSULTING HYPOTHESIS

컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)

브랜드라이즈가 풀 수 있는 것

6영역 중 콘텐츠 시스템 + 채널 운영 통합 + 데이터 대시보드가 1순위. 브랜드 정체성·BI는 이미 정착된 단계로 추정. - 30일: 브랜드 진단 (8개 지역 채널 메시지 일관성 진단 / 자사몰 ASO·전환 진단 / 본 계정 12K 인게이지먼트 분석) - 90일: 글로벌 콘텐츠 SOP 수립 (지역별 톤 가이드 + 본사 콘텐츠 풀 → 지역 큐레이션 시스템), 본 계정 인스타·유튜브 콘텐츠 빈도 정상화 - 180일: 자사몰 → 채널별 매출 분해 대시보드 + 비건 카테고리 인플루언서 글로벌 시드(미국·일본·EU 각 10명), 본 계정 30K+ 목표
