  • 네츄럴굿띵스 (가장 가까움): 자생 성장형 중견 D2C 건기식 + 카테고리 1위 + 글로벌 진출 시점 = 단계·구조 유사. 절대콜라겐은 NGT 대비 인력·매출 규모는 작으나 카테고리 모트는 동등 이상.

PAIN POINTS

핵심 페인 (가설)

사전 검색 기반 가설 페인

  • 국내 카테고리 1위 = 신규 성장 동력 부재 위기. 후기 19만+ 누적은 강력한 모트, 다만 국내 콜라겐 카테고리 자체 성장률 둔화·경쟁 격화(더코리아콜라겐·뷰티콜라겐·코스트코 PB 등) → 글로벌·신규 카테고리·B2B 등 다음 성장 축 미정 추정.
  • 브랜드 자산 = 후기 중심 vs 브랜드 스토리·아이덴티티 약함. "절대콜라겐"이라는 직설적 사명·일관성 있는 카피·BI는 확보되어 있으나, 12년 누적 자산을 활용한 콘텐츠·인플루언서·PR 시스템은 후기 의존으로 추정 약함.
  • 직원 3명 = 본사 슬림 + 매출 규모 vs 인력 격차 → 마케팅·콘텐츠·운영 외주 의존도 높음. 외주사 교체나 인하우스화 검토 시점 가능성.
  • 글로벌 진출 = 영국 Absolute Collagen 사명 충돌 = 영문 자산 구축 시 1순위 장애물. 영문몰·아마존·해외 SEO에서 두 브랜드 식별 비용 매우 큼.
  • 자매 브랜드(절대보이차) 분산 = 그룹 마스터 브랜드 전략 미정 추정.

브랜드라이즈가 풀 수 있는 것

6영역 중 브랜드 정체성(스토리 재정의) + 콘텐츠 시스템(후기 19만+ 자산 활용) + 채널 확장(글로벌·B2B) + 데이터 대시보드 4개 동시 핏. 1위 브랜드의 다음 단계 = HIZ가 가장 잘하는 "스케일 전환" 영역. - 30일: 브랜드 진단 — 12년 자산 인벤토리(후기 19만+·기능성 인증·재구매·KOL), 영국 동명 충돌 매핑, 차세대 성장 축 옵션 정리(글로벌/신규 SKU/B2B) - 90일: 콘텐츠 시스템(후기 카드뉴스·셀럽·전문가 PR·기능성 임상 콘텐츠), 영문 자산 구축 v1 + 영국 동명 차별화 한 줄 - 180일: 신규 SKU 런칭 캠페인(예: 음용 콜라겐 외 패치·앰플·이너뷰티 확장) + 글로벌 채널 v1(미국·일본·동남아 중 1개국 우선) + ROAS 대시보드

TIER ESTIMATE

추정 Tier 견적 범위

예상 견적 Tier·단가 범위 + 단계적 접근 옵션

  • 추천 Tier: T2 Build (3,000~5,000만 / 3~4개월) → 글로벌 진출 동반 시 T3 Scale (5,000만~7,000만+ / 6~12개월)
  • 근거: 12년차 + 카테고리 1위 + 후기 19만+ = 진단(T1) 단계 이미 통과. 빌드(T2) 풀스코프로 콘텐츠·채널·데이터 동시 구축 필요. 매출 규모 대비 직원 3명 = 외주 마케팅 예산 흡수 충분.
  • 대안:
  • 글로벌 진출(영국 동명 해결 + 미·일·동남아 영문 자산·해외 인플루언서 시드) 동시 추진 → T3 Scale(7,000만+/6~12개월) 적극 권장
  • 미팅 결과 의사결정이 단발성 신규 SKU 런칭 캠페인 단위면 → T1 Foundation(1,000~1,500만/4주) 진단부터
  • 단계적 접근(권장 케이스 X): 카테고리 1위 = 진단 단계 이미 자체 통과. T1 단계 회수 = 기회비용. T2 즉시 진입 + 글로벌 진출 합의 시 T3 승급이 표준.
  • Tier 기준선: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
