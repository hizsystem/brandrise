# 하예진코스메틱 (HAYEJIN) · 사전상담 PREP

## 한 줄 요약
**비건 클린 뷰티 K-코스메틱 브랜드** — 운영사 씨아이에스인터내셔날서비스㈜(대표 최재근, 2015년 설립). 영국 Vegan Society 인증 비건 라인 운영, 일본·인도네시아·미국·유럽 등 멀티 글로벌 인스타 채널 보유. "하예진"은 **인플루언서 IP가 아닌 자체 개발 브랜드 네이밍**("하늘처럼 깊고 아름다운 마음"의 한국어 표현 — 공식 브랜드 스토리).

## 추정 단계
**얼리 그로스 → 스케일 전환기 (11년차 K-뷰티 수출 도매상사 + 자사 브랜드 운영).** 잡코리아 공시 기준 직원 12명·매출 27.5억(2025-12-31 기준), 2015년 설립 → 2018년 카자흐스탄 100만 달러 첫 수출 → 2020년 매출 50억(과거 보도) → 현재 글로벌 다채널 운영. 자체 브랜드 + 타 K-뷰티 수출도매 병행 비즈모델. 인스타 본 계정 12K 팔로워.

## 유사 케이스 인접도
- **위타민(우르실) / 리솔츠**: 자사 뷰티·웰니스 브랜드 + 자사몰 + 글로벌 확장 시점이라는 점에서 구조 유사. 다만 하예진은 **이미 11년차 + 매출 27.5억 + 멀티 지역 채널 보유** = 위타민/리솔츠보다 1단계 위 스테이지(스케일 전환기).
- **레베로(LEVERO)**: 비건 라이프케어 + 멀티 SKU + 글로벌 채널 진입 단계 측면에서 인접. HIZ가 비건 카테고리 컨설팅 케이스 보유한 점에서 핏 양호.
- **마케소울/제이엘코스메틱** 등 K-뷰티 수출도매 + 자사 브랜드 병행 케이스 — 비즈모델 일치도 가장 높음 (HIZ 기존 케이스 풀에 정확히 매칭되는 사례는 없음, 인접 신규 카테고리).

## 핵심 페인 (가설)
- **글로벌 다채널 동시 운영 부담** — 8개+ 지역별 인스타 계정 분산 운영 중 → **브랜드 메시지·비주얼 일관성 + 현지화 톤 균형** 시스템화 미흡 가능성. 일본·미국·인도네시아·EU 멀티 톤 관리 = 콘텐츠·운영 리소스 분산.
- **본 계정 12K 팔로워 = 11년차·매출 27.5억 대비 인지도 약함** — B2B 수출도매 매출 비중이 크고 D2C 자사몰·SNS 마케팅이 후순위였을 가능성. 자사몰 트래픽·전환·LTV 데이터 시스템 미구축 추정.
- **자체 브랜드(하예진) vs 수출도매(타 K-뷰티)** 두 비즈 라인 분리 — 브랜드 자체 가치 극대화 전략 부재 가능성, "도매상사가 운영하는 브랜드" 정체성에 갇힘.

## 컨설팅 가설 (브랜드라이즈가 풀 수 있는 것)
6영역 중 **콘텐츠 시스템 + 채널 운영 통합 + 데이터 대시보드**가 1순위. 브랜드 정체성·BI는 이미 정착된 단계로 추정.
- **30일**: 브랜드 진단 (8개 지역 채널 메시지 일관성 진단 / 자사몰 ASO·전환 진단 / 본 계정 12K 인게이지먼트 분석)
- **90일**: 글로벌 콘텐츠 SOP 수립 (지역별 톤 가이드 + 본사 콘텐츠 풀 → 지역 큐레이션 시스템), 본 계정 인스타·유튜브 콘텐츠 빈도 정상화
- **180일**: 자사몰 → 채널별 매출 분해 대시보드 + 비건 카테고리 인플루언서 글로벌 시드(미국·일본·EU 각 10명), 본 계정 30K+ 목표

## 추정 Tier 견적 범위
- **추천 Tier**: **T2 Build (3,000~5,000만 / 3~4개월)**
- **근거**: 11년차 + 매출 27.5억 + 8개+ 지역 채널 = 진단형(T1)으로는 부족, 글로벌 콘텐츠 시스템·데이터 대시보드 구축까지 가야 효과. 매출 27.5억 대비 마케팅 예산 흡수력 충분.
- **대안**:
  - 미팅 결과 **자사 브랜드 vs 도매 매출 비중**이 도매 우위면 → T1 Foundation(500~1,500만/4주)으로 D2C 정체성 재정의부터 시작
  - 글로벌 멀티지역 콘텐츠 시스템 + 본격 퍼포먼스 광고까지 가면 → **T3 Scale(7,000만+/6개월+)**
- **Tier 기준선**: T0 100~300만/2주 · T1 500~1,500만/4주 · T2 2,000~5,000만/8~12주 · T3 7,000만+/16주+
