01Why Meta Ads

메타 광고를 하는 이유

핵심 3가지 — 콘텐츠 강점 보유 브랜드일수록 유리.
  • 오가닉 도달의 천장 — 팔로워·알고리즘이 정하는 도달 한계. 광고 = 도달을 구매하는 유일한 레버
  • 검증된 콘텐츠의 증폭 — 오가닉에서 이미 터진 소재를 광고로 확대. 소재 리스크 최소화
  • 데이터 자산 축적 — 집행 → 픽셀 데이터 축적 → 타겟 정밀화 → 광고비 효율 상승 사이클
한 줄 정리

광고는 콘텐츠를 대체하는 게 아니라, 잘 터진 콘텐츠를 증폭하는 확성기입니다. 확성기는 목소리 좋은 사람이 들 때 가장 쎕니다.

03Campaign / Ad Set / Ad

캠페인 구조 — 3층

광고관리자 화면이 복잡해 보이는 이유 = 이 3층이 겹쳐 있어서. 이것만 알면 됩니다.
1층캠페인 Campaign
무엇을 시킬지 — 목표 (트래픽? 판매?)
2층광고세트 Ad Set
누구에게(타겟) · 얼마나(예산) · 언제까지(기간)
3층광고 Ad
실제 눈에 보이는 이미지·영상·카피

캠페인 1개 → 광고세트 여러 개 → 광고 여러 개. 일상 운영 = 3층(광고) ON/OFF. 예산 배분 = 메타 자동(이긴 광고에 집중) — 소재 여러 개 경쟁 구조의 이유.

04실측 — 7/7~9 우리 계정

실전 — 숫자에서 액션으로

지표는 읽는 게 아니라 액션을 정하는 재료. 이번 주 실제 화면의 숫자 3장면으로 "이 숫자를 보면 무엇을 하는가"를 연습합니다.
A

소재 간 CTR 격차 → 교체 판단

SOS A (상시)₩79,3796,9393.42%0
진정 라인 타임딜₩41,3065,5290.67%0

같은 날, 같은 계정. 타임딜은 40% 할인 혜택까지 실렸는데 CTR이 1/5. 어떤 액션을 해야 할까요?

진단 & 액션

진단혜택 좋음 ≠ 소재 좋음. 타임딜 소재는 가격·기간 정보 위주 — 첫 3초에 스크롤을 멈출 훅 부재.

액션① 충분한 노출(수천)에서 CTR 3배 이상 차이 → 진 소재 OFF·예산 회수 ② 이긴 소재의 훅 문법을 타임딜 포맷에 이식 → 3~5개 후보로 재경쟁 ③ 타겟·예산은 유지 — 범인은 소재.

B

높은 CTR의 함정 → 표본 크기 확인

광고 (7/9)지출노출CTR구매구매당 비용
SOS B₩2,076613.28%1₩2,076
미니 3종 B₩98,6184,5221.13%4₩24,655

SOS B가 CTR도 높고 구매당 비용도 12배 저렴합니다. SOS B에 예산을 몰아야 할까요?

진단 & 액션

진단아직 판단 불가 — 표본 부족. 노출 61회 = 전단지 61장. 구매 1건은 실력일 수도, 우연일 수도. 미니 3종 B는 노출 4,522·구매 4건 = 검증된 승자.

액션① 판단 최소 기준부터 (예: 노출 1,000+ 또는 지출 2~3만원+) ② 기준 미달 소재는 결론 유보 ③ 초기 숫자에 예산 몰빵 금지.

C

한 소재의 예산 독식 → 자동 배분 이해

한 세트 안에서 미니 3종 B가 예산을 거의 다 먹었습니다. 나머지 소재는 기회조차 없었는데 — 문제일까요?

진단 & 액션

진단독식 자체는 메타의 정상 동작 — 초기 반응이 좋은 소재에 예산을 집중시키는 자동 배분입니다. 문제는 독식하는 소재가 "성과를 내면서" 먹는지, "지출만" 먹는지.

액션독식 판단 3갈래 — ① 독식 + 구매 발생(지금 상황) → 그대로 둔다. 이긴 소재다 ② 독식 + 구매 0 → OFF하고 예산을 다른 소재로 ③ 기회를 못 받은 소재 중 꼭 검증하고 싶은 게 있으면 → 별도 세트로 분리해 강제로 예산 배정.

숫자 → 액션 3원칙

① 표본이 충분한가(케이스 B) ② 독식은 성과와 함께인가(케이스 C) ③ 둘 다 통과하면 그때 소재·예산 액션(케이스 A). 순서를 지키면 잘못된 결정의 대부분을 피합니다.

액션① 직접 광고를 눌러 결제 직전까지 가보기 ② 상세페이지 첫 화면·가격·후기 보강 ③ 광고가 아니라 사이트를 고칠 자리.
