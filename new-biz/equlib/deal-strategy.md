## 서두: 왜 "3-Tier 패키징"이 아니라 "4-Track 관계 설계"인가

일반 수주 플레이북의 핵심 도구는 **3-Tier 패키징(Good/Better/Best)** 이다. 가격 앵커링과 가치 프레이밍으로 "사게" 만드는 설계다. 그런데 이번 건은 그 도구가 오히려 **역효과**를 낸다. 세 가지 이유.

### 이유 1 — 고객이 "사지 않아도 되는 사람"이다

이퀼립은 (주)플래티넘브릿지의 D2C 뷰티 브랜드이며, 플래티넘브릿지는 이미 **브랜드 기획–제작–유통–마케팅을 수직 통합**으로 운영하는 회사다([plateau bridge LinkedIn](https://kr.linkedin.com/company/platinumbridge), [KFE News](https://www.kfenews.co.kr/news/articleView.html?idxno=576217)). 김도현 부대표 본인이 "브랜드 인큐베이팅" 비즈니스를 이끄는 전문가다. **"맡겨주세요"가 먹히는 고객이 아니다.** 일반 제안서는 첫 장에서 폐기된다.

### 이유 2 — 이해관계 충돌 구조가 있다

우리가 접촉 중인 이퀼립의 **공구 유통 상품을 "데이나님"이 제조하고 있다**. 브랜드라이즈가 "이퀼립 패키지·브랜드 컨센선스 컨설팅"을 유료로 수주하는 순간, **데이나님 입장에서 우리가 고객사(이퀼립)의 편으로 이동한 것처럼 보인다.** 이는 브랜드라이즈의 기존 관계자(데이나님)와의 신뢰에 균열을 만든다. 즉, **수주 자체가 리스크다.**

### 이유 3 — 고객의 표면 니즈가 "진짜 니즈"와 다를 가능성이 크다

폼에 응답하지 않은 채 연락처만 공유되었고, 한나님(리솔츠) 경로로 유입되었다. 김도현 부대표는 **리솔츠를 팬처럼 좋아하고 내부 디자이너 교육 레퍼런스로 쓰는 사람**이다. 즉, 그가 원하는 건 "컨설팅 계약"이 아니라 **"리솔츠의 내부를 궁금해함"** 일 가능성이 높다. 이 경우 가격 제안서를 들이미는 건 상대방을 **"나를 영업 대상으로 보는구나"** 로 전환시켜 관계를 망친다.

### 결론: 목표를 "수주"가 아니라 "관계 자산"으로 재정의

이번 건의 1차 목표는 **"미팅 성사 + 경계심 해소 + 관계 설계"** 다. 따라서 우리는 "얼마짜리 제안"이 아니라 **"4가지 가능한 관계의 궤적"** 을 제시한다. 이것이 **4-Track 관계 설계**다. 각 트랙은 **①충돌 리스크, ②가치 교환 방식, ③성사 조건**이 다르다. 고객은 "어떤 트랙으로 갈지"를 스스로 선택하게 된다. 이 구조가 영업 냄새를 제거하면서 **"결국 브랜드라이즈와 뭔가 함께 하고 싶다"** 는 결론으로 유도한다.

> **핵심 재정의** — 이번 1차 미팅에서 "돈"은 주제로 꺼내지 않는다. 단, 4개 트랙의 "대략의 투자 규모대"는 명시하여 **상대가 스스로 적합도를 판단**할 수 있게 한다. 가격을 제시하는 것과 가격을 논의하는 것은 다르다.

## 섹션 1. 기회 적합도 평가 (표준 6축 + 충돌/파트너십 2축)

### 1.1 평가표

| 축 | 점수 (1-5) | 근거 |
|---|---|---|
| ① 니즈-서비스 핏 | **3** | 폼 응답 없음. 표면 니즈("패키지 + 컨센선스")는 우리 강점과 핏이지만, 이미 내부에 브랜드 기획팀 보유 → 실제 외주 의지 불명 |
| ② 예산 | **2** | 미확인. D2C 뷰티 브랜드 4년차 + Pre-A 미확인 단계 + 글로벌 확장 중 → 투자 여력은 있으나 컨설팅 지출 우선순위는 낮을 가능성 |
| ③ 의사결정 구조 | **4** | 김도현 부대표 = 실질 의사결정권자. 대표(이아영)와 투톱 체제. 결재선이 짧음 |
| ④ 긴급도 | **2** | 폼 응답 없음 = 긴급하지 않음. "있으면 좋다" 수준. 진짜 긴급했으면 직접 리솔츠를 컨택했을 것 |
| ⑤ 성장 잠재력 | **5** | 11개국 수출 중 + 17개국 확장 준비 + 백화점 팝업 + 글로벌 채널 확대 → 장기 파트너십 가치 매우 높음 |
| ⑥ 레퍼런스 가치 | **5** | "이퀼립"은 인플루언서 뷰티 업계에서 인지도 있는 브랜드. 포트폴리오로서 파괴력 있음 |
| ⑦ **이해관계 충돌 리스크** | **🔴 4 (높음)** | 데이나님 제조 공구 상품 = 직접 컨설팅 수주 시 충돌. "이퀼립 전체 브랜드 컨설팅" 범위가 공구 상품을 포함하면 즉시 충돌 |
| ⑧ **파트너십 잠재 가치** | **5** | 플래티넘브릿지 = "공픽(인플루언서 공구 플랫폼) + 이퀼립(자체 브랜드) + 브랜드 인큐베이팅" 모델 운영. 브랜드라이즈의 브랜드 인큐베이팅/디자인 역량과 **교차점 매우 큼**. 향후 플래티넘브릿지가 신규 브랜드 런칭 시 브랜드라이즈가 디자인/IMC 파트너로 참여 가능 |

### 1.2 종합 판정

- **표준 평균** (1~6): 3.5 / 5 — 애매함
- **충돌 리스크**: 🔴 높음 (공구 상품 영역 회피 필수)
- **파트너십 가치**: 🟢 매우 높음 (장기)

### 1.3 Go / No-Go 결정

| 결정 사안 | 판정 |
|---|---|
| **미팅 자체** | 🟢 **Go** — 경계심 해소 + 관계 자산 확보 가치가 충분 |
| **컨설팅 직접 수주 (공구 상품 포함)** | 🔴 **No-Go** — 이해관계 충돌. 절대 제안하지 않음 |
| **컨설팅 직접 수주 (공구 상품 제외 영역)** | 🟡 **Conditional Go** — 범위 명확히 분리 시 가능 |
| **파트너십/공동 프로젝트** | 🟢 **Best Go** — 가장 추천하는 궤적 |
| **교육/워크샵** | 🟢 **Go** — 충돌 없음, 관계 설계에 최적 |

## 섹션 2. 4-Track 관계 설계

> **원칙**: 미팅에서 이 4개 트랙을 **"가능한 관계의 종류"** 로 소개한다. "골라주세요"가 아니라 **"이런 여러 방식이 있는데, 오늘은 먼저 서로 알아가는 자리로 하시죠"** 로 접근한다.

### 2.1 4-Track 한눈에 보기

| 트랙 | 이름 | 충돌 리스크 | 투자 규모 (참고) | 성사 조건 | 추천도 |
|---|---|---|---|---|---|
| **A** | 단순 상호학습 미팅 | 🟢 없음 | 0원 (무상) | 상호 호감 | ⭐⭐⭐ (진입점) |
| **B** | 공구 상품 제외 영역 자문 | 🟡 중간 | 월 300~700만 / 3M | 범위 명확 합의 | ⭐⭐ |
| **C** | 파트너십 / 공동 프로젝트 | 🟢 낮음 | 프로젝트당 1,500~5,000만 | 신규 브랜드/라인 론칭 타이밍 | ⭐⭐⭐⭐⭐ (최고) |
| **D** | 교육 / 워크샵 | 🟢 없음 | 500~1,500만 / 1~2회차 | 내부 디자이너 학습 니즈 | ⭐⭐⭐⭐ |

### 2.2 Track A — 단순 상호학습 미팅 (**진입점**)

| 항목 | 내용 |
|---|---|
| **정의** | 컨설팅·계약 무관, 브랜드 인큐베이팅 사업자 간의 상호 학습 자리 |
| **범위** | 1회 대면 미팅 (60~90분). 상호 비즈니스 소개 + 산업 관점 교환 + 각자 놓친 인사이트 공유 |
| **가격** | **0원** — 미팅 비용조차 받지 않음 |
| **충돌 리스크** | 🟢 없음. "컨설팅"이 아니므로 데이나님 건과 무관 |
| **성사 조건** | 서로 만날 가치가 있다고 판단하면 성사 |
| **브랜드라이즈 이익** | ① 관계 자산 확보 ② 플래티넘브릿지의 "공픽+이퀼립+인큐베이팅" 구조 직접 학습 ③ Track C로 자연스럽게 연결 |
| **고객 이익** | ① 리솔츠 내부 철학 직접 청취 (그가 원하던 것) ② 브랜드 인큐베이팅 사업자 간 네트워크 ③ 무료 인사이트 |
| **이번 1차 미팅은 이 트랙으로 성사시키는 것이 기본값** | |

### 2.3 Track B — 공구 상품 제외 영역 자문

| 항목 | 내용 |
|---|---|
| **정의** | 이퀼립 본 브랜드 또는 신규 라인 중 **"공구 유통 상품이 아닌"** 영역에 대한 유료 자문 |
| **범위 예시** | - 이퀼립 본 브랜드의 글로벌(11개국→17개국 확장) 패키지 일관성 점검<br>- 백화점 팝업/오프라인 쇼룸의 공간-패키지-디지털 연결성<br>- 신규 라인 론칭(오 브리에/오 캄 외) 시 브랜드 컨센선스 설계 |
| **기간** | 3개월 단위 리테이너 또는 단발성 진단 |
| **가격 (참고)** | - 단발 진단 리포트: 500~700만<br>- 3개월 리테이너: 월 300~500만 |
| **충돌 리스크** | 🟡 중간 — 범위에서 **"공구 유통 상품"을 명시적으로 제외**하는 계약서 조항 필수 |
| **성사 조건** | ① 이퀼립이 실제 글로벌/오프라인 확장에서 페인을 느끼고 있을 것 ② 데이나님 건과의 분리 가능 여부 사전 확인 ③ 내부 디자인팀이 "외부 시각"에 대한 필요를 인정 |
| **전개 시점** | 1차 미팅 당일에는 **꺼내지 않음**. 2차 미팅 혹은 파트너십 대화가 진전된 이후 "만약 필요하시다면" 모드로 제시 |

### 2.4 Track C — 파트너십 / 공동 프로젝트 (**최고 추천**)

| 항목 | 내용 |
|---|---|
| **정의** | 플래티넘브릿지가 신규 브랜드/라인을 인큐베이팅할 때, 브랜드라이즈가 **공동 파트너**로 디자인·IMC·브랜드 컨센선스를 책임지는 구조 |
| **논리** | 플래티넘브릿지는 "인플루언서 커머스 + 브랜드 인큐베이팅" 강점. 브랜드라이즈는 "브랜드 전략 + 디자인 + IMC" 강점. **교차점은 있되 겹치지 않음**. 서로의 약점을 보완 |
| **모델 옵션** | ① **프로젝트 수주** (브랜드라이즈가 용역 수행, 건당 정산)<br>② **RevShare** (브랜드라이즈가 지분 혹은 매출 일정 %를 가져감)<br>③ **White-Label 파트너** (플래티넘브릿지의 인큐베이팅 프로그램에 "브랜드라이즈가 디자인/IMC 파트너"로 명시) |
| **가격 (참고)** | - 프로젝트 수주: 브랜드 론칭 1건당 1,500~5,000만 (범위: 네이밍–BI–패키지–런칭 캠페인)<br>- RevShare: 초기 비용 절감 + 장기 업사이드 (협의)<br>- White-Label: MOU + 건당 정산 |
| **충돌 리스크** | 🟢 낮음 — **새로운 브랜드**에 적용되므로 기존 이퀼립 공구 상품과 무관 |
| **성사 조건** | ① 플래티넘브릿지가 향후 12개월 내 신규 브랜드 론칭 계획이 있을 것 ② 상호 사업 철학 정렬 ③ 실적 레퍼런스에 대한 상호 신뢰 |
| **전개 시점** | 1차 미팅 후반부 자연스러운 흐름에서 **가설 질문**으로 꺼냄: *"혹시 다음 런칭하실 브랜드가 있다면 저희가 파트너로 도울 수 있을 만한 접점이 있는지 궁금한데요."* |
| **브랜드라이즈 전략 가치** | 🌟 **최고** — 단발 수주보다 장기 파트너십이 훨씬 가치 큼. 플래티넘브릿지의 매 런칭마다 수익 발생 가능 |

### 2.5 Track D — 교육 / 워크샵

| 항목 | 내용 |
|---|---|
| **정의** | 이퀼립/플래티넘브릿지 내부 디자이너·기획자 대상 **"리솔츠 브랜드 철학 & 브랜드 컨센선스 설계법"** 세션 |
| **범위** | - 1회차 반나절: 리솔츠 브랜드 철학 해부 + Q&A<br>- 2회차 구성: ①철학 세션 ②실전 워크샵(이퀼립 자체 소재 가지고) |
| **기간** | 1~2회차 (월 1회) |
| **가격 (참고)** | - 1회차 (반나절 세션): 500~800만<br>- 2회차 (철학 + 워크샵): 1,000~1,500만 |
| **충돌 리스크** | 🟢 없음 — 교육 콘텐츠이며 특정 상품 자문이 아님 |
| **성사 조건** | ① 김도현 부대표가 이미 "디자이너에게 리솔츠를 교육 레퍼런스로 쓴다"고 언급한 상태 → **니즈 이미 확인됨** ② 내부 디자이너 인원 5명 이상 |
| **전개 시점** | 1차 미팅에서 **자연스럽게 꺼낼 수 있는 유일한 유료 트랙**. "디자이너 분들이 리솔츠를 레퍼런스로 본다고 들었습니다. 그럼 한 번 직접 와서 철학을 공유하는 세션을 드려도 좋겠네요" |
| **전략적 의미** | 🌟 **관계 심화 장치** — Track A(무료 미팅) → Track D(유료 교육) → Track C(파트너십)로 자연스러운 에스컬레이션 경로 |

### 2.6 트랙 조합 전개 시나리오

| 시나리오 | 순서 | 기간 | 총가치 (참고) |
|---|---|---|---|
| **베스트** | A → D → C | 3~6개월 | 파트너십 규모 따라 억 단위 가능 |
| **미들** | A → D 또는 A → B | 1~3개월 | 500~2,000만 |
| **워스트** | A만 (관계 유지) | 분기별 뉴스레터 + 연 1회 미팅 | 0원 (but 레퍼런스 자산) |

## 섹션 3. Challenger Teaching 인사이트 3개

> **Why** — 일반 인사이트("브랜드 컨센선스가 중요합니다")는 고수 상대에게 무의미하다. **그가 이미 알고 있지만 명확히 언어화되지 않은 것**을 우리가 먼저 언어화해주면, **"아 이 사람들은 다르다"** 는 인상을 남긴다. 이것이 Challenger Sale의 핵심.

### 인사이트 1 — "리솔츠 롤모델 브랜드들이 공통적으로 놓치는 한 가지: 오프라인-디지털의 표면 일관성이 아니라 '시간 축의 일관성'"

**요지:**
리솔츠를 롤모델로 보는 브랜드들이 모방하는 건 대부분 **"한 시점에서의 비주얼 일관성"**(컬러/폰트/톤)이다. 그런데 리솔츠가 진짜 해낸 것은 **시간 축의 일관성** — 3년 전 런칭 시 포스팅과 지금 포스팅이 **"같은 사람이 성장한 것처럼 보이는 것"**. 이건 가이드라인 문서로 관리되지 않는다. **"브랜드 철학이 의사결정 원칙이 되어 있느냐"** 로 관리된다.

**이퀼립에 적용:**
이퀼립은 2021년 론칭 → 2024년 글로벌 진출 → 2026년 17개국 확장 중. **"시점별로는 좋은데, 3년 연속으로 보면 다른 브랜드처럼 보이는"** 리스크 구간에 진입하고 있다. 이 문제는 가이드라인이 아니라 **"다음 결정을 할 때마다 참조하는 의사결정 원칙 문서"** 로 해결된다.

**액션 기대치:**
이 인사이트를 미팅에서 꺼내면, 김도현 부대표는 **"그걸 어떻게 만드나요?"** 라고 물을 가능성이 높다. 이때 "저희는 그걸 '브랜드 컨센선스 디시전 프레임'이라고 부르는데요"로 받으면 자연스럽게 Track B 또는 Track D 대화가 열린다.

### 인사이트 2 — "브랜드 인큐베이팅 사업의 스케일업 한계점: 3개 브랜드 동시 운영 시 조직은 '가이드라인 공장'이 된다"

**요지:**
브랜드 인큐베이팅 사업자(링케치, 플래티넘브릿지, 브랜드라이즈 등)가 모두 부딪히는 구조적 벽이 있다 — **브랜드가 3개를 넘어가면 내부 디자이너가 "매 브랜드마다 가이드라인을 다시 만드는 공장 노동자"가 된다.** 창의력이 소진되고, 브랜드 간 차별화가 흐려진다.

**해결 프레임:**
성공하는 인큐베이팅 조직은 두 가지 중 하나를 선택한다.
- **옵션 α**: 브랜드 수를 2~3개로 묶고, 대신 각 브랜드의 depth를 극대화 (리솔츠 방식)
- **옵션 β**: 브랜드 수를 늘리되 **"메타 브랜드 OS"**(모든 브랜드가 공유하는 철학 DNA)를 먼저 구축 (플래티넘브릿지는 **아직 β를 안 한 상태로 보임**)

**이퀼립에 적용:**
플래티넘브릿지가 앞으로 신규 브랜드를 인큐베이팅한다면, "이퀼립의 성공 패턴"을 재활용하는 게 아니라 **메타 OS**를 먼저 구축해야 스케일업이 가능. 이 메타 OS를 만드는 게 Track C의 핵심 산출물이 될 수 있다.

**액션 기대치:**
"신규 브랜드 준비하시는 거 있으세요?"로 연결되는 자연스러운 브릿지. 있다고 답하면 Track C 대화 시작. 없다고 답해도 **"앞으로 있으실 때 이 관점으로 접근해보시면 좋겠다"** 로 씨앗을 심음.

### 인사이트 3 — "패키지 일관성의 진짜 적은 디자인이 아니라 공급망/MOQ 구조다"

**요지:**
"브랜드 컨센선스"를 디자인 문제로 접근하면 해결 안 된다. 대부분의 D2C 뷰티 브랜드가 패키지 일관성을 놓치는 이유는 실제로는 **공급망 문제** 다.
- 제조사가 달라서 플라스틱/유리 품질이 다르다
- MOQ(최소 주문 수량)가 다른 신제품은 스펙을 타협한다
- 해외 수출 시 현지 규제로 레이블이 바뀐다
- 리뉴얼 예산이 없어서 구 제품과 신 제품이 섞여 매대에 놓인다

즉, **디자이너를 아무리 뽑아도 공급망을 모르면 풀리지 않는다**.

**이퀼립에 적용:**
이퀼립은 **공구 채널(데이나님 제조)** + **본 브랜드(다른 제조처)** + **글로벌 11개국 채널**의 **3-way 공급망 구조**를 가진다. 패키지 일관성 문제의 70%는 이 3-way 공급망의 스펙 드리프트에서 발생한다고 추정. 디자인 가이드라인만 갱신하면 해결되지 않음.

**액션 기대치:**
이 인사이트가 김도현 부대표에게 **가장 임팩트 있는 teaching**이 될 가능성 높음 — **왜냐하면 그가 이미 막연히 느끼고 있지만 언어화하지 못한 것이기 때문**. 여기서 **"저희는 이걸 디자인 컨설팅이 아니라 '공급망-브랜드 얼라인먼트'로 풀어봅니다"** 로 포지셔닝. 경쟁 컨설팅사와 완전히 차별화됨.

**충돌 관리 주의:**
이 인사이트는 **"공구 상품 제조(데이나님)을 타게팅"하는 것처럼 들릴 수 있음**. 미팅에서 꺼낼 때는 **일반론으로만 설명**하고, **이퀼립 공구 상품에 직접 적용하지 않음**. "예를 들어 일반적인 D2C 뷰티 브랜드의 경우..." 톤으로.

## 섹션 4. 경계심 해소 시나리오 (선제 대응 스크립트)

> **원칙** — 김도현 부대표가 표출하지 않아도 **속으로 갖고 있을 4가지 경계심**을 선제적으로 해소한다. "내가 왜 이 미팅에 응해야 하지?"의 답을 상대방이 직접 묻기 전에 우리가 먼저 건넨다.

### 4.1 "데이나님 건 때문에 꺼려지실 텐데요…" (선제 투명화)

**고객의 속마음:** *"이 사람들이 우리 공구 상품 정보 빼가려는 거 아냐? 데이나님이랑 경쟁시키려는 거 아냐?"*

**선제 스크립트 (미팅 초반 10분 이내):**
> "솔직히 먼저 말씀드려야 할 부분이 있어서요. 저희가 이퀼립과 함께 공구로 나가는 제품의 제조 쪽에 저희의 오랜 파트너 분이 관여하고 계시다는 걸 알고 연락드린 겁니다. 그래서 오늘 이 자리는 **저희가 이퀼립에 컨설팅을 팔러 온 자리가 아니에요.** 브랜드라이즈도 브랜드 인큐베이팅을 하고, 플래티넘브릿지도 인큐베이팅을 하시니까, **같은 업을 하는 사람들끼리 서로 어떻게 일하는지 궁금해서 만나는 자리**로 생각해주시면 좋겠습니다. 혹시 나중에 협업할 일이 생기면 그건 그때 다시 얘기하는 걸로요."

**효과:** 이 한 문장으로 상대방의 경계심 80% 해소. "아 이 사람은 선수구나. 숨기지 않네"라는 신뢰 형성.

### 4.2 "컨설팅 받을 생각은 딱히 없는데…" (파트너십 모드 전환)

**고객의 속마음:** *"우리 내부에 다 있는데 왜 외부에 돈을 써야 하지?"*

**대응 스크립트:**
> "맞으실 거예요. 플래티넘브릿지처럼 수직 통합으로 돌아가는 조직에 컨설팅을 얹는 건 낭비인 경우가 많죠. 그래서 저희는 '가르쳐드리는 관계'보다는 **'공동 제작하는 관계'**를 더 선호해요. 예를 들어 다음에 새로 론칭하실 브랜드가 있을 때, 저희가 디자인/IMC 파트로 같이 들어가서 프로젝트 단위로 일하는 방식이요. 오늘은 그런 가능성이 있는지만 서로 감을 잡아보시면 될 것 같아요."

**효과:** "컨설팅" 프레임에서 **"파트너십"** 프레임으로 전환. Track C의 씨앗을 심음.

### 4.3 "공구 상품 관련해서 정보 교류 좀 가능한가요?" (명확한 경계 설정)

**고객의 속마음:** *"얘네가 데이나님 쪽 정보 좀 알려줄 수 있으려나?"* (혹은 반대로 떠보기)

**대응 스크립트:**
> "그 부분은 저희가 선을 명확히 두려고 해요. 저희 파트너 분의 이익과 이퀼립의 이익이 **교차 또는 충돌할 수 있는 영역**은 저희가 중간에서 어떤 정보도 옮기지 않습니다. 그게 저희가 양쪽 모두와 신뢰를 유지하는 방식이에요. 그래서 혹시 향후 저희와 함께 하실 일이 있다면 **공구 유통 상품과는 분리된 영역**(예: 신규 라인, 본 브랜드 글로벌 확장, 신규 브랜드 인큐베이팅 등)이 될 거예요."

**효과:** 경계가 명확한 사람이라는 인상. 신뢰 +50%. 공구 영역을 **논의 테이블에서 깔끔하게 제거**.

### 4.4 "가격이 얼마죠?" (가격 얘기 우회 방식)

**고객의 속마음:** *"일단 얼마인지 알아야 판단하지"*

**대응 스크립트:**
> "아, 오늘은 가격 얘기 드리러 온 자리가 아니라서요. 다만 궁금해하실 수 있으니까 범위만 말씀드리면 — 저희가 생각하는 협업 방식에는 크게 네 가지가 있는데, **0원짜리(오늘처럼 서로 배우는 자리)부터 브랜드 1건 공동 런칭 시 수천만 단위**까지 스펙트럼이 있어요. **어떤 방식이 맞는지는 서로 감을 잡은 후에 정하는 게 좋을 것 같아요.** 오늘 얘기하다 보면 자연스럽게 보일 거예요."

**효과:** 가격 논의를 **미루면서도 스펙트럼(0원~수천만)을 제시해 상대가 스스로 위치를 가늠**하게 만듦. "비싼 데 아니네" 또는 "장기 파트너십 얘기구나"라는 이해 생성.

### 4.5 추가: "리솔츠는 왜 브랜드라이즈가 알려줄 수 있는 거예요?" (리솔츠 연계 정당화)

**고객의 속마음:** *"한나님이 알려준 번호라는데, 얘네가 리솔츠랑 무슨 사이지?"*

**대응 스크립트:**
> "리솔츠는 저희가 오래전부터 브랜드 철학/디자인 파트로 깊이 협업해온 팀이에요. 그래서 리솔츠의 내부 의사결정 방식, 가이드라인 운영 방식, 팀 문화 같은 것까지 저희가 체화하고 있고요. 부대표님이 리솔츠를 레퍼런스로 보신다는 얘기를 한나님 통해 들었을 때, **'아 이건 직접 만나서 이야기를 나누는 게 훨씬 도움이 되시겠다'** 싶었습니다."

**효과:** **"한나님 소개 → 리솔츠 레퍼런스 → 직접 학습"** 의 논리가 완결. 미팅을 하는 **정당한 이유**가 세워짐.

## 섹션 5. 클로징 — "미팅 후 다음 단계" 3가지 시나리오

> **원칙** — 1차 미팅 끝에 반드시 **다음 접점**을 확정한다. "다음에 연락드릴게요"는 관계 자산 누수. 시나리오별로 다른 클로징 멘트를 준비.

### 5.1 🟢 베스트 시나리오: Track C 파트너십 씨앗 발견

**신호:**
- 김도현 부대표가 "아 그런 방식 우리도 고민 중이었어요"라고 반응
- 신규 브랜드 론칭 계획을 언급함
- 내부 디자이너/기획자 소개 의사를 밝힘

**클로징 멘트:**
> "오늘 얘기 나눈 거 정리해서 **저희가 그려본 가능한 협업 구조 몇 개**를 2주 안에 정리해서 보내드릴게요. 부담 없이 보시고, 혹시 하나라도 괜찮아 보이시면 그때 다시 만나서 구체적으로 얘기해보는 거로요. 그전에 혹시 저희가 추가로 알아두면 좋을 내부 정보(가령 준비 중인 신규 라인 컨셉 같은 거)가 있다면 NDA 걸고 공유해주셔도 좋고요."

**후속 액션 (브랜드라이즈 내부):**
1. 2주 내 **파트너십 가설 제안서** 작성 (Track C 구체화, NDA 템플릿 첨부)
2. 2차 미팅 2주 뒤로 예약 제안
3. 데이나님에게 **투명하게** "이퀼립과 신규 브랜드 파트너십 가능성 논의 중, 공구 영역과는 완전 분리" 공유

### 5.2 🟡 미들 시나리오: Track B/D 소규모 유료 프로젝트 탐색

**신호:**
- "당장은 신규 브랜드 계획은 없고"
- "디자이너들이 리솔츠 얘기 듣고 싶어할 것 같긴 해요"
- "글로벌 확장 쪽은 좀 어렵긴 해요"

**클로징 멘트:**
> "그럼 가장 부담 없이 시작할 수 있는 방식으로, **디자이너 분들 대상으로 리솔츠 철학 공유 세션**을 반나절 가볍게 해드리는 거부터 어떠세요? 저희한텐 그게 서로 어떻게 일하는지 느껴보는 자리가 되고, 그쪽 팀에는 리솔츠 내부 생각을 한 번에 듣는 자리가 되고요. 그다음에 **본격 협업이 필요한 영역이 보이면** 그때 다시 얘기하는 거로."

**후속 액션:**
1. 1주 내 **교육 세션 제안서** 송부 (500~800만 / 반나절)
2. 일정 확정 시 2주 내 세션 실행
3. 세션 당일 자연스럽게 Track B/C 대화로 브릿지

### 5.3 🔴 워스트 시나리오: 상호 인사 수준 + 관계 유지

**신호:**
- 예의는 있지만 대화의 깊이가 얕음
- 구체적 후속 제안에 모호한 답
- "좋은 얘기 잘 들었습니다" 수준 마무리

**클로징 멘트:**
> "오늘 귀한 시간 내주셔서 정말 감사해요. 뭔가 함께 하게 되든 안 되든, 업계에서 좋은 관계로 남으면 좋겠어서요. **저희가 분기별로 업계 트렌드 정리한 짧은 뉴스레터**를 보내드리는데, 받아보셔도 괜찮으시면 이메일만 하나 주시면 감사하겠습니다. 그리고 혹시 나중에라도 협업 필요하시면 언제든 연락 주세요. 도움이 될 수 있는 방식이면 주저 없이 달려올게요."

**후속 액션:**
1. 분기별 뉴스레터 리스트에 추가 (**고객 아님, 관계 자산으로 취급**)
2. 3개월 뒤 안부 메시지 (신제품/소식 트리거로)
3. 6개월 뒤 재접촉 (플래티넘브릿지의 신규 움직임 감지 시)

### 5.4 3 시나리오 공통 클로징 원칙

| 원칙 | 설명 |
|---|---|
| **반드시 "다음 접점"을 걸어둔다** | 제안서 송부 / 세션 / 뉴스레터 — 어떤 형태든 |
| **가격은 마지막에도 꺼내지 않는다** | 베스트/미들 시나리오에서도 "제안서에서 보시죠"로 미룸 |
| **데이나님 언급을 다시 한 번 투명화** | 돌아가는 인사 때 "데이나님 건은 깔끔히 분리" 재확인 |
| **상대의 소개 요청 기회를 잡는다** | "혹시 업계에 저희가 도움 될 분 계시면 소개해주세요" — 자연스러운 추천 요청 |

## 섹션 6. 오늘 첫 전화 스크립트 (수주 전략 관점 요약)

> **주의** — 상세 통화 스크립트는 🎤 상담 코치가 별도로 작성함. 이 섹션은 **수주 전략 관점의 핵심 구조**만.

### 6.1 통화 목표

**유일한 목표: 미팅 약속 확정 (날짜·시간·장소·방식)**

그 외 모든 것은 부차적. 가격·범위·방법론 전혀 꺼내지 않음.

### 6.2 통화 구조 (3분 이내)

| 단계 | 시간 | 핵심 멘트 |
|---|---|---|
| **①오프닝** | 30초 | "한나님(리솔츠) 통해 소개받은 브랜드라이즈 우성민입니다. 부대표님 리솔츠 좋아해주신다는 얘기 듣고 연락드렸어요." |
| **②정당화** | 30초 | "리솔츠 내부 얘기를 직접 전해드리면 좋을 것 같다 싶어서요. 저희가 리솔츠와 오래 협업해왔거든요." |
| **③경계심 선제 해소** | 30초 | "**사실 한 가지 먼저 투명하게 말씀드려야 할 게 있어요.** 이퀼립에서 공구로 나가는 제품의 제조 파트너가 저희와 오래 일해온 분이에요. 그래서 오늘 연락은 컨설팅 영업이 아니라, **서로 같은 업 하는 사이에 한 번 만나뵙는 자리**로 드리는 제안입니다." |
| **④미팅 제안** | 30초 | "60~90분 정도 시간 내주실 수 있으시면, 서울이든 어디든 부대표님 편하신 곳으로 저희가 찾아뵙겠습니다. 다음 주 중 가능하신 시간이 있으실까요?" |
| **⑤일정 확정** | 60초 | (일정 2~3개 제안, 바로 확정) |

### 6.3 통화 중 절대 하지 말 것 (수주 전략 관점)

| ❌ 하지 말 것 | 이유 |
|---|---|
| "컨설팅 제안드리고 싶어서 연락드렸어요" | 경계심 즉시 발동 |
| "견적 드릴까요?" | 이번 건은 가격 영업이 아님 |
| "저희 포트폴리오 먼저 보여드려도 될까요?" | 영업 냄새 |
| 데이나님 이름을 먼저 꺼내지 않기 | "데이나님과 어떤 관계인지" 물어오면 답하되, 먼저 이름 거명은 상대 반응 봐가며 |

### 6.4 통화 중 반드시 해낼 것

| ✅ 해낼 것 | 왜 |
|---|---|
| 한나님/리솔츠 연결 명시 | 정당성 확보 |
| 공구 파트너 관련 선제 투명화 | 신뢰 +50% |
| "영업 아님" 명시 | 경계심 해제 |
| 미팅 일정 2-3개 바로 제안 | 클로징 속도 |

## 섹션 7. 수주 전략 체크리스트 (미팅 전 최종 점검)

### 7.1 D-7 (미팅 1주 전)

- [ ] 데이나님에게 미팅 사실 + 공구 영역 분리 방침 사전 공유
- [ ] 한나님(리솔츠)에게 소개 감사 인사 + 김도현 부대표 성향 리서치 (30분 통화)
- [ ] 플래티넘브릿지 최근 6개월 언론/SNS 동향 체크 (신규 브랜드 기미?)
- [ ] 이퀼립 최신 제품 라인 및 글로벌 출시 국가 정리

### 7.2 D-1 (미팅 전날)

- [ ] 4-Track 설명 자료 미리 출력 (최대 A4 2장, 보여주지 않고 **대화로만 풀되** 필요 시 제시)
- [ ] 리솔츠 내부 철학 사례 3개 미리 뽑아두기 (Teaching Insight용)
- [ ] 경계심 해소 스크립트 3개 음독 연습
- [ ] 클로징 3 시나리오 멘트 숙지

### 7.3 미팅 당일 (60~90분 배분 예시)

| 시간 | 내용 |
|---|---|
| 0~10분 | 오프닝 + 선제 투명화 (데이나님 건) |
| 10~25분 | 플래티넘브릿지/이퀼립 비즈니스 청취 (경청 모드) |
| 25~45분 | 리솔츠 레퍼런스 공유 + Teaching Insight 3개 전달 |
| 45~70분 | 4-Track 관계 설계 소개 (구두로만, 강요 없이) |
| 70~85분 | 질의응답 + 경계심 잔여 해소 |
| 85~90분 | 클로징 (3 시나리오 중 하나 실행) |

### 7.4 미팅 후 24시간 내

- [ ] 감사 메시지 발송 (간단, 2문단 이내)
- [ ] 내부 랩업 노트 작성 (/post-meeting 활용)
- [ ] 데이나님에게 결과 공유 (투명성 유지)
- [ ] 한나님에게 결과 간략 보고 + 감사
- [ ] 시나리오별 다음 액션 실행 (제안서 / 세션 제안 / 뉴스레터)
