# 모수 v2 톤 시드 v1 (클로드 생성)

> **사용법**
> 1. 각 항목의 ★ 추천 1순위를 먼저 본다.
> 2. OK → 슬랙 #diagnosis-v2 또는 본 문서에 `FS·별명=OK`로 회신.

후보 4:

2. ★ 마케터가 만든 4개 채널이 4개의 회사처럼 보이고 있습니다 *(27자, 직설)*
3. 채널마다 다른 회사로 보이는데, 사장님이 일일이 검수해야 합니까 *(28자)*
4. 같은 회사인지 다른 회사인지 채널 4개에서 답이 다릅니다 *(26자)*

부연 ★ (1순위 매칭):
> 3개 채널 이상에서 톤·메시지·비주얼이 따로 가는 회사는 위임 6개월차의 흔한 패턴입니다. 검수자가 사장님 한 명이면, 사장님이 멈출 때 일관성이 멈춥니다.
