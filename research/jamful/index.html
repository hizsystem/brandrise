한 줄 진단
잼플은 "공급(800 센터·100 기업)은 확보, 수요(브랜드 자산)는 비대칭"인 구조. 시리즈A IR 시점에 이 비대칭이 밸류에이션 디스카운트로 이어진다. 우리가 증명할 것은 B2C/B2B 채널 분리 + 시리즈A 트랙션 라인 + 800 트레이너 자산화를 6개월 안에 끝낼 수 있는 곳은 브랜드라이즈뿐이라는 것.
① 표면 요청 추정: "인스타·콘텐츠·광고 잘해주세요" — 폼 미작성·메일 미회신 = "어디부터 정리해야 할지 모르는" 시그널.
② 진짜 니즈: B2B 우선 vs B2C 우선의 비즈니스 우선순위 합의가 먼저, 그 위에서 채널·콘텐츠·광고가 정렬돼야 한다. 이게 L4(근본 원인).
③ 상담의 본질: 60분 안에 김 대표 본인이 우선순위를 입으로 말하게 만드는 자리. Professional 6개월 800만원대가 자연스럽게 만나는 각.

핵심 문제점 & 진단 — 상담 15분 안에 꺼낼 카드

진단 ① 메시지 분열 — "한 채널에 B2C·B2B를 같이 올리면 둘 다 잃는다"

현재
  • 인스타 피드: 운동복 트렌드(B2C) + HR 세미나(B2B) + "남친의 착각"(라이프 릴스) 3트랙 혼재
  • 도메인: jamful.co(B2C) + jamful.co/b2b 분리 운영, 톤은 "자유" vs "효율" 양극
  • 알고리즘이 계정 페르소나를 못 잡음 → 도달 정체 (팔로워 452, 게시물 166)
필요한 것
  • 채널 3축 분리: 인스타 = B2C 단일 페르소나 / 링크드인+블로그 = B2B HR / 잼플 웰니스 나이트 = 오프라인 PR
  • B2B 콘텐츠는 인스타에서 빼고 LinkedIn·뉴스레터로 이전
  • 알고리즘이 "이 계정은 누구 타겟"인지 분류 가능한 일관성 회복

진단 ② 공급-수요 비대칭 — "800 센터 / 기업 100사 vs 팔로워 452 / 앱 1K+"

현재
  • 공급: 800+ 센터 제휴, 도입 기업 100사, 50종목 운영
  • 수요: 인스타 팔로워 452, 앱 다운로드 1K+, 리뷰 29건 — 인프라 대비 1~2% 수준 (추정)
  • 외부 PR(중기이코노미·벤처스퀘어·ZDNet) 자산 보유, 그런데 디지털에서 재유통 안 됨
필요한 것
  • 대표 IP 시리즈화: 영국 IB 출신 창업 서사를 인스타·LinkedIn·뉴스레터에 3채널 재포맷
  • B2C 페이드 테스트 (Meta CPI 기준선 1차 측정)
  • B2B 케이스 스터디(ByteDance·Dr.Blet)를 jamful.co/b2b 상단 임베드

진단 ③ 미사용 자산 — "800 트레이너 = 마이크로 인플루언서 풀"

현재
  • 제휴 센터 트레이너 평균 인스타 팔로워 2,000~5,000 보유 (추정)
  • 잼플이 이를 콘텐츠 협업으로 활성화한 흔적 미확인
  • 코니아랩플러스 윙크 제휴(2025.6)는 채널 파트너십 — 콘텐츠 자산 활용은 별개
필요한 것
  • "잼플 트레이너 50인" 분기 캠페인 → 트레이너 자기 채널에서 잼플 언급 (광고비 0)
  • 벤치: 동종 케이스 클래스파이는 강사 풀 활용으로 CAC -60% (추정)
  • 시리즈A IR에 "Network Effect 작동 중" 1줄로 환산
3개 진단을 관통하는 한 문장
"잼플은 콘텐츠가 부족한 게 아니라, '누구에게 말하는 회사인지'가 합의되지 않은 상태."

Situation — 객관 현재

Implication — 방치 시

① 시리즈A IR 시 "B2B 트랙션은 100사로 증명, B2C 자산 0"으로 디스카운트

③ 팔로워 452 = HR 세일즈 신뢰도 깎임
④ 콰트의 B2B 확장 시 후발 인식

Need-Payoff — 해결 시

① B2B 인바운드 엔진(블로그·웨비나·HR 콘텐츠)
② 시리즈A IR "공급+수요 양쪽 발화" 스토리
③ B2C 인스타 = 입소문 자산 + B2B 영업 시 신뢰자료
④ 1인 마케팅 의존도 해소

Pain Hierarchy (L1 → L4)

L1 표면 요청 "잼플 인스타 운영 좀 도와주세요 / 콘텐츠 퀄리티 높이고 싶어요 / 광고 효율 봐주세요"

L3 전략적 문제 타겟 페르소나 2개로 갈라짐 + 채널 1개 운영 → 알고리즘이 대표 페르소나를 못 잡음. 콘텐츠 KPI(인지/리드/전환) 우선순위 미정의 (추정)
