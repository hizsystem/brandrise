2. 카테고리 정체성 모호 — "종합몰" vs "생활 불편 해결" (High)

사업자 등록상 "전자상거래 소매업/광고대행"이고, 위세브·moneypin 분류상은 "종합몰". 그러나 회사 자기소개(Wanted)는 "생활에서 발생하는 불편함을 없애주는 기업"으로 컨셉 슬로건이 잡혀 있음. → "종합몰 = 무엇이든 판매" vs "불편 해결 기업 = 미션 기반 셀렉션" 사이의 갭. 대표가 머릿속에 그리는 그림은 후자, 외부 노출은 전자일 가능성. 이 갭이 정리되면 자사몰·콘텐츠·SKU 선택의 의사결정 기준이 단번에 분명해짐.

3. 영세 규모 임계점 — 매출 13억대·종업원 2~3인 (High)

매출 13.4억대(2024)는 한국 e-커머스 셀러 기준 "오픈마켓 단독 운영의 천장" 부근. 같은 규모대 셀러들은 보통 (a) 자사몰 D2C로 점프 (b) 카테고리 1개 집중해 광고 효율 극대화 (c) 법인전환·외부 파트너십 도입 셋 중 하나로 다음 챕터를 결정. 종업원 2~3명은 인하우스 마케팅·디자인·CS 풀세트 운영이 사실상 불가 → 외부 파트너 의존도가 자연스럽게 높아질 단계. 브랜드라이즈가 들어갈 자리가 명확함.

S · Situation (현재 상황)

개인사업자 6년차. 네이버·위메프·쿠팡 3채널 100%. 매출 13.4억(2024). 종업원 2~3명. 자사몰·자체 미디어 0. "생활 불편 해결" 슬로건은 자기소개에 정리됨. 위메프 3년 계약 2026.01 만료.

P · Problem (대표 인식 문제 가설)

"오픈마켓 매출은 천장에 도달했다 / 광고 효율이 매년 나빠진다 / 자사몰 만들어야 한다는 건 아는데 어디서 시작할지 모르겠다 / 슬로건은 있는데 외부에 어떻게 쓰는지 모르겠다." — 미팅에서 어떤 단어로 들어올지 청취 후 매핑.

I · Implication (방치 시 비용)

(a) 네이버·위메프 정책·수수료에 매출 직격 위험 (b) 샤오미·중국 직배 가격 공세에 점유율 잠식 (c) 광고 ROAS 매년 악화 → 영업이익 잠식 (d) 6년차 임계점 못 넘기면 정체 또는 역성장 시작 (e) 종업원 2~3인 인건비 부담 가중.

N · Need-Payoff (해결 시 가치)

자사몰 + 자체 미디어 보유 시 매출 30~50억 점프 가능 [추정]. "생활 불편 해결" 미션 D2C 브랜드 = 와이즐리·바램스·올리브영(소형) 영토 진입. 법인 전환 + 외부 투자 옵션 확보. KOTRA 해외 진출 로드맵 가능.

Pain Hierarchy (4계층)

L1 · 표면 요청 "자사몰 만들기 / 인스타 운영 / 광고 대행 / 브랜드 리뉴얼" — 미팅에서 어떤 단어로 들어올지 다양
L2 · 전술적 문제 자사몰 도메인·플랫폼 미정 · 자체 미디어 0 · SKU 비주얼 격차 · 광고 ROAS 악화 · 위메프 재계약 의사결정
L3 · 전략적 문제 "와이림은 무슨 브랜드인가" 정의 공백 — 종합몰? 라이프스타일 D2C? 단일 카테고리 가전? · 채널 포트폴리오 미정 · 법인 전환 타이밍
L4 · 근본 원인 6년 인하우스 → 외부 파트너 협업 근육 부재 · 슬로건은 있으나 외부 표현 경험 없음 · 종합몰 셀러 정체성에 익숙 → 브랜드 셀렉션 사고로 전환 어려움
