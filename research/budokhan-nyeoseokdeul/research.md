### 2.2 가격대

- **개별**: 4,900원 / 355ml 캔 (편의·온라인)
- **6캔 세트**: 27,000원 정가 / 쿠폰가 24,000원 → **캔당 약 4,000~4,500원**
- **포지셔닝**: 일반 무알콜 맥주(카스 0.0, 하이트 제로 등 1,500~2,500원대)의 **2배 프리미엄 가격**

### 2.3 USP

1. **국내 유일 논알콜 전문 크래프트 브루어리** (대기업 라인업의 부수 카테고리가 아닌 본업)
2. 독일산 맥아 + 미국산 홉 + 드라이호핑 — 일반 수제맥주와 동일한 양조 공정
3. **알코올 1% 미만 = 주류면허 없이 온라인 직판 가능** → D2C 채널 자유도
4. 4년 연속 국내외 대회 수상 — **품질 객관 증명**

### 2.4 생산능력·매출 지표 (공개치)

- **월 생산능력**: 약 20,000캔 (세계일보 르포 2024.07 기준)
- **온라인 매출**: YoY 약 2배 증가 (2023→2024)
- **연 매출 성장률**: 70~80% (벤처스퀘어, 2025년 인터뷰)
- **재구매율**: 40%+ (식음료 카테고리 평균 대비 매우 높음)

### 2.5 채널 믹스

- **저알콜/무알콜 합산 판매량**: 2018→2023, 5년간 **약 4배** (약 2,300만L 도달)

- **MZ세대 중심**: "맨정신 라이프스타일"이 멋의 코드로 재정의
- 2026년 2월 기준 "절주·금주" 언급량 **MoM +36%**
- **여성 소비자 확대**: 홈파티/모임에서 무알콜 칵테일·맥주 공유 문화
- 헬시 플레저(Healthy Pleasure), 웰니스 트렌드와 결합
- 글로벌: 프랑스 와인업계가 포도밭 갈아엎을 정도의 구조적 전환

### 3.3 TAM / SAM / SOM

| 구분 | 정의 | 규모 |
|---|---|---|
| **TAM** | 국내 전체 맥주 시장 | 약 4조 원 |
| **SAM** | 국내 무·논알콜 맥주 | 700억 → 946억(2027E) |
| **SOM** | 프리미엄 크래프트 논알콜 (가격 2배대) | SAM의 5~10% = **35~95억** |
| **현재 SOM 점유 추정** | 어프리데이가 SOM 내 1위 가능성 (정확치 미공개) | **5~15억** 추정 |

> SOM은 작아 보이지만 **CAGR 30%+**의 가장 가파른 카테고리. 가격 프리미엄이 마진을 보존.

## 7. 3-Tier 견적 초안

> VAT 별도 / 모든 금액 추정·논의용 / 미팅 후 스코프 확정

### Tier 1 — Foundation: 진단 + 전략 (3주, **800만 원**)

**Why**: 시리즈A IR 직전 단계에서 브랜드 자산 정리가 우선. 미팅 후보 클라이언트 첫 진입에 적합.

- 브랜드 진단 (사명-제품 이원화 정리, 톤·비주얼 일관성 감사)
- 경쟁사·시장 1차 리포트 (HIZ 표준 포맷)
- 마스터 내러티브 1안 ("국내 1호 크래프트 논알콜" 스토리 프레임)
- SNS/유튜브 채널 진단 + 90일 콘텐츠 로드맵
- 워크샵 2회 (대표·핵심 인력)
- 산출물: 진단 보고서 + 전략 페이퍼 + 콘텐츠 캘린더 v0

### Tier 2 — Build (★추천): 전략 + 실행 4개월, **3,000만 원** (월 750)

**Why**: 시리즈A 전 4개월 압축런. 유튜브 0→1, B2B 자산화, IR 데크 보조.

- Tier 1 전체 포함
- BI 톤 가이드 경량 리뉴얼 (CI 풀 리뉴얼 X)
- 인스타 콘텐츠 시스템 (월 12~16건, 운영 + 제작)
- **유튜브 채널 0→1**: 컨셉·포맷 설계, 첫 3~6편 제작 (대표 인터뷰·양조장 비하인드·B2B 케이스)
- B2B 세일즈 시트 + 케이스 스터디 (MUJI / 올리브영 / 삼성 / 호텔)
- PR 보도자료 1회 (시리즈A 연계 가능)
- 월간 KPI 리포트
- 산출물: 위 전부 + IR/세일즈 보조 데크

### Tier 3 — Premium: 풀 파트너십 8개월, **8,000만 원** (월 1,000)

**Why**: 시리즈A 직후 스케일 페이즈. 카테고리 리더십 확정.

- Tier 2 전체 + 기간 확장
- 퍼포먼스 마케팅 운영 (월 광고 집행비 별도, 권장 월 500~1,500)
- 인스타·유튜브 풀 운영 + 숏폼 트랙 추가
- PR 캠페인 분기 1회 (소버 큐리어스 트렌드 연계)
- 패키지 디자인 리프레시 옵션
- 글로벌 진출 준비: 영문 브랜드 자산, Untappd 활성화 전략
- 데이터 대시보드 구축 (월간 KPI 자동화)
- HIZ 전담 그로스 파트너 1인 + 디자인·콘텐츠 풀
