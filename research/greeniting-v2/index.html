핵심 문제점 & 진단

1. 가격 허들의 진짜 원인은 "가격"이 아니라 "카테고리 프레임" (Critical)

현재 상태
  • DPPH 92%·안토시아닌 186mg 검증 데이터 보유 → 상세페이지에 미기재
  • 소비자가 "식품(주스) 가격표"로 제품을 평가 → 네이버 GFA "가격대 높아 성과 부진"
  • 스무디 카테고리 = 우후죽순 경쟁사 난립(해독·염증 키워드)
이상적 상태
  • 공인 데이터를 첫 화면 전면화 → "데이터로 증명된 항산화 솔루션"
  • 가격 비교 기준선 전환(주스 3천원 → AG1식 일당 환산 웰니스)
  • "항산화 1축" 카테고리 점프로 레드오션 탈출

2. 성장 정체의 구조적 원인 = 단일 의존 (High)

공구 53% + 유료채널 메타 only. 폭발 성장(4억→18억)은 인플루언서 직접 공구가 견인했으나 공구는 휘발·확장 한계. GFA·유튜브·틱톡 안착 실패 = 리브랜딩 없이는 새 채널에서 가격 저항 반복. "그 이상의 성장요인이 없는 상황"의 진짜 원인.

3. 9월 김미경TV(189만) = 준비 없으면 "스무디 영구 각인" 리스크 (High)

9월 1주 김미경TV 출연 + 토마토스튜 런칭 동시. 전환 장치(랜딩·CTA·카테고리 내러티브) 없이 189만을 받으면 도달만 크고 유입 0(검증 패턴: 미례 3만뷰/매장 유입 -22.6%) → 189만 명에게 "또 하나의 스무디 브랜드"로 도장.

표면 요청 (대표 원문)

"제품이 식품 카테고리에 속하다보니 높은 가격이 구매 허들이 되어 성장에 한계 / 더 이상 식품·스무디 브랜드가 아닌 '데이터·효과·과학적' 입증된 항산화/웰니스 브랜드로 성장하고 싶다 / 우후죽순 경쟁사 / 월 매출 2억+이나 그 이상의 성장요인이 없는 상황 / 가장 시급한 과제: 리브랜딩"
감정 톤: 자부심("데이터로 증명된 건강") + 답답함("그게 매출/가격에 반영 안 됨")의 공존. 준비된 대표(벤치 4종 정확히 호명) + 조급함(9월에 큰 베팅 집중).

SPIN 분석

Situation (현황)

매출 18억·공구 53%·메타 ROAS 3.2·4인 조직. KATR DPPH 92%·HPP·비가열·유기농·저당 보유.

Problem (표면 문제)

식품 카테고리라 가격이 허들 → 성장 한계. 공구 의존 외 자생 엔진 없음. 우후죽순 경쟁사로 차별성 희석.

Implication (방치 시)

공구 식으면 매출 직하강 / 스무디 가격 출혈전 → ROAS·마진 잠식 / 데이터 사장 → 경쟁사가 "데이터" 선점 / 9월 189만을 "스무디"로 받아 카테고리 점프 기회 영구 상실.

Need-payoff (해결 가치)

카테고리 점프 → 가격 비교 기준선 전환(주스 3천원→일당 환산 웰니스). 공구 53%→30%대 분산으로 매출 안정. 189만을 "웰니스 브랜드"로 수신 = 변곡점.

고민 계층 분석 (Pain Hierarchy)

L1 표면 ★★★★★ "리브랜딩을 하고 싶다 (식품→웰니스 브랜드로)" — 대표 직접 명시
L2 전술 ★★★★★ 보유 과학 데이터(DPPH 92%)를 마케팅 자산으로 전환 못 함 — 상세페이지 미기재(즉시 검증 가능 팩트)
L3 전략 ★★★★☆ 스무디 카테고리 안에서 경쟁하는 한 가격 프리미엄을 정당화할 카테고리 내러티브가 없음
L4 근본 ★★★★☆ 공구 53% 의존을 탈피한 자생적 성장엔진(브랜드 자산·검색·구독)이 부재 = "성장요인 없음"의 진짜 원인
핵심 진단: 대표가 말한 "리브랜딩"(L1)은 진짜 통증이 아니다. 리브랜딩(네이밍·CI)만 하고 L2~L4를 안 풀면 = "이름만 바꾼 똑같은 스무디 브랜드"가 되어 통증이 재발한다. 이것이 막아야 할 Type III Error(잘못된 문제를 정확히 푸는 오류).

핵심 질문 스크립트 (반드시 물어볼 5개)

Q1 · 예산 (필수)
"6개월 호흡으로 월 단위 마케팅·브랜딩 예산은 어느 선이세요? 지금 메타에 월 1,200만 안팎 쓰시는 것 같은데 — 유지하면서 추가인지, 이 안에서 재배분인지요?"
의도: Professional(월 500~700) vs Premium(월 900~1,000) 적합도. 메타에 묶였으면 "재배분 설계"로 부담 제거.
분기: "여력 있다"→Professional 정조준 / "추가 부담"→재배분 재프레임+Quick Win / "모르겠다"→"효과부터 보시죠" 파일럿 연결
Q2 · 리브랜딩 범위 (필수, 갭의 핵심)
"'리브랜딩'이 네이밍·로고·CI까지 새로 가는 시각적 리브랜딩이세요? 아니면 '그리니팅'은 두되 카테고리 포지셔닝·메시지·상세페이지·콘텐츠 차원에서 '항산화 웰니스 브랜드'로 다시 입히는 쪽이세요?"
의도: L1(이름 교체) vs L2~L3(의미 재정의) 갭 확인. 9.7만 팩·관심고객 9,932·리뷰 4.9 자산을 버리는 풀 리네이밍은 위험.
분기: "이름까지"→⚠️신중("AG1도 'Athletic'만 뺐지 브랜드는 안 버렸어요") / "의미 재정의"→정답, 즉시 Teaching 1 연결
Q3 · 수출 우선순위 (필수)
"6개월 안에 무게를 국내 리브랜딩에 두세요, 미국 진출에 두세요? 상온 스튜 개발이 수출용인지 국내 카테고리 점프용인지도 궁금하고요."
의도: 4명 조직에 둘 다는 과부하 → 우선순위 못 정하면 둘 다 실패(Daily Harvest 패턴). Premium 전환 여부 판별.
분기: "국내 먼저"→Professional / "수출 1순위"→Premium GTM / "둘 다"→⚠️우선순위 강제+Daily Harvest 카드
Q4 · 김미경TV 189만 활용 (필수, 타이밍)
"9월 1주 189만 노출을 무엇으로 받으실 계획이세요? 검색해 들어온 사람이 도착할 랜딩·구매 동선, 그리고 '스무디'가 아니라 '데이터 웰니스 브랜드'로 보일 준비가 돼 있을까요?"
의도: 9월이 사실상 리브랜딩 데드라인임을 인지하는지. 전환 장치 없으면 유입 0 위험을 189만 규모로 반복 = Teaching 2 직결.
분기: "준비 안 됐다"→데드라인=긴급성 전환 / "이벤트 준비 중"→"내러티브가 먼저, 역산하면 6월 착수가 마지노선"
Q5 · 자사몰 재구매율 (필수, L4 검증)
"공구로 처음 산 분들이 자사몰로 다시 오시는 비중이 어느 정도세요? 공구 고객이 그리니팅 '브랜드 팬'으로 남는지, 인플루언서 따라 한 번 사고 마는지가 궁금해서요."
의도: L4(공구 의존 vs 자생 엔진) 검증. 재구매 안 넘어오면 = 매출이 "인플루언서 자산"에 귀속 = 채널 다변화 시급성 증명.
분기: "재구매 낮다"→"그게 정체의 진짜 원인, 리브랜딩이 이 엔진을 만드는 일" / "재구매 된다"→구독 설계로 연결

Challenger Teaching 포인트 (Phase 4)

대표보다 깊게 메커니즘을 해부해야 "막연한 브랜딩 업체"와 차별화
① 리브랜딩은 비용이 아니라 가격 프리미엄 엔진 (AG1) — Athletic 제거+카테고리 재정의+RCT 전면화 → 개명 후 매출 2배. "DPPH 92%는 AG1의 RCT와 같은 무기인데, 그걸 상세페이지에 안 걸고 계세요."
② 189만을 내러티브 없이 받으면 '스무디'로 영구 각인 — 전환 장치 없는 대형 노출은 도달만 크고 유입 0(미례 3만뷰/유입 -22.6%). "9월은 '데이터 웰니스 브랜드'로 보일 마지막 무대."
③ KATR 공인데이터 안 쓰는 건 무료 탄약 방치 — ARMRA가 '의사 개발' 권위로 프리미엄화한 것보다 더 강한 정량 데이터를 묻어둠. "추가 비용 0원으로 전환율 올릴 무기."
🃏 Daily Harvest 보너스 (권위+안심) — 모토 브랜드가 2022 리콜→2025.5 Chobani 피인수(독립 실패). "그들이 무너진 원물 검증은 대표님이 HPP로 막고 계시고, 흐린 카테고리는 '항산화 1축'으로 좁히면 됩니다."
