## 2. 브랜드 정보

### 2-1. 브랜드 아이덴티티

| 항목 | 내용 |
|------|------|
| **브랜드명** | 집지켜 (SaveHome) |
| **슬로건** | "가장 싸고 안전한 집 찾기" / "전월세 거래를 쉽게" |
| **핵심 철학** | 세입자의 보증금을 지키고, 주거 정보 비대칭을 해소 |
| **타겟 고객** | 자취생, 신혼부부, 청년 (20~30대 전월세 세입자) |
| **포지셔닝** | "전월세 세입자를 위한 원스톱 주거 안전 플랫폼" |
| **차별화 요소** | 주소 하나로 40가지+ 데이터 기반 1분 권리분석, 행복주택 청약 데이터 분석 |

### 2-2. 브랜드 톤 & 매너

- **친근하고 실용적**: "이런 것도 공짜라니" 같은 사용자 반응을 유도하는 접근성
- **데이터 중심**: 숫자와 분석 결과를 명확하게 제시
- **세입자 편**: 명확한 고객 관점 (임대인이 아닌 임차인)
- **UI/UX**: 사용자 평가 "토스급으로 세련되고 군더더기 없이 깔끔"

### 2-3. 컨셉 키워드

`안전` `보증금` `권리분석` `행복주택` `청년주거` `전세사기 예방` `데이터 기반`

1. **인스타그램 139K 팔로워** -- 프롭테크 스타트업 중 최상위권, 오가닉 바이럴 역량 입증
2. **앱 평점 4.9** -- 사용자 만족도 최고 수준, "토스급 UX"라는 평가
3. **데이터 기반 콘텐츠** -- 청약 분석 리포트 등 자체 데이터 활용한 PR 가능
4. **명확한 타겟/포지셔닝** -- 청년/자취생/신혼부부로 일관된 메시지
5. **토스 등 대형 파트너와의 제휴** -- 브랜드 신뢰도 확보

**약점:**
1. **네이버 채널 부재/부실** -- 부동산 정보 검색의 핵심 채널에서 존재감 약함
2. **유튜브 비활성** -- 부동산/주거 콘텐츠의 핵심 플랫폼 미활용
3. **멀티채널 전략 부재** -- 인스타그램 1개 채널에 과도하게 의존
4. **퍼포먼스 마케팅 미집행** (추정) -- 유료 유저 획득 채널 부족
5. **콘텐츠 시스템 미확인** -- 브랜드 가이드라인, 콘텐츠 캘린더 등 체계적 운영 여부 불명

## 13. 상담 전략

### 13-1. 핵심 레버리지 문장

> "집지켜는 앱 평점 4.9, 인스타 14만 팔로워라는 **검증된 제품력**을 갖고 있지만, 유튜브/네이버/퍼포먼스 채널이 사실상 비어 있어 **44만 유저의 2~3배 성장 잠재력이 잠자고 있습니다.** 시리즈A 전 지금이 마케팅 기반을 갖출 최적의 타이밍입니다."

### 13-2. 상담 흐름 시나리오

**1단계: 공감 형성 (5분)**
- "집지켜 서비스를 직접 써봤는데, 주소 하나로 1분 만에 분석되는 UX가 정말 인상적이었습니다"
- "인스타그램 14만 팔로워, 앱 평점 4.9라는 숫자가 제품력을 증명하고 있습니다"
- "전세사기 예방이라는 사회적 가치를 만드는 서비스라서 의미 있게 봤습니다"

**2단계: Pain Point 탐색 (10분)**
- "현재 마케팅은 어떤 팀/체계로 운영하고 계시나요?"
- "인스타그램 외에 유튜브나 네이버 채널은 계획이 있으신가요?"
- "시리즈A 일정은 어떻게 잡고 계시나요? 투자자들이 마케팅 지표를 어떻게 보시나요?"
- "유료 전환율이나 리텐션 지표 중 개선하고 싶은 부분이 있으신가요?"

**3단계: GAP 제시 (10분)**
- "리서치 결과, 인스타그램은 잘 하고 계시지만, 유튜브/네이버/퍼포먼스 3개 채널이 거의 비어 있습니다"
- "부동산 정보를 검색하는 사람의 60%+가 네이버를 이용하는데, 네이버에서 집지켜가 노출되지 않고 있습니다"
- "44만 유저가 있는데, 이 중 리텐션/재방문 비율은 어떠신가요? 콘텐츠 시스템이 갖춰지면 이 유저를 다시 데려올 수 있습니다"

**4단계: 제안 (5분)**
- Tier 2 (Build)를 중심으로 제안
- "4개월 안에 네이버/유튜브 채널을 만들고 콘텐츠 시스템을 갖추면, 시리즈A 피칭 시 마케팅 KPI를 보여줄 수 있습니다"
- "월 600~800만원이면, 정규직 마케터 1.5명 채용비와 비슷한데, 전략 + 실행 + 4개 채널 운영이 가능합니다"

**5단계: 클로징 (5분)**
- "Tier 1 (진단 500만원)으로 시작하고, 결과 보시고 Tier 2로 확장하는 것도 좋은 방법입니다"
- "다음 주에 무료 30분 진단 미팅을 잡아볼까요?"
