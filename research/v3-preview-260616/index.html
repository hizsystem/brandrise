· IMC 파트너십 = 급별(월 60·실무 / 140·표준 / 300·디렉터). 단가 공유 원칙(브랜드 불문 동일).
· PREP의 📊엑셀/📄HTML 내보내기 = 실제 견적 SELECTION RESULT 디자인(전 항목×L1/L2/L3 + 선택 표시 + TOTAL).
