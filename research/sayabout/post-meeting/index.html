0-4. 3-Tier 제안 재조정 가이드

중요 변경 · 클라이언트 명시 예산 "1차 2천만 전후"는 기존 Essential(1,500~2,500만) 대역. Professional(3,000~4,500만)을 그대로 추천 Tier로 가면 클라이언트 기대치 위로 1,500만 이탈 → 수주 확률 손실. Essential 가격대에서 출발 + BI 정비·해외 상표권을 모듈로 분리해 단계적 추가 구조로 재구성.
재조정 포인트구체 액션
① Tier 무게중심 이동 Essential = 추천 Tier로 전환 · Professional은 "BI/해외 상표권 정비 모듈 추가 시" 단계 업그레이드 경로로 재배치
② 모듈식 견적 구조 "파데프리 2종 디자인 (Essential 코어)" + "BI 체계 정비 모듈" + "해외 상표권·현지화 모듈" + "기존 pppf 리뉴얼 모듈" — 각각 가격·기간 분리 제시
③ 출시 압박 명시 "빠르면 6월 출시" → 5월 1~3주 본 작업 완료 필요 · 견적 회신에 역산 일정표 첨부로 의사결정 가속
④ 클로징 후크 대표 발언 "객관화 안 된 부분"을 인용한 객관화 진단 1장 무료 부속 · 단가 인상 없이 수주 신호 강화
⑤ 업셀 경로 명시 대표 본인이 매출/입점 전략 수반 희망 → 견적 본문 끝에 "디자인 본 작업 종료 후 마케팅·입점 별도 단계 가능" 한 줄 명시

범위상세

기존 pppf 2종리뉴얼 병행 준비 — 기존 재고 소진 시점 기준으로 신 디자인 교체 · 디자인 체계는 지금부터 통일 설계

2. 권윤정 대표가 짚은 4가지 핵심

CORE A · 자사몰 퍼널이 먼저

2-Section 프레임 — 보여지는 섹션 vs 보여지지 않는 섹션

" 보여지는 섹션(콘텐츠·마케팅)과 보여지지 않는 섹션(자사몰 세팅)이 있다. 들어왔을 때 이걸 받아들일 준비가 되었는가, 퍼널이 완성되었는가가 먼저다. 이게 안 되는데 이것부터 하면 새어 나간다. 이게 다 돈이다.
— 권윤정 대표

자사몰 기본 3요소 (Phase 1): ① 제품 필요 정보 ② 리뷰 세팅 ③ 결제 편의 (회원가입 유도 · N페이 비회원 결제는 고객 정보 탈취 우려로 지양)

Phase 2 (제품 라인업 확장 시): 잔여 시간 표시 · 무료배송 트리거 등 업셀링 기능

CORE B · 타겟·페르소나 없이는 채널 전략 없음

"20~50대"는 너무 넓다 — 라이프스타일까지 특정 인물

" 20~50대 이렇게 말하면 너무 넓다. 페르소나는 특정 인물. 라이프스타일까지 딱 집어야 한다. 이 시장에 대한 이해를 바탕으로 현금을 누가 많이 사용하는지부터. 올리브영에서 2만원대로 싸게 사는 로직이라면 그 풀에 갇혀 있는 상태를 꺼내놓아야 된다.
— 권윤정 대표

리솔츠·베지어트 플레이 공식: 프리미엄 라인으로 미국·일본 디자인부터 설계 → 그 포지션에 맞춰 국내도 좋아하게 플레이. (유비발디가 "아네사·시세이도 일본에서 잘 나가는 것" 언급에 대한 권 대표 설명)

CORE C · VIP 직접 관리

리솔츠·미슐란 방식 — 개인번호·이름·물류까지

" 저는 개인번호를 바꿔야 돼요. 누가 뭘 했는지 다 안다. VIP들에게 편지 쓰고 1년마다 선물. 미슐란 대표도 고객 이름·전화번호 다 외움, 물류도 직접 다 한다.
— 권윤정 대표

유비발디 즉시 액션: 주문 들어온 10명에게 문자/전화로 구매 감사 + 유입 경로 질문 → 작아도 부정적 안 가고 오히려 "나만 알고 싶은 브랜드" 애착 생성. CS 부담 우려 대표에게 권 대표 답변: "한 번도 부정적 경험 없었다."

CORE D · 메타 광고 운영 호흡

템플릿만 줘도 의미 없다 — 라이브·전환·A/B 호흡이 핵심

" 광고 소재 템플릿을 만들어줘서 이미지·카피 넣게 세팅해드려도, 라이브하고 빠르게 소재 전환하고 A/B 테스트 호흡이 안 되면 아무 의미 없다.
— 권윤정 대표

  • 큐텐 입점 직전 + 일본·동남아 시딩 진행 중 → Premium "해외 확장" 모듈 즉시 연결
  • 4/28 대인아 세션 공동 참석 = 자연스러운 추가 접점 · 차파도(히즈 클라이언트) 연결 가능
  • "클라이언트끼리도 친해지는 상생 구조" 권 대표 본인 선언 → 심리적 진입 장벽 낮음
  • 자사몰 세팅 미완 = 즉시 가치 제공 포인트 · Quick Tip 메모 재료

리스크 (Risk)

  • 대표 브랜딩 영역 약점 자인 → 의사결정 방향 변동 가능 (음표 유지·폐기 / 타겟 / 클래식 vs 세련 / 출연 여부 등 다중 고민 진행 중)
  • 본업·유비발디 1인 캐리 → 출산 후 콘텐츠 출연 NO 선언 + 대표 대역폭이 가장 큰 병목
  • 예산 구두 선 미확정 → 견적 과소·과대 모두 리스크 · 매니저를 통한 기대치 선 탐지 필요
  • 기존 pppf 재고 소진 vs 리뉴얼 단절 관리 — 잘못하면 재고 폐기 또는 무드 충돌
  • 브랜드명 "유비발디" 가독성 — 미팅에선 불거지지 않았으나 권 대표 사전 지적 · 클라이언트 감정선 관리해서 전달 방식 설계 필요
