핵심 문제점 & 진단

1. 광고/마케팅 경험 제로 — 목표 매출과의 괴리 (Critical)

현재 상태
  • 퍼포먼스 광고 집행 경험 전무 (Meta/네이버/구글 모두)
  • 인스타그램 @kymu_official = 유일한 마케팅 채널
  • 네이버/유튜브/틱톡 채널 없음
  • "브랜드 마케팅"이 가장 시급한 과제라고 대표 직접 언급
이상적 상태
  • Meta/네이버 광고 셋업 + ROAS 기반 최적화
  • 인스타 파운더 스토리 시리즈 + 릴스 운영 체계
  • 네이버 브랜드 검색 최적화 + 블로그 시딩
  • 올리브영 입점 레디 수준의 브랜드 자산

2. 자사몰 오픈했으나 유입 구조 부재 (Critical)

자사몰(kymubeauty.com, Cafe24)은 오픈 완료. Base / Lip+Cheek / Set 3개 카테고리 구성. 그러나 유입을 만드는 구조(광고, SEO, 콘텐츠)가 전혀 없어 자사몰이 있어도 트래픽이 발생하지 않는 상태. "가게는 열었는데 간판도, 길 안내도 없는" 상황.

3. 파운더 스토리 자산 미활용 (High)

암 투병 경험 + 글로벌 뷰티 기업 7년 경력 = PR 관점에서 최상급 소재. 그러나 보도자료 1건(대한청년일보) 외에 체계적 스토리텔링이 없음. 이 스토리를 체계적 브랜드 시스템 없이 소비하면 다시는 "첫 론칭" 모멘텀을 활용할 수 없음.

7. SWOT 분석

Strength (강점)

  • 글로벌 뷰티 기업 7년 실무 경력 (SCM/운영)
  • 진정성 있는 파운더 스토리 (암 투병 → 창업)
  • 제품 차별화 (스킨케어 77% BB, 12회+ 샘플링)
  • 사회적 가치 비전 (암 환자/경력단절 여성 지원)
  • 클린뷰티 + 하이브리드 = 시장 트렌드 적합

Weakness (약점)

  • 브랜드 인지도 제로 (네이버 검색 미노출)
  • 광고 집행 경험 전무 (Meta/네이버/구글 모두)
  • 인스타그램 외 마케팅 채널 없음
  • 자사몰 오픈했으나 유입 트래픽 부재
  • 소규모 팀, 재무/자금력 불명
  • 대표 전문 분야가 SCM — 마케팅 역량 갭

Opportunity (기회)

  • 클린뷰티 + 하이브리드 = 글로벌 메가트렌드
  • 파운더 스토리가 PR/바이럴에 극도로 강력
  • K-뷰티 글로벌 성장세 지속
  • 올리브영 클린뷰티 카테고리 확대
  • 인디 브랜드 크라우드펀딩 → DTC → 올리브영 경로 확립

Threat (위협)

  • BB크림 카테고리 성숙/레드오션
  • 대기업(아모레, LG생건) 및 인디 강자 자원 격차
  • 1인 창업 리스크 (번아웃, 실행 한계)
  • 텀블벅 펀딩 실패 시 모멘텀 상실
  • 클린뷰티 경쟁 과열 (차별화 어려움)

11. 상담 전략 & 코칭

상담 전 필수 확인 사항

설문 사전 확인 완료 사항: 타겟(민감성 20-40 여성), 시급 과제(브랜드 마케팅), 목표 매출(7,800만/6개월), 광고 경험(전무), 기대 결과(런칭 체크리스트 조언). 아래는 상담에서 추가로 확인할 사항.
핵심 Q1. 텀블벅 결과 & 초기 반응
"텀블벅 펀딩이 어떻게 마무리됐나요? 고객 피드백 중 가장 많이 들은 건?"
달성률 = 시장 검증 강도. 후원자 피드백 = 마케팅 메시지 소재. "커버력+예쁘게 무너짐"이 실제 반응과 일치하는지 확인
핵심 Q2. 마케팅 월 예산
"광고비 포함해서 월 마케팅에 투자할 수 있는 금액이 얼마 정도인가요?"
월 300만(광고+컨설팅) = 최소 권장선. 매출 시나리오에서 월 광고비 300~500만이 목표 달성 구조. 현실적 예산 파악
핵심 Q3. 자사몰 현황
"자사몰 오픈 후 일 방문자가 어느 정도인가요? 주문이 들어오고 있나요?"
현재 트래픽/전환 데이터 = 기준선. 0에 가까우면 "가게는 열었는데 손님이 없는 상태" 프레이밍 활용

60분 상담 타임라인

0~10분

Phase 1: 아이스브레이킹 + 정보 확인

텀블벅 결과 축하/공감. 클라시크-키므 관계, 법인 구조, 자금 상황 자연스럽게 확인

10~25분

Phase 2: Pain Point 탐색

"텀블벅 이후 어떤 채널에서 판매하실 계획이세요?" → 대부분 명확한 답 없을 것. 현재 디지털 프레즌스 부재를 객관적으로 제시

25~40분

Phase 3: 파운더 스토리 자산 강조

"대표님의 경험과 이야기가 가장 큰 브랜드 자산입니다. 이것을 체계적으로 만들어야 합니다." 스토리 자산의 PR 가치를 구체적으로 제시

40~50분

Phase 4: 솔루션 제안

예산에 맞는 Tier 제안. Foundation(300만)부터 시작 → 결과 보고 Build 확장 제안

50~60분

Phase 5: 다음 단계 합의

Foundation 착수 시점 합의. 또는 추가 검토 후 피드백 날짜 확정
