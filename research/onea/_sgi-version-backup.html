핵심 문제점 & 진단

1. 광고 의존 → 브랜드 자산 0 리셋 (Critical)

클라이언트가 직접 명시한 가장 시급한 과제 = "광고 의존형 판매 구조에서 벗어나 브랜드 기반의 자연 유입·인지도 생성". 본인의 표현: "플랫폼에서 아무리 매출을 내도 신제품을 런칭할 때마다 제로부터 다시 시작." 매출이 CPC에 종속돼, 광고비를 멈추면 매출도 멈추고, 쌓아온 게 다음 제품으로 이전되지 않는 구조. = 지금 버는 4,000만은 "자산"이 아니라 "임대".
교차 확인 — 제품은 되는데 '브랜드'가 없다: 광고기간 4,000만 매출 + 구매의도 키워드 전환 개선은 제품·운영 역량이 이미 작동한다는 증거(0→1 통과). 문제는 컨셉이 없는 게 아니라, 그 매출 위에 재구매·자연유입·신제품 레버리지를 만드는 브랜드 자산층(인지도·콘텐츠·팬)이 비어 있는 것. 즉 과제는 "광고를 더 잘 돌리기"가 아니라 "광고가 만든 트래픽을 브랜드 자산으로 환전하는 시스템"을 까는 일.
현재 구조증상구조적 결과판단
CPC 광고 → 오픈마켓 단건 전환광고 ON=매출 / OFF=0매출이 광고비에 선형 종속, 마진이 광고·수수료로 유출의존 — 끊어야 할 고리
플랫폼(스토어·쿠팡) 노출 의존알고리즘·수수료 종속고객 데이터·관계를 플랫폼이 소유, 브랜드는 임차인탈종속 필요
브랜드 자산층 (콘텐츠·인지도·팬)사실상 부재 (자가진단 "콘텐츠 자산 부족")신제품마다 재사용 가능한 자산 = 여기서 만들어짐코어 과제 — 여기에 투자

2. "미니멀 디자인" 포지션 — 이미 선점됨 (High)

현재 상태
  • 오네아의 핵심 차별점 = "미니멀한 디자인 + 실용성"
  • 그러나 ±0(일본 미니멀 원조)·레꼴뜨(감성 컬러 디자인가전)가 이미 그 자리를 점유
  • "미니멀"만으로는 0.5초에 "왜 오네아인가"가 서지 않음 = 차별 약함
이상적 상태
  • 단품 "미니멀 그라인더" → "미니멀 키친 라이프스타일 시스템 브랜드"로 격상
  • 그라인더 → 커피용품 → 소형가전을 하나의 룩·세계관으로 묶는 우산 = 경쟁사가 못 가진 축
  • 오늘의집 연출컷·생활제안으로 "식탁 위 오브제"의 맥락을 파는 브랜드

3. 1인 기업 — 실행 리소스가 최대 제약 (High)

대표 1인이 상품기획·소싱·온라인운영·CS·광고를 모두 직접, 디자인·콘텐츠만 외주. SNS·숏폼·인플루언서·공동구매·신제품을 동시에 돌릴 손이 없음. → 전략을 "많이" 주면 떠버린다. 컨설팅 성패는 실행 가능한 경량·고레버리지 SOP(외주 디렉션 체계 + 우선순위 압축)에 달림.

4. 타겟–채널 정합성 점검 필요 (Medium)

타겟은 30~50대 주부인데, 향후 전략은 숏폼·인플루언서. 숏폼 주 소비층과 타겟 사이에 간극이 있을 수 있음 → 인스타·오늘의집 콘텐츠·공동구매(30~50대 주부 친화)와 숏폼의 비중을 데이터로 재설계해야 예산이 새지 않음.

2. 주력 제품 / 서비스

제품 구성·특징 전략 포인트
오네아 전동 소금·후추 그라인더
(페퍼밀) — 현 대표상품		 미니멀 디자인 + 한 손 사용 실용성. 카테고리 표준 = 소금+후추 2개 세트 구성. (충전/조절/세라믹날 등 상세 스펙 확인 필요) "식탁 위 오브제" 가치 = 인테리어 구매동기. 세트·기프트 수요 존재
커피용품 (확장 예정) 드리퍼·캐니스터·핸드밀 등 추정 — 홈카페 트렌드 연계 그라인더와 동일 세계관(미니멀 키친)으로 묶을 수 있는 인접 카테고리
소형 주방가전 (확장 예정) 가전테리어 트렌드 직결 — 디자인 우선 소형가전 브랜드 우산이 서면 신제품마다 0에서 시작하지 않는 레버리지
제품 해석: 라인업 로드맵(그라인더 → 커피 → 소형가전)이 이미 "미니멀 키친 라이프스타일"이라는 하나의 세계관을 가리키고 있음. 즉 오네아의 본질은 "그라인더 셀러"가 아니라 키친 라이프스타일 큐레이터에 가깝다. 그런데 현재는 그라인더 단품을 광고로 파는 단계에 머물러, 제품 로드맵 ↔ 브랜드 내러티브의 간극이 가장 큰 미실현 자산. 이 간극을 메우는 것이 차별화의 핵심.

현재 마케팅 성과 (클라이언트 자가진단)

구분내용전략 함의
성과 좋았던 것네이버·쿠팡 CPC, 구매의도 높은 키워드 중심 집행 → 광고기간 4,000만 매출, 데이터 기반 노출·전환율 개선퍼포먼스 운영 = 상대 강점(이미 작동)

아쉬웠던 것 ②외부 플랫폼 의존성 높음탈종속(자사 채널·자연유입) 필요
핵심 발견: 클라이언트의 자가진단이 이미 정답의 방향을 가리킨다 — "광고는 되는데(강점), 콘텐츠 자산이 없고(약점) 플랫폼 의존이 높다(약점)". 즉 컨설팅은 백지에서 시작하는 게 아니라, 작동 중인 퍼포먼스 엔진 위에 '브랜드 자산 + 자사 채널'을 얹어 의존을 끊는 일. 상담에서 이 자가진단을 그대로 인정하며 진입하면 신뢰가 빠르게 선다.

반드시 물어볼 질문 (5개)

Q1 — 실매출 구조 (라포+진단) ★
"광고기간 4,000만이면 운영을 정말 잘하신 거예요. 그 기간이 몇 개월이었고, ROAS·객단가·재구매율은 대략 어느 정도였나요?"
의도: 데이터 언어로 진입 + 광고 의존 강도·재구매 베이스 확인. 재구매가 낮으면 "자산화"의 필요를 본인이 체감.
Q2 — 자산 0 리셋 (Phase 3) ★ 핵심
"'신제품 런칭마다 제로부터'라고 하셨는데 — 지금 광고를 멈추면 매출이 어느 정도 남나요? 자연유입·재구매로 남는 베이스가 있으세요?"
의도: "광고 OFF=0"을 본인 입으로 확인 → 브랜드 자산층 부재를 자각. 여기서 불안이 드러나면 Professional 명분.
Q3 — 차별 축 (Phase 3)
"미니멀 디자인이 강점이신데, ±0·레꼴뜨도 미니멀로 유명하잖아요. 고객이 '왜 오네아'냐고 물으면 지금 한 문장으로 답이 되세요?"
의도: "미니멀만으론 차별 약함"을 자각시키고 "확장 라인업·라이프스타일 세계관"이라는 답으로 자연 연결(Teaching 2).
Q4 — 범위/실행 주체 ★ 1순위 (Phase 5)
"전략·브랜드만 정리해드리면 되나요? 아니면 인스타·숏폼·오늘의집 콘텐츠에 그걸 적용하는 것까지 같이 갈까요? 그 실행은 지금 누가 하세요?"
의도: 서비스 범위 직접 확인(Essential vs Professional 분기) + 1인 실행 손 한계 자각 → "적용까지" = Professional 수주.
Q5 — 채널·자사몰 (Phase 2)
"지금 스토어·쿠팡·오늘의집에 다 입점해 계신데, 자사몰이나 인스타 본 계정은 운영하세요? 오늘의집 콘텐츠는 직접 올리시나요?"
의도: 자사 채널·고객데이터 소유 여부 + 오늘의집 활용도 확인. 미활용이면 "여기가 가장 큰 미개발 레버"로 연결.

Challenger Teaching 3종 ("사실 공유" 톤)

① 지금 광고로 버는 매출은 '자산'이 아니라 '임대'다 — 끊으면 0, 자산은 콘텐츠·팬에 쌓인다

② '미니멀'은 ±0·레꼴뜨가 이미 가졌다 — 오네아의 자리는 '미니멀 키친 라이프스타일 시스템'이다

③ 오늘의집은 '판매처'가 아니라 '브랜딩 무대'다 — 연출컷이 곧 자연유입·전환이다
