한 줄 진단
이퀼립은 "매출 트랙·수출 트랙은 있지만 브랜드 자산이 쌓이지 않는" 구조. 28개국 확장·IR 단계에서 이 구조가 그대로 가면 브랜드 가치 평가가 매출 체급을 따라오지 못한다. 오늘 우리가 증명할 것은 브랜드 리빌딩 + 글로벌 OS + 포트폴리오 메타 OS를 한 팀으로 통합 설계할 수 있는 곳은 브랜드라이즈뿐이라는 것.
① 부대표의 직접 호소: "패키징·브랜드 컨센서스 컨설팅 받고 싶다" — 본인이 브랜드 전문가로서 자기 브랜드 객관화가 어려운 상태. 동경하는 기준점(리솔츠)에 부합하는지 확인받고 싶은 욕구가 저변.
② 풀서비스 범위: 브랜드 아이덴티티 재정립 · 패키지 리뉴얼 · 본계 자산화 · 글로벌 브랜드 OS · 공픽 포트폴리오 메타 OS · IR용 자산 · 내부 조직 역량 워크샵 — 6대 모듈 전체 가능.
③ 오늘 14시 60분: 상대가 우리 사무실로 직접 오는 자리 = 의지 매우 강함. 진단은 대담하게, 제안은 구체적으로. Professional 티어(3~5천만 / 3~4개월)에서 자연스럽게 만나는 각.

핵심 진단 5가지 — 오늘 15분 안에 꺼낼 카드

진단 ① 브랜드 일관성 — "본인이 자인한 것, 외부에서도 동일하게 보임"

현재
  • 오 브리에 / 오 캄 / 로맨티크 바디 / 굿즈 — 4+ 라인
  • 라인별 비주얼 톤·타이포·컬러 분기 감지
  • 프렌치 감성 슬로건은 있으나 시스템 부재
  • 본인 발화 "컨센서스 컨설팅 받고 싶다"
필요한 것
  • 마스터 브랜드 OS — 톤·컬러·타이포·보이스 시스템
  • 라인별 변주 룰 + 금지 룰 명시
  • 디자이너·협업사가 쓸 수 있는 실전용 가이드 (PDF 10p 수준)

진단 ② 패키지 전략 — "가격은 프리미엄, 시각은 중간대"

현재
  • 패드 3.6~3.9만원 / 세럼 4.6~5.5만원 = 아누아·메디큐브 대비 10~30% 프리미엄
  • 하지만 매대·올영·H&B 진열 시 시각적 격차 주장 약함
  • 라인 확장 시마다 패키지 언어 흔들림
필요한 것
  • 히어로 SKU 오 브리에 패드 리뉴얼 (프리미엄 격차 시각화)
  • 라인 공통 패키지 언어 시스템 (소재·구조·타이포)
  • 수출국 규제 대응 다국어 레이어

진단 ③ 팔로워 5.4K — "결핍이 아니라 자산 이관 부재"

현재
  • 공구·인플·라이브커머스 = 매출 발생 채널
  • 그러나 트래픽이 인플 계정에서 종료 · 이퀼립 본계로 유입 없음
  • 이메일 리스트·멤버십 데이터베이스 미확보 (추정)
  • IR 시 "공구 의존 브랜드" 로 평가절하 리스크
필요한 것
  • 본계 자산화 퍼널 — 공구 구매자 → 본계 팔로우 → 이메일 리스트 → 재구매
  • SNS 콘텐츠 전략 (주 3회, 히어로-허브-헬프 프레임)
  • 오가닉 UGC 수집 시스템 (체험단 아님, 고객 후기 큐레이션)
  • 목표: 6개월 5.4K→20K, 12개월 50K+

진단 ④ 28개국 글로벌 OS — "3년 내 깨질 수밖에 없는 구조"

현재
  • 11개국 수출 · 17개국 추가 확장 중
  • 프렌치 감성 컨셉 · 슬로건 정립
  • 국가별 파트너/유통사 개별 커뮤니케이션 (추정)
  • 글로벌 브랜드북 부재
필요한 것
  • 글로벌 브랜드북 (영문+다국어)
  • 국가별 로컬라이제이션 가이드 (일본·북미·동남아 등)
  • 클레임·규제 매트릭스
  • IR용 브랜드 자산 덱 (Pre-A/A 준비)

진단 ⑤ 공픽 메타 OS — "120 브랜드 규모는 이미 임계점"

현재
  • 공픽 120 브랜드·1,200 인플 운영
  • 인큐베이팅 사업 확장 의지
  • 이퀼립이 '쇼케이스' 역할 — 하지만 쇼케이스 자체가 흔들림
  • 브랜드별 가이드 개별 관리 (추정)
필요한 것
  • 인큐베이팅 메타 OS — 마스터 톤 1개 + 브랜드별 변주 레이어
  • 포트폴리오 KPI 대시보드 (공통 브랜드 자산 지표)
  • 이퀼립 자체의 브랜드 리빌딩이 선행 조건
5개 진단을 관통하는 한 문장
"이퀼립의 진짜 문제는 패키징이 아니라,
매출은 오르는데 브랜드는 자산이 안 되는 구조입니다.
IR·글로벌·인큐베이팅 스케일 전에 지금 바꿔야 합니다."

통합 솔루션 — 6대 모듈 (풀서비스)

각 모듈은 단독 수주 가능. Professional/Premium 티어에서 3~6개 결합 제안.

M1 · 브랜드 코어

브랜드 아이덴티티 재정립

마스터 브랜드 OS 구축. 톤·컬러·타이포·보이스·스토리 체계. 라인별 변주·금지 룰.
산출물: 브랜드북 30p · 버벌 시스템 · 브랜드 스토리 재정립 / 기간 6~8주
M2 · 패키지

히어로 SKU 리뉴얼 + 패키지 언어

오 브리에 패드 리뉴얼 디자인 + 전 라인 패키지 언어 시스템. 프리미엄 시각화.
산출물: 히어로 SKU 리뉴얼 시안 3종 · 패키지 가이드 · 양산 디렉션 / 기간 8~10주
M3 · 본계 자산화

SNS·콘텐츠 퍼널 설계 + 운영

공구/인플 트래픽 → 본계 이관 퍼널. 히어로-허브-헬프 콘텐츠 프레임. UGC 큐레이션 시스템.
산출물: 콘텐츠 전략 덱 · 3개월 운영 캘린더 · 월간 리포트 / 기간 3개월 리테이너 이상
M4 · 글로벌

글로벌 브랜드 OS + 로컬라이제이션

영문+다국어 브랜드북. 국가별 로컬 가이드. 클레임·규제 매트릭스. IR 덱용 글로벌 자산.
산출물: Global Brand Book · 국가별 가이드 3~5개국 · 규제 매트릭스 / 기간 10~12주
M5 · 포트폴리오

공픽 인큐베이팅 메타 OS

마스터 톤 + 브랜드별 변주 레이어. 포트폴리오 KPI 대시보드. 신규 브랜드 입점 프로세스.
산출물: 메타 OS 프레임워크 · 운영 대시보드 템플릿 · 입점 SOP / 기간 8~12주
M6 · 조직 역량

리솔츠 브랜드 철학 전수 워크샵

이퀼립 내부 디자이너·마케터 대상 집중 세션. 리솔츠 결정 기준·실수담·원류 전수. 장기 디자인 자립.
산출물: 3회 세션 (총 9~12시간) · 세션 아카이브 자료 / 기간 1개월

3-Tier 제안 패키지

가격 제시 원칙: 오늘 미팅 Phase 4에서는 티어별 참고가 범위로만 언급. 구체 견적은 범위 확정 후 별도 제안서. 오늘 자리에서 계약 유도 없음.

니즈 분석 (SPIN + Pain + JTBD)

SPIN 레이어링

S — Situation

이퀼립(스킨케어) + 공픽(인플 커머스) 병행. 국내 공구·인플 중심, 해외 11→28국 확장.

P — Problem

패키징·브랜드 일관성 흔들림 자가 진단. 공구 트래픽이 브랜드 자산으로 이관 안 됨. 글로벌 OS·메타 OS 부재.

I — Implication

글로벌 확장 시 브랜드 붕괴 리스크. 인큐베이팅 사업 스케일 시 내부 모순. IR·투자 유치 시 브랜드 자산 평가 절하.

N — Need-Payoff

"리솔츠 수준의 브랜드 체계를 내 포트폴리오에 이식" — 벤치마크 자체가 브랜드라이즈. 매출 체급에 맞는 브랜드 자산 확보.

Pain Hierarchy

L1 표면"패키징·컨센서스 컨설팅 받고 싶다"
L2 전술라인별 패키지 톤 분화 · 리뉴얼마다 일관성 흔들림
L3 전략포트폴리오 브랜드 간 세계관 분기 · "이퀼립 스튜디오" 정체성 희석
L4 근본전문가 역설 — 본인이 브랜드 전문가라 자기 객관화 어려움. 동경하는 기준(리솔츠)에 부합하는지 확인받고 싶음
