핵심 문제점 & 진단

1. 채널 0 + D-50 + 50만 KPI — 압축 일정 (Critical)

현재 상태 (2026-05 기준)
  • 6채널 (텔·인스타·틱톡·유튜브·디스코드·X) 전부 미개설
  • 광고 리포트 0건 — 마케팅 활동 자체가 시작 전
  • "7월 초 상용서비스" 목표 = D-50일대
  • "6월부터 마케팅 시작" = D-20일대 시작
  • 50만 가입자 = 월 ~8.3만 명 · 일 ~2,800명 페이스
필요: D-50 ~ Launch 시드 라인
  • 코어 3채널 우선 점화: X(Twitter) · 유튜브 쇼츠 · 텔레그램
  • D-30: 시드 콘텐츠 30~50개 비축 (한류 숏 + 학습 변형 숏)
  • D-14: 한류 마이크로 KOL 30명 사전 컨택
  • D-7: 디스코드 DAO 사전 등록 동선 + 가입 인센티브
  • D-Day: 6채널 동시 발화 + "1만 팔로워 기다리는 출발선"

2. "Web3 + 한류 + 교육" 3중 메시지 — 정렬 난도 (High)

노다지파이랩스의 강점이자 위험. "Solve-and-Earn 토크노믹스" + "한류 팬덤 1차 게이트" + "세상의 모든 교육은 유료, 우리는 역발상" — 3개 모두 강력하지만 한 줄로 정리되지 않으면 의심 반사(Web3) · 시류 반사(한류) · 진정성 반사(교육)가 동시에 발생할 위험. 첫 미팅 핵심 진단: 6월 첫 콘텐츠에서 한류로 들어갈지 / Web3로 들어갈지 / 교육으로 들어갈지 순서 정의.

3. 콘텐츠 공장 ↔ 채널 정합 갭 (High)

클라이언트 명시 자산: "학습 콘텐츠 자동 생성 공장" + "한류 문화 소개 숏 콘텐츠". 핵심 무기지만, 공장 산출물이 X(텍스트) · 유튜브 쇼츠(영상) · 텔레그램(커뮤니티) 톤·길이·해시태그에 자동 정합되는가는 별개 문제. K-Pop 팬덤 X 톤과 학습 유튜브 쇼츠 톤은 다름. 정합 룰 없으면 공장이 돌수록 정합성 부담만 증가. 브랜드라이즈가 들어갈 자리: 채널별 콘텐츠 가이드라인 1세트 + 자동 변환 룰 1장.
