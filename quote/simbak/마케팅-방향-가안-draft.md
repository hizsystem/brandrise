## 0) 브랜드 기획 — 현행 (한 셀 압축, 3단 전환 보류)

```
● 리뉴얼 제안
법인 '심박' 뒤 제품들 → 하나의 프리미엄 K-푸드 브랜드로 통합
· 개선: 콤부차·샤인머스켓빵·심박 브랜드명 분산 → 소비자가 '심박'을 모름
· 목표: 수출 10개국·특허·인증 자산을 소비자 언어로 번역 + IR/투자 자료 베이스
· 예시 브랜드: 베지어트(휴닉) · 정관장
```

## 1) 패키지 — 3단 가안 ✅ 확정

**개선지점)**
농식품 프리미엄 선물 시장은 커지는데, 심박은 제품력(수출 10개국·동결건조·영천 산지)에 비해 패키지에 브랜드명조차 안 보이고 매력이 약함 / 샤인머스켓은 생과 가격 폭락에 프리미엄 인식이 동반 추락

**목표)**
- 제품 라벨을 가격대·용도로 구분 + 심박 시각 아이덴티티 개발·라벨 리뉴얼
- 샤인머스켓을 포도(생과)에서 '빛나는 것' 프리미엄으로 리포지셔닝
- 명절·외국인·기업 선물 라인(기프트셋) 개발 → 고급 선물 시장 공략

**타겟 브랜드)**
서로인(한우) · 워커비(꿀) · 호랑이건강원(인삼즙) · 타이거모닝(진저샷) — 프리미엄 농식품 선물 브랜드
※ 우리 작업: 타이거모닝 · 워커비
*이미지: 선물 패키지·기프트셋 목업*

## 2) 인스타그램 — 3단 가안 ✅ 확정

**개선지점)**
인스타 3개 계정을 사장님이 직접 운영 — 합산 2,000 팔로워, 비주얼·콘텐츠로 유입력 없음

**목표)**
- 심박 제품력(수출 10개국·동결건조·영천 산지)에 맞춘 프리미엄 브랜드 스토리텔링 콘텐츠 개발
- 소셜미디어 노출 강화 + 자사몰 유입 연결 설계
- 정기 브랜딩 콘텐츠 발신 — 사람·제품·원물 3축 (제품력 텔링 / 레시피 / 패키징 노출)

**운영방식)**
- 인스타는 브랜드 보유 기본 사진·영상이 핵심 → 그래서 촬영을 이 블록에 포함
- 1개월: 브랜디드 사진·영상 콘텐츠 집중 개발 (촬영)
- 이후 운영은 심박 내부, 기획·디렉션은 브랜드라이즈 (브랜드사 인원 협력 필요)

*예시: 베지어트(원물 3축) · 스타벅스앳홈(신세계X) + 콘텐츠 캘린더*

## 3) 자사몰 (풀만) — 예시 = 워커비·요헤미티 ✅ 확정

```
● 리뉴얼 제안
회사 소개 페이지 수준 자사몰 3개 → 신뢰자산이 보이고 구매전환되는 단일 D2C
· 개선: 자사몰 분산(D2C 매출 12.5%) / 상시 할인이 프리미엄 죽임 / 신뢰자산 미노출
· 방향: 통합 IA(개발 자산은 살림) + 신뢰자산 배지 + 구매 퍼널
· 예시 브랜드: 워커비(D2C 자사몰 운영) · 요헤미티(사이트 퀄리티 벤치마크)
```

> 변경: 베지어트·요헤미티 → **워커비·요헤미티** (워커비=우리 D2C 사례 / 요헤미티=사이트 벤치마크 캡처 활용)
> 타이거모닝은 자사몰엔 약해 패키지 블록에 유지.
