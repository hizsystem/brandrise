2. 작업 방식

4단계 발표·피드백 사이클로 진행합니다. 각 단계 종료 시 발표 + 피드백 회의 1회를 운영합니다.

STEP 1
무드보드 + 방향 3안
컨셉 탐색 · 방향성 협의
STEP 2
1차 디벨롭 시안
선택 1안 디벨롭 · 컬러 체계 확정
STEP 3
2차 시안 + 단상자 전개
단상자 6면 + 디테일 보완
STEP 4
최종 납품 + 인쇄 가이드
파일 전달 · 인쇄 감리 가이드

커뮤니케이션

  • 주 1회 정기 발표 + 피드백 회의 (대면 또는 화상, 1시간 이내)
  • 중간 진행 사항은 공유 폴더(드라이브 또는 노션) + 슬랙 채널로 실시간 공유
  • 긴급 문의는 메일 또는 모바일 (24시간 내 회신)

모듈 추가 시 일정 영향

  • Module A (BI 정비): 본 일정 내 병행 가능. STEP 1~2 단계에서 BI 룰 정의 후 코어 디자인에 반영.
  • Module B (해외 매대): 본 일정 + 1주 (6/10까지). 코어 납품 직후 별도 가이드 정리.
  • Module C (pppf 리뉴얼): 코어 납품 후 별도 1~2주 진행 (6월 중순~말).
  • Module D (클렌징): 본 일정 외, 7~8월 별도 진행.

4. 수정 범위

단계수정 범위
STEP 1 — 방향 3안 3안 중 1안 선택 (방향성 협의 단계로 별도 수정 횟수 미적용)
STEP 2 — 1차 디벨롭 선택 1안 기준 2회 수정 (컬러 · 그래픽 · 타이포 등)
STEP 3 — 2차 시안 + 단상자 1회 미세 수정 (디테일 · 인쇄 사양 보완)
STEP 4 — 최종 납품 후 1개월간 인쇄 감리 미세 수정 1회 무상
별도 견적 영역 · 이미 선택한 1안에서 다른 방향으로 회귀하는 경우 (컨셉 방향 변경) · 신규 컬러 추가 (3종 이상) · 단상자 추가 사이즈 (현 2종 외) · 양산 시작 후 디자인 재설계 — 위 항목은 별도 견적으로 진행하며, 사전에 영향 범위와 가격을 공유드립니다.
