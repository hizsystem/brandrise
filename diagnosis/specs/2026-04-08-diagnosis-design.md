# Brandrise 마케팅 성숙도 진단 — 설계 스펙

> 작성일: 2026-04-08
> 프로젝트: 모수 프로젝트 핵심 퍼널
> 기술 접근: 독립 HTML + Vercel Serverless (C안)

---

## 1. 개요

스타트업 대표가 5분 만에 자사 마케팅 성숙도를 자가진단하고, 즉시 점수+등급을 확인한 뒤, 24시간 내 맞춤 심화 리포트를 받는 시스템.

**목적:** 불특정 다수 → 진단 유입 → 상담 전환 → 수주

**비목적:** 
- 풀 홈페이지가 아님 (진단 전용 랜딩)
- 즉시 결과가 핵심이지, 상담 예약 강요가 아님

---

## 2. 프로젝트 구조

```
brandrise-diagnosis/
├── index.html            # 랜딩 페이지
├── diagnosis.html        # 진단 폼 (7문항, 1문항씩 카드형)
├── result.html           # 즉시 결과 페이지
├── styles.css            # 공통 스타일
├── js/
│   ├── diagnosis.js      # 폼 진행 + 점수 계산 + API 호출
│   └── result.js         # URL params 파싱 → 결과 렌더링
├── api/
│   └── submit.js         # Vercel serverless → 슬랙 + 구글시트
├── vercel.json
└── package.json
```

## 3. 페이지별 상세

### 3-1. 랜딩 (index.html)

**URL:** / (루트)

**구성:**
1. Hero: 메인 카피 + CTA
2. 가치 제안: 6개 영역 소개
3. 사회적 증거: 진단 수 + 후기
4. CTA 반복

**Hero 카피:**
```
우리 브랜드 마케팅,
지금 어디쯤일까요?

5분이면 알 수 있습니다.
무료 마케팅 성숙도 진단.

[5분 무료 진단 시작하기]  ← 오렌지 CTA 버튼
```

**사회적 증거:**
```
✓ 52개 기업이 이미 진단받았습니다
✓ 24시간 내 맞춤 리포트 제공
✓ 상담 강요 없음, 진단만 받아도 OK
```

**6개 영역 아이콘 섹션:**
- 브랜드 일관성 / 성과 추적 / 콘텐츠 체계
- 고객 이해 / 성장 병목 / 실행 속도
- 하단 카피: "6개 영역을 진단해서 지금 가장 시급한 1가지를 알려드립니다."

**레퍼런스 섹션 (선택, 있으면):**
- 진단 후 성과 개선 사례 1-2개

**디자인:**
- 다크 배경 (#0a0a0a)
- 오렌지 액센트 (#E8700A)
- Pretendard Variable 폰트
- 모바일 퍼스트 (타겟이 대표 → 모바일에서 볼 확률 높음)

### 3-2. 진단 폼 (diagnosis.html)

**URL:** /diagnosis.html

**UX 방식:** 1문항씩 카드형 진행 (전체 폼을 한 번에 보여주지 않음)

**흐름:**
```
[기본 정보 입력 (2단계)]
  ├── Step 0: 회사명 + 이름 + 이메일 + 연락처
  └── Step 0-2: 업종 (드롭다운) + 현재 단계 (드롭다운)

[진단 문항 (7단계)]
  ├── Q1: 브랜드 일관성
  ├── Q2: 성과 추적
  ├── Q3: 콘텐츠 체계
  ├── Q4: 고객 이해
  ├── Q5: 성장 병목
  ├── Q6: 실행 속도
  └── Q7: 시급 과제 (복수선택)

[제출]
  → api/submit.js 호출 (비동기)
  → result.html로 이동 (query params)
```

**기본 정보 입력:**

| 필드 | 필수 | 비고 |
|------|------|------|
| 회사명 | 필수 | 텍스트 |
| 이름 | 필수 | 텍스트 |
| 이메일 | 필수 | 심화 리포트 발송용 |
| 연락처 | 선택 | 전화번호 |
| 업종 | 필수 | 드롭다운: 식품·음료 / 뷰티·화장품 / 패션·의류 / 테크·SaaS / 라이프스타일 / 교육 / 기타 |
| 현재 단계 | 필수 | 드롭다운: 예비창업 / 초기(매출 1억 미만) / 성장기(1-10억) / 확장기(10억+) |

**진단 문항 (확정):**

**Q1. 브랜드 일관성**
> 새 콘텐츠나 마케팅 자료를 만들 때, '우리 브랜드답다'고 느껴지나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 뭐가 우리 브랜드다운 건지 기준 자체가 없다 |
| 2 | 느낌은 있는데, 문서화된 건 없다 |
| 3 | 가이드는 있지만 잘 안 지켜진다 |
| 4 | 누가 만들어도 일관된 톤과 비주얼이 나온다 |

**Q2. 성과 추적**
> 지난 달 마케팅에서 '이 활동이 매출에 기여했다'고 증명할 수 있나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 마케팅이 매출에 기여하는지 솔직히 모른다 |
| 2 | 감으로는 아는데 데이터로 증명 못 한다 |
| 3 | 주요 채널의 유입·전환은 추적 중이다 |
| 4 | 채널별 ROAS를 알고 예산을 데이터로 배분한다 |

**Q3. 콘텐츠 체계**
> 다음 달에 어떤 콘텐츠를 어디에 올릴지, 지금 바로 말할 수 있나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 그때그때 생각나면 올린다 |
| 2 | 대략 계획은 있지만 자주 밀린다 |
| 3 | 캘린더가 있고 주 2회 이상 꾸준히 한다 |
| 4 | 시리즈별 기획이 있고 성과 기반으로 조정한다 |

**Q4. 고객 이해**
> 우리 고객이 경쟁사 대신 우리를 선택하는 '진짜 이유' 1가지를 말할 수 있나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 솔직히 잘 모르겠다 |
| 2 | 우리 생각은 있는데, 고객에게 직접 물어본 적은 없다 |
| 3 | 고객 인터뷰·리뷰에서 패턴을 발견했다 |
| 4 | 명확한 차별점이 있고 마케팅에 반영 중이다 |

**Q5. 성장 병목**
> 지금 매출을 2배로 만들려면, 가장 먼저 뭘 해야 할지 알고 있나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 어디서부터 손대야 할지 모르겠다 |
| 2 | 여러 가지 시도 중인데 뭐가 맞는지 확신이 없다 |
| 3 | 핵심 레버 1-2개는 파악했지만 실행이 부족하다 |
| 4 | 액션플랜이 있고 실행 중이다 |

**Q6. 실행 속도**
> 마케팅 아이디어가 나왔을 때, 실행까지 보통 얼마나 걸리나요?

| 점수 | 선택지 |
|------|--------|
| 1 | 아이디어는 많은데 실행된 게 거의 없다 |
| 2 | 1-2개월 (대표가 직접 하느라) |
| 3 | 2주 내 가능 (담당자 or 파트너 있음) |
| 4 | 1주 내 테스트하고, 결과 기반으로 빠르게 돈다 |

**Q7. 시급 과제 (복수선택, 점수 미반영)**
> 지금 가장 시급한 마케팅 과제는?

- [ ] 브랜딩 정립 (로고, 톤앤매너, 가이드라인)
- [ ] 콘텐츠 체계화 (SNS, 블로그, 영상)
- [ ] 광고 시작 또는 최적화 (메타, 구글)
- [ ] 매출 전환 개선 (랜딩페이지, 퍼널)

**폼 인터랙션:**
- 선택지 클릭 → 300ms 딜레이 → 자동 다음 스텝 (Q1-Q6)
- Q7은 복수선택이라 "결과 보기" 버튼으로 제출
- "← 이전" 버튼으로 뒤로 가기 가능
- 상단 프로그레스 바 (오렌지 그라데이션)
- 키보드 단축키: 1/2/3/4로 선택 가능

### 3-3. 결과 페이지 (result.html)

**URL:** /result.html?s=3,2,4,1,3,2&q7=branding,content&company=...&name=...

**점수 체계:**
- Q1~Q6: 각 1-4점, 총 24점 만점
- Q7: 점수 미반영 (방향성 파악용)

**등급:**

| 점수 | 등급 | 라벨 | 진단 메시지 |
|------|------|------|-----------|
| 6-9 | D | 시작 전 | 마케팅의 기초 체력부터 만들어야 합니다. 지금이 가장 빠른 타이밍이에요. |
| 10-13 | C | 기초 단계 | 시작은 했지만, 체계 없이 달리고 있어요. 방향부터 잡으면 속도가 붙습니다. |
| 14-18 | B | 성장 단계 | 기본기는 있어요. 몇 가지 병목만 풀면 확실히 달라집니다. |
| 19-24 | A | 고도화 단계 | 잘하고 계세요. 다음 레벨로 가려면 정밀한 튜닝이 필요한 시점입니다. |

**영역별 "다음 레벨" 조언 매핑 (점수별):**

조언 원칙: 순수하게 도움이 되는 액션만. 브랜드라이즈 서비스 언급 없음.

| 영역 | 1점 → 2점 | 2점 → 3점 | 3점 → 4점 | 4점 (유지) |
|------|-----------|-----------|-----------|-----------|
| 브랜드 일관성 | 로고·컬러·폰트 3가지만 정하면 기준이 생깁니다 | 1장짜리 가이드를 만들어 팀/외주에 공유하세요 | 실행 체크리스트를 만들면 지켜지기 시작합니다 | 잘하고 계세요. 이 일관성을 유지하면서 다른 영역에 집중하세요 |
| 성과 추적 | GA4 설치부터. 무료이고 검색하면 10분이면 됩니다 | UTM 태깅을 시작하면 채널별 기여도가 바로 보입니다 | 주간 리포트 루틴 하나만 잡으면 데이터 기반 의사결정이 됩니다 | 잘하고 계세요. 데이터 기반 운영이 가장 큰 강점입니다 |
| 콘텐츠 체계 | 주 1회, 1개 채널부터 시작하세요. 그것만으로 충분합니다 | 월간 캘린더 하나만 만드세요. 노션이나 시트면 됩니다 | 콘텐츠별 성과(저장/공유/유입)를 보고 잘 되는 포맷에 집중하세요 | 잘하고 계세요. 체계적 운영이 큰 자산입니다 |
| 고객 이해 | 고객 3명에게 "왜 우리를 선택했나요?"만 물어보세요 | 리뷰·문의에서 반복되는 키워드를 뽑아보세요 | 세그먼트별로 메시지를 다르게 테스트해보세요 | 잘하고 계세요. 고객 이해가 마케팅의 기초 체력입니다 |
| 성장 병목 | 지금 매출의 80%가 어디서 오는지 먼저 파악하세요 | 2가지 중 ROI 높은 1개를 골라 4주만 집중하세요 | 실행 로드맵에 주간 마일스톤을 넣으세요 | 잘하고 계세요. 집중력이 가장 큰 무기입니다 |
| 실행 속도 | 이번 주 1가지만 정해서 금요일까지 해보세요 | 마케팅 전담 or 파트너를 정하면 속도가 완전히 달라집니다 | 주간 스프린트 → 결과 체크 → 조정 루틴을 만드세요 | 잘하고 계세요. 빠른 실행력이 최고의 경쟁력입니다 |

**"지금 상태" 텍스트 매핑 (선택한 답변 리프레이밍):**

| 영역 | 1점 | 2점 | 3점 | 4점 |
|------|-----|-----|-----|-----|
| 브랜드 일관성 | 브랜드다운 게 뭔지 기준이 아직 없는 상태입니다 | 느낌은 있지만 공유가 안 되고 있어요 | 가이드는 있는데 실제로 안 지켜지고 있어요 | 누가 만들어도 일관된 결과가 나옵니다 |
| 성과 추적 | 마케팅이 매출에 기여하는지 파악이 안 되고 있어요 | 감으로는 아는데 데이터로 증명이 안 됩니다 | 주요 채널의 유입과 전환을 추적하고 있어요 | 데이터 기반으로 예산을 배분하고 있습니다 |
| 콘텐츠 체계 | 계획 없이 그때그때 올리고 있어요 | 계획은 있지만 자주 밀리고 있어요 | 캘린더 기반으로 꾸준히 발행하고 있어요 | 성과 기반으로 기획을 조정하고 있습니다 |
| 고객 이해 | 고객이 왜 우리를 선택하는지 아직 모릅니다 | 가설은 있지만 고객에게 직접 검증하진 않았어요 | 고객 피드백에서 패턴을 발견했어요 | 차별점이 명확하고 마케팅에 반영 중입니다 |
| 성장 병목 | 뭘 먼저 해야 할지 아직 파악이 안 됐어요 | 여러 가지를 하고 있지만 확신이 없어요 | 핵심 레버는 알지만 실행이 부족해요 | 실행 중인 액션플랜이 있습니다 |
| 실행 속도 | 아이디어는 많은데 실행된 게 거의 없어요 | 대표가 직접 하다 보니 1-2개월씩 걸려요 | 담당자가 있어서 2주 내 실행 가능해요 | 1주 내 테스트하고 빠르게 조정합니다 |

## 4. API (Serverless Function)

### POST /api/submit

**Request:**
```json
{
  "company": "회사명",
  "name": "대표자명",
  "email": "email@example.com",
  "phone": "010-1234-5678",
  "industry": "식품·음료",
  "stage": "성장기(1-10억)",
  "scores": [3, 2, 4, 1, 3, 2],
  "q7": ["branding", "content"],
  "totalScore": 15,
  "grade": "B",
  "weakest": "고객 이해",
  "timestamp": "2026-04-08T10:30:00+09:00"
}
```

**Response:**
```json
{ "ok": true }
```

**동작:**
1. 슬랙 webhook으로 알림 전송
2. (선택) Google Sheets API로 데이터 기록

**슬랙 알림 포맷:**
