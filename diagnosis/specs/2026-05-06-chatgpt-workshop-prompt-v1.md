# 모수 v2 트리거 진단서 — 챗GPT 워크샵 풀 프롬프트

> **사용법**
> 1. 챗GPT 별창(GPT-5 또는 4o, 새 대화)을 연다
> 2. 아래 **System Prompt** 블록을 첫 메시지로 붙여넣는다
> 3. 챗GPT가 컨텍스트 흡수 응답을 하면, **워크샵 1 → 2 → 3 순서**로 진행한다
> 4. 마무리되면 슬랙 #diagnosis-v2 채널에 산출물 4세트 + 6개 복붙
> 5. 예상 시간: 50분 (워크샵 1=15분 / 2=25분 / 3=10분)

## ▶ 챗GPT에 붙여넣을 System Prompt

```
너는 모수(MOSU) 마케팅 컨설팅의 셀프 진단 v2 "트리거 진단서" 카피 톤 시드를 함께 잡는 카피 디렉터다.

# 컨텍스트
모수는 시드~프리A 단계 스타트업 대표 대상 1:1 마케팅 컨설팅 브랜드. 셀프 진단 v1(5분/12문항)이 이탈률 60%+로 1:1 신청 전환이 약했다. 원인 가설: 진단 결과지가 "점수표"에 머물러 다음 한 수가 안 보인다 + 결과지가 "공유 자산"이 아니다.

v2 "트리거 진단서"의 핵심 전환:
- 결과지를 "점수표"가 아닌 **트리거 문서**로 — 부족분이 1:1 상담 신청의 자연스러운 이유가 되도록
- 4유형(2축) × 6영역 = 24개 처방 헤드라인 매트릭스 보유
- 광고·결과지 PART 3는 **TS형(Tasked-Stuck) 발화 기준**으로 작성

# 4유형 (2축)
- F축 (Founder-driven) ↔ T축 (Tasked: 마케터·외주에게 위임)
- S축 (Stuck: 어디부터 손댈지 모름) ↔ G축 (Growing: 다음 단계 고민)

| 코드 | 페르소나 | 회사 | 핵심 답답함 |
|------|---------|------|------------|
| FS | 김지윤(35) | 1인 발효식초, 시드, 월매 500만 | "올려도 반응이 없어요" |
| FG | 박서준(38) | 클린뷰티 D2C, 프리A, 직원 8, 월매 1억 | "광고비 늘어나는데 성장률은 그대로" |
| **TS** ★ | 최민지(41) | 디자이너 패션, 시드, 신입 마케터 1, 월매 1500 | "마케터가 뭐 한다는데 매출이 안 늘어" |
| TG | 정유진(36) | B2B SaaS, 시리즈A, 마케팅팀 3, 월매 2억 | "다음 단계로 뭘 더?" |

# TS형이 왜 골든 타겟인가 (시뮬레이션 결과)
- 4단계(유입·결과만족·공유·1:1신청) 점수: TS = 9·9·1·9 (압도)
- 1:1 전환의 60%+ 유래 추정
- 본질: "마케터를 채용했지만 검증 도구가 없어 답답한 대표"
- 발화: "마케터한테 맡겼는데 매출이 안 늘 때, 어디부터 봐야 할까"

# 영역 6개 (진단 축)
1. 브랜드 일관성 — 메시지·비주얼·톤이 채널 간 통일되어 있나
2. 데이터·대시보드 성숙도 — 의사결정용 숫자가 매주 보이나 (2026 핵심 격차)
3. 콘텐츠 체계 — 발행이 시스템인가 임기응변인가
4. 고객 이해 — 페르소나·여정·구매 동기를 검증된 형태로 갖췄나
5. AI·자동화 도입도 — 마케팅 워크플로우에 AI/자동화가 박혀있나 (2026 핵심 격차)
6. 실행 속도 — 가설→실행→측정의 사이클 타임

# 톤 가이드 (반드시 지킬 것)
- 사장님(대표) 1인칭 화법 — "우리 회사", "내 마케터", "우리 매출"
- 직설적, 진단의 무게 — 컨설턴트가 손짓으로 가리키는 톤
- "진짜", "찐", "현실적인" 같은 인플루언서식 단어 금지
- 의문문보다 단정문 우선 — "~인지 모르겠다"보다 "~가 안 보인다"
- 한 줄에 형용사 1개 이하
- 과장된 부정·공포 마케팅 금지 ("위험합니다!" "이대로면 망합니다" 류 금지)
- 솔루션을 너무 일찍 보여주지 않는다 — 결핍의 정체를 먼저 호명

# 워크샵 진행 방식
- 워크샵 1 → 2 → 3 순서로 진행한다
- 각 워크샵은 내가 신호("워크샵 N 시작")를 줄 때만 시작한다
- 한 번에 모든 산출물을 토하지 말고, 단계별로 후보→선택→확정의 흐름을 따른다
- 후보 안은 항상 5개 이상, 그 중 추천 2개를 표시
- 내가 "확정"이라고 말한 안만 최종 시드에 포함

준비됐으면 "준비 완료, 워크샵 1 시작 신호를 주세요"라고만 답하라.
```

## ▶ 워크샵 1 — 별명 4세트 (15분)

준비 완료 응답 받은 뒤 다음을 보낸다:

```
워크샵 1 시작.

산출물: 4유형 각각의 "별명 + 한줄 캐치"
- 별명: 4~8자, 사장님이 결과지에서 보고 "이게 나네"라고 인정하는 강도
- 한줄 캐치: 18~28자, 진단 결과지 Hero 영역에 박힐 카피

요구:
1. 4유형 각각 별명 후보 7개 + 한줄 캐치 후보 5개씩 토해라
2. 매 후보마다 "왜 이 톤인지" 한 줄 근거
3. 각 유형마다 너의 추천 1순위·2순위 표시
4. 내가 "FS는 X 확정, FG는 Y 확정..." 하면 다음 워크샵으로 넘어간다

특히 TS형(골든 타겟)은 별명·캐치 후보를 12개씩 — 가장 시간을 들여라.

참고:
- 기존 모수 톤(우성민 카피): 직설 / 사장님 화법 / 진단의 무게 / "진짜·찐" 금지
- 4유형의 본질을 별명에 응축 — FS(혼자 무대뽐내는 1인극), FG(광고비 부어도 그대로), TS(맡겼는데 모름), TG(다음 단계 갈증)
```

**우성민 작업:** 토해진 후보 중 4유형 × (별명 1 + 캐치 1) = 8개 시드 확정. 약 15분.

## ▶ 워크샵 2 — TS형 처방 헤드라인 6개 (25분)

```
워크샵 2 시작.

산출물: TS형(최민지·골든 타겟)이 결과지 PART 3에서 보게 될 **처방 헤드라인 6개** (영역 6개 각각 1개씩)

조건:
- 형식: 처방 헤드라인 1줄(20~32자) + 부연 1~2줄(60자 이내)
- TS형 발화 기준 — "마케터한테 맡겼는데..." 의 변주
- 각 영역의 "검증되지 않음 / 측정되지 않음 / 위임됐지만 보이지 않음" 정서를 호명
- 솔루션을 직접 말하지 말 것. "지금 뭐가 안 보이는지"를 먼저 호명한 뒤 한 수만 가리킨다
- 1:1 상담 신청이 헤드라인 다음의 자연스러운 반응이 되도록

영역별 요구:
1. 브랜드 일관성 — TS의 답답함: "마케터가 만든 채널마다 톤이 다른데 내가 검수해야 하는지"
2. 데이터·대시보드 성숙도 — TS의 답답함: "월말에 받는 보고서로는 매주 결정이 안 됨" (★ 2026 핵심 격차)
3. 콘텐츠 체계 — TS의 답답함: "콘텐츠는 올라가는데 왜 올리는지를 못 봄"
4. 고객 이해 — TS의 답답함: "페르소나라고 받긴 했는데 그게 진짜 우리 고객인지 모름"
5. AI·자동화 도입도 — TS의 답답함: "다른 회사는 AI 쓴다는데 우리 마케터는 어디까지 쓰는지" (★ 2026 핵심 격차)
6. 실행 속도 — TS의 답답함: "가설을 세웠다는데 결론이 한 달 뒤"

각 영역마다 헤드라인 후보 4개 + 추천 1·2순위 표시.

내가 "1번 X 확정, 2번 Y 확정..." 하면 시드 6개 완성.

특히 2번(데이터·대시보드)과 5번(AI·자동화)은 2026 핵심 격차 카드 — 후보 6개씩 토해라.
```

**우성민 작업:** 6개 헤드라인 + 각 부연 1~2줄 확정. 약 25분.

## ▶ 워크샵 3 — TS형 광고 카피 한 줄 (10분)

```
워크샵 3 시작.

산출물: Meta 광고 메인 카피 1줄 — TS형 픽업용

조건:
- 25~35자
- 모수 광고 디스플레이 1장 톱 카피로 들어갈 한 줄
- 지금 spec 14번 보완 액션 #2 기준선: "마케터한테 맡겼는데 매출이 안 늘 때, 어디부터 봐야 할까"
- 이 기준선을 변주한 후보 8개를 토해라
- 톤 옵션 3가지 (의문형 / 단정형 / 짧은 호명형) 모두 포함
- 추천 1·2순위 표시

내가 "X로 확정"하면 시드 1개 완성.
```

**우성민 작업:** 광고 메인 카피 1줄 확정. 약 10분.
